pycsw¶

メタデータ・カタログ¶

pycsw により、ユーザーは、標準準拠のインタフェースとフォーマットを使用して、データ・セット記述 (メタデータ) のカタログを Webに公開することができます。これにより、カタログ・クライアントによる照会および更新が可能になります。pycsw は Apache Web サーバー (または任意の WSGI サーバー ) 内で簡単に設置できるため、データベース内の XML ファイルとして保管されるメタデータのレコードにアクセスできます。

メタデータは 『データに関するデータ』 であり、作成日、作成者、タイトル、興味分野などといったものを格納しており、 ISO 19115、 ISO 19139 、 FGDC CSDGM などの国際標準に従って XML ファイルとして通常エンコードされています。

pycsw は Python による Catalogue Service 実装です。このプロジェクトは、 OGC 準拠 として認定されており、 OGC リファレンス実装 です。CSW 標準は、空間データ、 Web サービス、および関連リソースに関するメタデータを発見、参照、および照会するための共通インタフェースを定義します。

pycsw はオープンソースで、MIT ライセンスでリリースされ、すべての主要プラットフォーム (Windows、Linux、Mac OS X)上で動作します。

主な機能¶

  • OGC CSW 2.0.2 および CSW 3.0.0 の OGC 仕様の準拠および OGC 参照実装の認定
  • WMS, WFS, WCS, WPS, WAF, CSW, SOSのサポート
  • INSPIRE Discovery Services 3.0の実装
  • ISO Metadata Application Profile 1.0.0の実装
  • FGDC CSDGM Application Profile for CSW 2.0の実装
  • Search/Retrieval via URL (SRU)検索プロトコルの実装
  • フルテキスト検索機能を実装
  • OGC OpenSearch Geo/Time 拡張の実装
  • メタデータ収集のためのOpen Archives Initiative Protocol実装
  • ISO, Dublin Core, DIF, FGDC, Atom, GM03のメタデータモデルの実装
  • CGI または WSGIによる設置
  • Python 2と3に互換
  • 単純な設定
  • トランザクション機能 (CSW-T)
  • 柔軟なリポジトリー構成
  • GeoNodeの接続性
  • HHypermapの接続性
  • Open Data Catalogの接続性
  • CKAN の接続性
  • 分散カタログの分散検索
  • リアルタイム XML スキーマ妥当性検査
  • 拡張可能プロファイル・プラグイン・アーキテクチャー
  • 非常に軽量(Python + CGI)

実装されている標準規格¶

OGC Standards:

  • Catalogue Service 2.0.2, 3.0.0
  • Filter Encoding (FE) 1.1.0, 2.0.0
  • OWS Common 1.0.0, 2.0.0
  • Geography Markup Language (GML) 3.1.1
  • Simple Feature Access (SFS) 1.2.1
  • OGC OpenSearch 1.0

Metadata Standards:

  • Dublin Core 1.1
  • SOAP 1.2
  • ISO 19115:2003
  • ISO 19139:2007
  • ISO 19119:2005
  • NASA DIF 9.7
  • FGDC CSDGM 1998
  • GM03 2.1
  • SRU 1.1
  • OAI-PMH 2.0
  • A9 OpenSearch 1.1
