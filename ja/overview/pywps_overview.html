PyWPS¶

Web Processing Service¶

PyWPS は、カスタムの地理空間操作 ( プロセスと呼ばれる ) への Web アクセスを、 Web Processing Service (WPS) 標準で提供しています。

プロセスは、 Python プログラミング言語で実装されており、 GRASS GIS, R, GDAL/OGR, proj などのライブラリを、Pythonバインディングを通じて統合できます。ソース・データは、 WPS 要求で提供することも、 WPS サーバー上のデータからアクセスすることもできます。

主な機能¶

  • 小型で軽量

  • インストールとセットアップが容易かつ迅速

  • 純粋な Python で実装されている

  • シームレスに統合されたPython エコシステムのパッケージ:

    • GRASS 7

    • R

    • GDAL/OGR

    • Proj

    • Numpy

    • Shapely

実装されている標準規格¶

OGC Standards:

  • WPS 1.0.0
