iD エディタ¶

OpenStreetMap 用ブラウザベースエディタ¶

iD エディタは、事実上のOSM標準のブラウザベースの OpenStreetMap エディタです。iD は高速で使いやすく、衛星や航空画像、GPS、Field Papers、Mapillary などのさまざまなデータソースからのマッピングが可能です。

iD エディタは、より高度な JOSM エディタの高機能を必要としない、小さくて簡単な変更を編集するときに、優れた方法です。

主な機能¶

  • OpenStreetMap 編集

  • 高速

  • 直感的

  • ブラウザベース

実装されている標準規格¶

  • Web Map Tile Service (ベースマップの場合)

詳細¶

Webサイト: https://wiki.openstreetmap.org/wiki/ID

ライセンス: ISC

ソフトウェアバージョン: 2.2.2

対応プラットフォーム: Windows, GNU/Linux, Mac, Android, iOSのブラウザならどれでも

サポート: https://help.openstreetmap.org/
