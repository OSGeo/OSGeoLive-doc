ncWMS¶

Web Map サービス¶

ncWMS は OGC 準拠の Web Map Service (WMS) として多次元データ ( 環境や気象データなど ) を公開しています。ncWMS は、通常、環境データに関連付けられた非常に大きなデータ・ファイルから、マップ・イメージを迅速かつ効率的に抽出します。

主な機能¶

  • データの高速抽出
  • タイムシリーズの生成と垂直プロファイルの作図
  • 断面にそったデータのプロットができる
  • ベクタ成分の自動認識 - ベクタ矢印の生成
  • ( ソース・データとターゲット・イメージの両方について ) 様々な投影法のサポート
  • アニメーション作成のサポート
  • ブラウザ内蔵
  • Google Earth KML フォーマットへのエクスポート
  • 入力フォーマット: NetCDF, GRIB, OPeNDAP
  • 出力フォーマッット: PNG, JPEG, animated GIF, KML

実装されている標準規格¶

  • Web Map Service (WMS) (v1.1.1, v1.3.0)
  • KML
