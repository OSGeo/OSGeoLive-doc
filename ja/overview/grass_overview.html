GRASS GIS¶

フル機能のGIS¶

GRASS（Geographic Resources Analysis Support System、地理空間情報リソースの解析補助システム） はラスタ、ベクタ、地理情報処理エンジンを一つのソフトウェアに統合した地理情報システム (GIS) です。GRASS GIS には、空間モデリング、ラスタおよびベクタデータの視覚化、地理空間データの管理と解析、衛星および航空写真画像の処理のためのツールが含まれています。高度な時系列処理のための時間的フレームワークと、高速な地理空間プログラミングのための Python API が付属しています。GRASS GIS は、パフォーマンスおよび大規模な空間データ分析用に最適化されています。さらに、プレゼンテーション用のすばらしい画像の作成、地図の印刷も可能です。GRASS GIS は多くの言語に翻訳されており、多数のデータフォーマットに対応しています。

GRASS GIS は、多くの政府機関や環境コンサルティング会社だけでなく、学術的および商業的な環境で世界中のあらゆる所で使用されています。さまざまな一般的なハードウェアプラットフォーム上で動作し、GNU General Public License の条件に基づいてリリースされたフリーのオープンソースソフトウェアです。

GRASS GIS ソフトウェアには、400以上の解析モジュールが組み込まれています。さらに、コミュニティが提供する100を超えるモジュールとツールボックスは、アドオンリポジトリから無償で利用できます。GRASS GIS は、30年以上にわたって継続的に開発されてきたオープンソースGISの中で、最も長く維持され、かつ最大のものです。これは簡単にスクリプト化でき、超並列スーパーコンピュータ上でのバッチデータ処理にも、ハンドヘルドPDAやネットブック用の便利なデスクトップGISにも十分に使用できます。
