52 North WPS¶

About¶

52°North Web Processing Service (WPS) は、Sextante、ArcGIS Server、R、GRASS 7、または独自に開発した機能によって提供される空間処理アルゴリズムへのWebアクセスを可能にします。アルゴリズムは、2つの異なる季節の間のインフルエンザ症例の差を決定するようなシンプルなものであっても、地球規模の気候変動モデルのような複雑なものであってもかまいません。

主な機能¶

コアGIS機能:

  • SimpleBufferAlgorithm

  • DouglasPeuckerAlgorithm

  • IntersectionAlgorithm

  • AddRasterValuesAlgorithm

  • CoordinateTransformAlgorithm

バックエンドアルゴリズムは次の方法で利用可能:

  • Sextante

  • ArcGIS Server

  • R

  • GRASS 7

サポートされている入力形式と出力形式:

  • ベクタデータ: GML2, GML3, SHP files, KML, WKT (Well-known Text), DGN, JSON

  • ラスタデータ: Geotiff, AsciiGrid, NetCDF, JPEG, PNG, HDF-EOS, ERDAS HFA

  • 出力は GeoServer/MapServer に保存可能
