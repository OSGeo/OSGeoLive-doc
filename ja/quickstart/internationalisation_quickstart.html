OSGeoLive 国際化 クイックスタート¶

言語の変更¶

OSGeoLive は標準で多くの言語に対応しています。言語を切り替えるには以下を実行します:

  1. ログアウトします
  2. user=」user」, password=」user」 でログインします
  3. 画面上部のパネルメニューから別の言語を選択します
  4. Log in ボタンをクリックします
  5. ほかの言語に変更したいときは再度言語を選択します

言語を選択すると、システムメニューと多くのアプリケーションが選択した言語で表示されます。特定のアプリケーションで英語が表示され続ける場合は、そのプラグラムで翻訳が利用できない状態になっています - ボランティアでの翻訳参加検討をお願いします。詳しくは興味のあるプログラムまで直接お問い合わせください。

その他の言語については、メインメニューから Preferences ‣ Language Support を確認してください。(インターネット接続が必要となります)

キーボードサポート¶

国際化キーボード環境はインストール済みですが、ご利用のキーボードに合わせて設定する必要があります。設定を変更するには、メインメニューから Preferences ‣ Keyboard Input Methods を開き、 Input Method タブをクリックします。次に Customize active input methods をクリックし、 Select an input method ボタンをクリックします。右側の Add ボタンをクリックして追加のレイアウトを追加し、 IBus Preferences window の Close をクリックします。

これで、デスクトップ下部のメニュー上のキーボードアイコンをクリックし、アクティブなキーボードレイアウトを選択することができるようになりました。
