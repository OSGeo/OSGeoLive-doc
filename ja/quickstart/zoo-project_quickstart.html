Zoo プロジェクト クイックスタート¶

ZOOプロジェクトは、Web Processing Servicesを作成およびチェーンするための開発者向けのWeb Processing Service(WPS)フレームワークを提供します。WPSは空間アルゴリズムを実行する機能へのWebアクセスを提供します。

ZOOプロジェクトを開始する¶

[スタート]メニューから Geospatial ‣ Web Services ‣ ZOO Project を選択します。

FirefoxはZOOプロジェクトのデモランディングページを開きます。ここから、デモアプリケーションを選択できます。

OGR空間ツールのデモ¶

  1. ZOOプロジェクトのデモランディングページで、OGR空間ツールの view をクリックします。

  2. 地図をクリックして GeoServer WFS Serverから地物を選択すると、地物が青で表示されます。

  3. バッファ、重心、凸包、境界などの単純なジオメトリ処理をクリックすると、結果が黄色で表示されます。

  4. 最初に選択した地物の近くにある別のジオメトリを選択し、[和](Union)、[交差](Intersection)、[対称差](Symmetric Difference)、[差](Difference)などの複数のジオメトリの処理ボタンをクリックして、結果を地図上に赤で表示します。

CGAL三角形分割デモ¶

  1. ZOOプロジェクトのデモランディングページで、CGAL三角形分割の view をクリックします。

  2. ボロノイまたはドロネーボタンを押して、以下に示すように、WPSサービスの実行結果をマップに表示します。

OTBアプリケーションのデモ¶

  1. ZOOプロジェクトのデモランディングページで、OTBアプリケーションの view をクリックします。

  2. 左側のフォームに入力して、WPSサービスの実行を設定します。

  3. 処理する値を選択したら、左側のフォームの下部にある Execute ボタンを押します。

  4. 出力形式としてimage/pngを選択した場合、処理結果が地図に表示されます。image/png出力フォーマットを選択すると、サンプルのKMeansClassification実行が表示されます。

次のステップ¶

  • ZOO WPSサーバーの機能にアクセスできます。

    http://localhost/cgi-bin/zoo_loader.cgi?Request=GetCapabilities&Service=WPS

  • 詳細については、ZOO Projectメーリングリスト: zoo-discuss@lists.osgeo.org に問い合わせてください。
