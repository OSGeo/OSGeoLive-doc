Mapnik クイックスタート¶

Mapnikは、マッピング・アプリケーションを開発するためのツールキットです。Mapnikの目的は、何よりも美しい地図を作ることです。簡単に拡張でき、デスクトップとWebの両方の開発に適しています。

Mapnik & Python¶

Mapnikとそのpythonバインディングがインストールされ、このマシンでのスクリプト作成に使用できるようになります。

Mapnikを使えば、Pythonでマップを簡単に作成できます。ターミナルウィンドウを開き、コマンドラインに`python`と入力して、pythonインタプリタを起動します。次の操作を試してください:

>>> import mapnik, os
>>> m = mapnik.Map(600,400)
>>> style = '/usr/local/share/mapnik/demo/population.xml'
>>> mapnik.load_map(m,style)
>>> m.zoom_all()
>>> mapnik.render_to_file(m,'map.png')
>>> os.system('xdg-open map.png')

注釈

上記のコードは、Mapnikが読み取れるXMLスタイルシートを用意し、データを読み込むレイヤーと、それらのレイヤーに適用するスタイルを定義することに依存しています。Quantumnikプラグインを使えば、QGISの中で以下のいずれかを作ることができます: https://plugins.qgis.org/plugins/quantumnik/

Mapnik & Leaflet¶

OSM/GoogleタイルスキームでOpenLayersウェブマップを表示する、タイル提供用にMapnikを使用する基本的なデモアプリケーションが利用可能です。

このデモでは、Mapnik用に設計された`TileStache`と呼ばれるtileserverを使っています。詳細については、TileStacheのウェブサイトを訪れてください: http://tilestache.org/

  1. Click Desktop ‣ Spatial Tools ‣ Start Mapnik & TileStache. The server starts in a terminal window (which stays open and outputs basic debugging information).
  2. サーバからタイルを要求して、サーバが動作していることを確認します。このリンク http://localhost:8012/example/0/0/0.png をクリックして、ターミナルウィンドウの出力を確認します。
  3. Leafletデモアプリケーションのページ http://localhost/leaflet-demo.html. にアクセスしてください。
