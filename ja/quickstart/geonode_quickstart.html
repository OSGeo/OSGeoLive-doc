GeoNodeクイックスタート¶

オープンソース地理空間コンテンツ管理システム

GeoNodeは、地理空間情報システム(GIS)の開発および空間データインフラストラクチャ(SDI)の展開のためのWebベースのアプリケーションおよびプラットフォームです。

目次

  • GeoNodeを起動

  • 登録ユーザとしてサインイン

  • 新しいレイヤを追加

  • 新規マップの作成

  • あなたのマップを共有

  • 次のステップ

GeoNodeを起動¶

OSGeoLive DVD上でGeoNodeを起動するには、 Geospatial ‣ Browser Clients ‣ GeoNode ‣ Start GeoNode を選択する必要があります。しばらくすると、GeoNodeのWebページが http://geonode で自動的に開きます。ページは次のようになります。

登録ユーザとしてサインイン¶

上記のインタフェースから、既存の空間レイヤとマップをビューおよび変更したり、他のGeoNodeユーザに関する情報を検索したりできます。しかし、ログインしていないと、パブリックレイヤへの読み取り専用アクセスに制限されます。マップを作成してレイヤを追加するには、最初にサインインする必要があります。

Webインタフェースの任意のページから、ページの右上に[サインイン]リンクが表示されます。そのリンクをクリックし、表示されるダイアログでユーザ認証情報を入力します。ユーザ名とパスワードに "admin" と "admin" を使用してみてください。

新しいレイヤを追加¶

レイヤは、ラスタまたはベクタ空間データソースを表すパブリッシュされたリソースです。レイヤはメタデータ、評価、コメントに関連付けることもできます。

  1. 既存のレイヤを表示するには、ページの最上部にあるナビゲーションバーから [データ] メニューを選択し、 [レイヤ] リンクをクリックします。

結果は次のようになります。

  1. アカウントにレイヤを追加するには、 Layers ページの右上にある Upload ボタンをクリックします。

他のページからデータをアップロードする場合は、ナビゲーションバーから [Data] メニューを選択し、 [Upload Layers] リンクをクリックします。

上記の [アップロード] リンクのいずれかをクリックすると、アップロードページが表示されます。

  1. ファイルを追加する方法は2つあります。これは、 Drop Files Here 領域の ドラッグ&ドロップ を使用するか、 Choose Files ボタンを使用して選択できます。shp 、 prj 、 dbf 、 shx ファイルからなる完全なファイルセットをアップロードする必要があることに注意してください。いずれかが欠けていると、アップロード前にGeoNodeから警告が表示されます。

  2. 次に、 ファイルを選択 ボタンをクリックすると、ファイル選択ウィンドウが表示されます。/home/user/data/natural_earth2 フォルダに移動し、ne_10m_admin_0_countries ファイルをすべて選択します。

これで、アップロードするすべてのファイルが表示されます。

ちなみに

GeoNodeには、レイヤのビュー、編集、管理が可能なユーザを制限する機能があります。ページの右側には Permission セクションが表示され、レイヤのアクセスを制限できます。[Who can view and download this data] の下で、 [Any registered user] を選択します。これにより、匿名のビューアクセスが無効になります。同じ領域で、 [Who can edit this data] の下からユーザ名を選択します。これにより、レイヤのデータを編集できるのはあなただけになります。

  1. データをアップロードするには、下部の Upload ボタンをクリックします。

  2. レイヤのアップロードが完了したら、 Layer Info ボタンを使用してレイヤをプレビューするか、 Edit Metadata を使用してレイヤメタデータを編集するか、 Manage Styles ボタンを使用してGeoServer上のレイヤスタイルを編集できます。

  3. Layer Info ボタンを押して、レイヤページを表示

新規マップの作成¶

次のステップは、マップを作成し、新しく作成したレイヤをこのマップに追加することです。

  1. [Maps] ツールバーの [Explore Maps] リンクをクリックします。これにより、地図の一覧が表示されます。

  2. 現在、地図はありません。追加するには、 Create a New Map ボタンをクリックします。

マップ合成インタフェースが表示されます。

404ポップアップメッセージが表示された場合は無視してください。これは、OSGeoLiveにインストールされていないGeoServerプラグインが存在しないためです。

このインタフェースには、ツールバー、レイヤ一覧、マップウィンドウがあります。マップウィンドウには、デフォルトでOpenStreetMapレイヤが表示されます。

  1. New Layers ボタンをクリックし、 Add Layers を選択します。

これで、利用可能なすべてのレイヤが表示されます。この場合、これには前に追加したものが含まれます。

  1. 最初の項目をクリックし、Ctrlキーを押しながら残りの項目をクリックして、いくつかのレイヤを選択します。 Add Layers をクリックして、すべてのレイヤをマップに追加します。

レイヤがマップに追加されます。

  1. マップを保存するには、ツールバーの Map ボタンをクリックし、 Save Map を選択します。

  2. マップのタイトルと要約を入力してください。

  3. 保存 をクリックします。ページの右上のリンクが、マップの名前を反映するように変更されていることに注意してください。

    このリンクには、マップへのパーマリンクが含まれています。このリンクを新しいウィンドウで開くと、マップは保存された状態で表示されます。

  4. Add Layers ボタンの横の Done をクリックして、マップの凡例を表示します。

あなたのマップを共有¶

次に、マップを終了します。

  1. USA Population レイヤを右クリックして、 Zoom to layer extend を選択します。

  2. ズームやパンの設定など、必要に応じてマップの構成を最終的に調整します。

  3. ツールバーの Map ボタンをクリックし、 Publish Map をクリックします。

  4. 前に作成したタイトルと要約が残っているはずです。必要に応じて調整し、 Save をクリックします。

  5. 新しいダイアログが表示され、このマップをWebページに埋め込む方法がコードスニペットを含めて指示されます。必要に応じてパラメータを調整できます。

これで、マップを共有できます。
