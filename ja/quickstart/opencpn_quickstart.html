OpenCPNクイックスタート¶

OpenCPNは、簡潔なCPN(Chart Plotter and Navigator)です。常に、ソフトウェアや電子機器を唯一のナビゲーション手段として使用したり、窓から目を離さないでください。

チャートを設定する¶

LiveDVDでOpenCPNを実行する前に、サンプルデータを有効にする必要があります。これは、NOAAの自由な海図を再配布するための条件です。OpenCPNを初めて開くと、自動的に開くオプションが表示されます。

  1. スタートメニューから Geospatial ‣ ナビゲーションと地図 ‣ OpenCPN を選択します。
  2. Click I agree to the NOAA Nautical Chart Data Agreement.
  3. OK をクリックして、OpenCPN ライセンスに同意します。

https://www.charts.noaa.gov/ のNOAAから追加の米国海図をダウンロードできます。

GPSを設定する¶

GPSに接続している場合は、gpsdサービスを開始します。これを行うには、ターミナルを開き、次のコマンドを実行します。

sudo dpkg-reconfigure gpsd

and

sudo /etc/init.d/gpsd start

Blue tooth GPSユーザーは、no-probeオプションについて詳しく調べる必要があります。

ちなみに

『xgps』 は、GpsdがあなたのGPSを確認できるかどうかチェックしてくれる良いプログラムです。

試してみること¶

GRIBプラグインを使用すると、現在の気象や波の予測データをダウンロードして、グラフ上にドレープすることができます。zyGrib software (このディスクにもあります)は、簡単にダウンロードできます。

  1. ツールバーの Options アイコンをクリックします。オプションウィンドウが表示されます。
  2. Select the Plugins tab.
  3. GRIBを選択し、 Enable をクリックします。
  4. Click OK. A windsock icon is now displayed on the toolbar.

Run through the zyGrib quickstart and load the zyGrib file over your chart.

注釈

zyGribファイルをホームディレクトリに保存した場合は、 /home に移動して、 user ディレクトリをGRIBデータディレクトリとして選択してください。次に、ファイルリストの背景を右クリックして、GRIBダウンロードファイルを含む .zygrib ディレクトリなどの隠しファイルを表示します。

ファイルを解凍する必要はありません。ファイル名の左側にある 「>」 をクリックし、利用可能な予測時間のリストから選択します。
