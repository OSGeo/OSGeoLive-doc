XyGrib クイックスタート¶

XyGrib は、標準的なGRIB配布フォーマットで天気予報データをダウンロードして視覚化するプログラムです。

目次

  • XyGribを開始する

  • データのダウンロード

  • ビューの変更

  • アニメーションを作成する

  • 次のステップ

XyGribを開始する¶

  1. デスクトップから Navigation and Maps ‣ XyGrib GRIB file viewer を選択します。

  2. 使用する言語を選択し、 OK をクリックします。プログラムが開き、世界のマップが表示されます。

  3. データサーバーへの接続をチェックしてください。画面右上のツールバーで、電源プラグアイコンをクリックします。ファイルサーバーの状態ウィンドウが開き、しばらくすると接続の状態が確認されます。

注釈

GRIB天気予報データをダウンロードするにはインターネット接続が必要です。多くのソースからGRIBデータを取得することができますが、XyGribの作者は便宜上、XyGribで使用するデータサーバーを維持管理しています。

データのダウンロード¶

  1. マウスを使用して、地図キャンバス上のボックスをクリックして、関心のある世界の一部にドラッグします。メキシコ湾やアフリカの南端にはしばしばダイナミックな風があります。

  2. 地図にボックスを描画した状態で、タスクバーのボックスズームアイコン(小さなボックスが下にある虫眼鏡)をクリックします。

  3. 再度、左クリックドラッグで対象のボックスをマークします。ただし、今回は拡大されたマップキャンバスのほとんどまたはすべてをカバーしています。

  4. File ‣ Download GRIB を選択し、ダウンロードしたい地物を選択します。よくわからない場合は、デフォルトを受け入れてください。

  5. 準備ができたら、 Download ボタンをクリックし、ファイルをディスクに保存します。

アプリケーションは、現在の時刻に設定されたタイムスライダが下部に表示された気象マップを表示します。マウスホイールを使用してズームインすることもできます。

ビューの変更¶

  1. マップの周りでマウスを動かすと、左側のペインに正確な条件が表示されます。

  2. 時間を変更するには、下部に沿ってスライダを移動するか、ウィンドウの左上にあるドロップダウンメニューから移動します。

  3. Weather map メニューから、視覚化する変数を選択します。たとえば、 Weather map ‣ Cloud cover を選択すると、ビューが変更されます。

メキシコ湾のデータをダウンロードした場合、CAPEデータのビューに興味があるでしょう。 "CAPE" は対流利用可能ポテンシャルエネルギー (Convection Available Potential Energy)の略です。これは大気の不安定性の尺度であり、高い値(1000 J/kg)は雷雨や竜巻の脅威を示します。

アフリカ南部のデータをダウンロードした場合は、雲量の方が興味深いかもしれません。

アニメーションを作成する¶

  1. ツールバーの右端のアイコン(ロケットシップ)をクリックします。新しいウィンドウが開き、アニメーションが生成されます。

  2. 最初に実行されるまで待ちます。終了すると、時系列の先頭に戻ります。

  3. 再生、一時停止、巻き戻しが可能です。

  4. 各フレームを画像ファイルとして保存して、後で 他のソフトウェア を使用してAVI、MPEG、またはアニメーションGIFにエンコードすることもできます。

  5. 終了したら、赤い電源ボタンをクリックして閉じます。

次のステップ¶
