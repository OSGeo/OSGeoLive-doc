ハードディスクに OSGeoLive をインストール¶

OSGeoLive は Ubuntu をインストールするのと同様にハードディスクにインストールして利用できます。

OSGeoLive の DVD や ISO イメージから、永続的な仮想マシンを作成するために、仮想マシン内でインストールすることもできます。

動作環境¶

  • 20 GB のハードディスクが必要となります

OSGeoLive のインストール¶

Install RELEASE を選択します

パスワード 「user」 を入力します

言語を選択します

タイムゾーンを選択します

キーボードを選択します

インストールするディスクを指定してください。(インストールされている既存のパーティションが削除される可能性があることに注意してください)。

デフォルトでは以下のような設定になっています:

  • Name = 「user」
  • username = 「user」
  • password = 「user」
  • computer name = 「osgeolive」
  • Log In automatically

Install を選択します

インストールが終わるまでしばらくお待ちください。

これで終了です。

ちなみに

インストール Tips: ユーザ名に 「user」 以外を指定してインストールした場合、Tomcat アプリケーションでは https://trac.osgeo.org/osgeo/ticket/1153 に記載のように、サービスコマンドへの権限が必要となります
