Mapbender¶

Ramverk för geoportal¶

mapbender <https://mapbender.org/en>`_ är ett avslutat OSGeo-projekt. Det är ett webbaserat geoportalramverk för att publicera, registrera, visa, navigera, övervaka och bevilja säker åtkomst till infrastrukturtjänster för spatiala data.

Mapbender är toppen på isberget, ett metalager av programvara som ger tillgång till SDI-komponenter som följer internationella standarder. Mapbender gör det möjligt för användare att enkelt skapa anpassade webbläsarklienter från ett brett utbud av widgets (utan någon/med minimal) programmering krävs. Widgets integreras med serverfunktionalitet för att tillhandahålla avancerade funktioner som säkerhetsproxy, digitalisering, auto snapping med mera.

Mapbenders hanteringsapplikationer är enkla och lättanvända och ger mycket fokuserad funktionalitet för drift av storskaliga distribuerade SDI-komponenter. Det är det självklara verktyget för SDI-administratörer som behöver underhålla, kategorisera och övervaka kart- och funktionstjänster. Mapbender ger åtkomst till individer och grupper och hanterar åtkomst från maskin till maskin. Genom att följa standardiserade tjänster, t.ex. OGC WMS, kan man dra nytta av interoperabla tjänster från en mängd olika serverplattformar.

Mapbender är baserad på Symfony, OpenLayers och jQuery.

Kärnfunktioner¶

  • Skapa enkelt gränssnitt för webbkartläggning med hjälp av mallapplikationer

    • lägga till/ta bort/anpassa applikationer

  • Skapa applikationer med digitaliserad funktionalitet

  • Skapa mobila applikationer

  • Skriv ut kartor av hög kvalitet med hjälp av dina egna PDF-mallar.

  • Create individual search configurations

  • Digitalisering via SQL

  • Ladda upp och dela OGC WMS på distans (cachelagring av funktioner)

    • ordna uppladdad WMS för överlagring

    • redigering av skiktets synlighet

    • order

    • format

  • Skapa användare och grupper, dela och ge tillgång till tjänster och moduler

  • Begränsa användaråtkomst med OWS säkerhetsproxymodul via betrodda servrar

  • Krypterad protokollövervakning och statusmeddelande samt automatisk uppdateringstjänst för OGC-fjärrtjänster

  • … och mycket mer!

Implementerade standarder¶

  • OGC Web Map Service (WMS) 1.1.1, 1.3.0

  • OGC Web Map Tile Service (WMTS) 1.0.0

  • Tile Map Service (TMS)
