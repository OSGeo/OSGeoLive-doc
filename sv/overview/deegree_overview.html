deegree¶

Om¶

deegree provides geospatial webservices based upon Open Geospatial Consortium (OGC) and INSPIRE Network Services specifications. deegree supports:

  • Web Feature Service (WFS): Serverar vektordata från geospatiala data

  • Web Map Service (WMS): Serverar kartor som återges från geospatiala data

  • OpenGIS Web Map Tile Service (WMTS): Serverar förrenderade kartplattor

  • Catalogue Service for the Web (CSW): Utför sökningar efter geospatiala dataset och tjänster

  • Web Processing Service (WPS): Exekverar geospatiala processer

Kärnfunktioner¶

WFS

  • Implementerar WFS-standarderna 1.0.0, 1.1.0 och 2.0.0

  • Official OGC reference implementation for WFS 1.0.0 (Transactional), 1.1.0 (Basic and Transactional) and WFS 2.0.0 (Basic)

  • Fullständigt transaktionell (även för rika datamodeller)

  • Stöd för KVP-, XML- och SOAP-förfrågningar

  • Supports GML 2, 3.0, 3.1, and 3.2 for both output and input

  • Stöd för GetGmlObject-förfrågningar och XLinks

  • Hög prestanda och utmärkt skalbarhet

  • Koordinatomvandling i realtid

  • Utformad för rika datamodeller från botten och upp

  • Backends stöder flexibel mappning av GML-applikationsscheman till relationsmodeller

  • Geometrimodell i enlighet med ISO 19107: Komplexa geometrier (t.ex. icke-linjära kurvor)

  • Avancerat stöd för filteruttryck baserat på XPath 1.0

  • Stöd för många backends, t.ex. PostGIS, Oracle Spatial, MS SQL Server, Shapefiler eller GML-instansdokument

WMS

  • Implementerar WMS-standarderna 1.1.1 och 1.3.0

  • Officiell OGC-referensimplementering för WMS 1.1.1 och WMS 1.3.0

  • Omfattande stöd för stylingspråken SLD/SE version 1.0.0 och 1.1.0

  • Hög prestanda och utmärkt skalbarhet

  • Rendering av hög kvalitet

  • Skalberoende styling

  • Stöd för SE eliminerar behovet av en mängd proprietära tillägg

  • Enkel konfiguration av HTML och andra utdataformat för GetFeatureInfo-svar

  • Använder strömbaserad dataåtkomst med minimalt minnesavtryck

  • Nästan fullständigt stöd för rastersymbolisering enligt definitionen i SE (med vissa tillägg)

  • Fullständigt stöd för TIME/ELEVATION och andra dimensioner för både feature- och rasterdata

  • Stöd för många backends, t.ex. PostGIS, Oracle Spatial, Shapefiler eller GML-instansdokument

  • Kan rendera rika datamodeller direkt

WMTS

  • Implementerar grundläggande WMTS-standard 1.0.0 (KVP)

  • Official OGC reference implementation for WMTS 1.0.0

  • Hög prestanda och utmärkt skalbarhet

  • Stöd för olika backends, t.ex. GeoTIFF, fjärr-WMS eller filsystemets bildhierarkier

  • Stöd för on-the-fly-caching (med hjälp av EHCache)

  • Stöd för GetFeatureInfo för fjärrstyrda WMS-backends

CSW

  • Implementerar CSW-standard 2.0.2

  • Fullständigt transaktionell

  • Stöd för KVP-, XML- och SOAP-förfrågningar

  • Hög prestanda och utmärkt skalbarhet

  • ISO-profil för tillämpning av metadata 1.0.0

  • Pluggable and modular data access layer allows to add support for new APs and backends

  • Modulär inspektörsarkitektur gör det möjligt att validera poster som ska infogas mot olika kriterier

  • Standardkontrollanter: schemavaliditet, identifieringsintegritet, Inspire-krav

  • Hanterar alla definierade sökbara egenskaper (för både Dublin Core och ISO-profilen)

  • Komplexa filteruttryck

WPS

  • Implementerar WPS-standard 1.0.0

  • Stöd för KVP-, XML- och SOAP-förfrågningar

  • Pluggbart lager för processleverantör

  • Användarvänligt API för implementering av Java-processer

  • Stöd för alla varianter av inmatnings-/utmatningsparametrar: literal, bbox, komplex (binär och xml)

  • Streamingåtkomst för komplexa inmatnings-/utmatningsparametrar

  • Bearbetning av stora datamängder med minimalt minnesavtryck

  • Stöd för lagring av svarsdokument/utdataparametrar

  • Stöder inmatningsparametrar som anges inline och som referens

  • Stöd för RawDataOutput/ResponseDocument-svar

  • Stöd för asynkron exekvering (med polling av processstatus)

Implementerade standarder¶

  • Web Feature Service (WFS) 1.0.0, 1.1.0 och 2.0.0

  • Web Map Service (WMS) 1.1.1 och 1.3.0

  • OpenGIS Web Map Tile Service (WMTS) 1.0.0

  • Catalogue Service for the Web (CSW) 2.0.2

  • Web Processing Service (WPS) 1.0.0

  • Geography Markup Language (GML) 2.12, 3.0.1, 3.1.1 och 3.2.1
