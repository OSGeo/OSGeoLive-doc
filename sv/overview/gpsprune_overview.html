GpsPrune¶

Redigering av GPS-spår¶

GpsPrune är ett verktyg för visning, redigering och konvertering av koordinatdata från GPS-system. Det kan användas för att planera resor i förväg eller för att analysera de spår som registreras av GPS-mottagare.

Det använder OpenStreetMap-bilder för att visa inspelade spår och waypoints och tillhandahåller en mängd olika verktyg så att du kan redigera, beskära och beskära datapunkterna. Det finns också funktioner för att korrelera foton och ljudfiler till koordinaterna med hjälp av filernas tidsstämplar.

GpsPrune använder GPSBabel, om det finns tillgängligt, för att läsa data direkt från en GPS-mottagare och kan även skicka tillbaka de redigerade data till mottagaren.

GpsPrune körs på Mac OSX, Windows och Linux och finns tillgänglig på över tio olika språk.

Kärnfunktioner¶

  • Ladda data från textfiler, gpx-filer, kml/kmz-filer eller direkt från en GPS-mottagare

  • Zoombara kartbilder från OpenStreetMap eller annan kompatibel server (t.ex. HikeBikeMap, Reit- und Wanderkarte etc.)

  • Visning av profil för höjd eller hastighet

  • Redigeringsverktyg för att skapa, modifiera, radera, backa och komprimera spår

  • Interaktiv 3D-vy av data eller export till POV-format för 3D-rendering, båda inklusive kartbilder målade på terrängen

  • Spara data till textfiler, gpx-filer, kml/kmz-filer eller direkt till en GPS-mottagare

  • Automatisk korrelation av fotografier och ljudfiler med inlästa data med hjälp av tidsstämplar

  • Skapande av diagram, t.ex. höjd mot avstånd, hastighet mot tid

  • Avstånd, stigningar, nedförsbackar, tidsuppskattningar och mycket mer

  • Online-resurser som Wikipedia, GPSies, OpenCaching.de och lokala väderprognoser
