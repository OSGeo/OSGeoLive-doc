QGIS¶

Desktop GIS¶

QGIS är en användarvänlig GIS-klient med öppen källkod där du kan visualisera, hantera, redigera och analysera data samt skapa utskrivbara kartor. Den innehåller kraftfulla analytiska funktioner genom integration med GRASS, SAGA, Orfeo Toolbox , GDAL/OGR och många andra algoritmleverantörer. Det körs på Linux, Unix, Mac OSX och Windows och stöder många vektor-, raster- och databasformat och funktionalitet.

Kärnfunktioner¶

  • Användarvänligt grafiskt användargränssnitt:

    • identifiera/välja funktioner

    • redigera/visa/söka attribut

    • projektion i realtid

    • skriv ut kompositör

    • märkning av funktioner

    • ändra vektor- och rastersymboler

    • lägg till ett graticule-lager

    • och mer …

  • Enkel visning av många vektor- och rasterformat

    • databastabeller: PostgreSQL/PostGIS, Oracle Spatial, MS SQL Spatial, SpatiaLite

    • de flesta vektorformat: inklusive ESRI shapefiler, MapInfo, SDTS och GML, OpenStreetMap-vektorer

    • rasterformat såsom digitala höjdmodeller, flygfotografier eller landsat-bilder

    • GRASS-platser och kartor

    • spatiala data online som serveras som OGC-kompatibla WMS, WMTS, WFS eller WCS

    • 2.Stöd för 5D-rendering

  • Skapa, redigera och exportera spatiala data med hjälp av:

    • digitaliseringsverktyg för GRASS- och shapefilformat

    • plugin för georeferenser

    • GPS-verktyg för import och export av GPX-format, konvertering av andra GPS-format till GPX eller nedladdning/uppladdning direkt till en GPS-enhet

  • Utföra spatial analys med hjälp av integrerat stöd för SAGA, OTB, MMGIS och GRASS

    • karta algebra

    • terränganalys

    • hydrologisk modellering

    • nätverksanalys

    • omfattande sökbar verktygslåda av funktioner

    • definiera och köra återanvändbara analysrutiner genom att kedja funktioner med hjälp av ett grafiskt modelleringsverktyg

    • och många andra

  • Publicera på internet

  • Utökad plugin-arkitektur

Implementerade standarder¶

  • Överensstämmer med OGC-standarder (WMS, WFS, … )
