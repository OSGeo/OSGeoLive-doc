OpenStreetMap¶

Verktyg för att kartlägga världen¶

openStreetMap <https://www.openstreetmap.org>`_ är ett projekt som syftar till att skapa och tillhandahålla gratis geografiska data, t.ex. gatukartor, till alla som vill ha dem. Projektet startades eftersom de flesta kartor som du tänker på som fria faktiskt har juridiska eller tekniska begränsningar för deras användning, vilket hindrar människor från att använda dem på kreativa, produktiva eller oväntade sätt.

Ett antal program har inkluderats: JOSM, Merkaartor, Osmosis, Osm2pgsql och ett antal små kommandoverktyg. Dessutom ingår aktuella exempeldata för FOSS4G-värdstaden och ett antal JOSM-plugins.

Kärnfunktioner¶

  • Globalt dataset med massiv insamling från massor av människor

  • Ett antal redaktörer att välja mellan

  • Ett antal renderingsmaskiner att välja mellan (inklusive anpassade och online)

  • Gratis att använda

  • Utbredd integration av geoapplikationer
