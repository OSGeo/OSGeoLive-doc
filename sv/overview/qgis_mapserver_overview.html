QGIS-server¶

Webbtjänst¶

QGIS Server tillhandahåller webbkartor och funktionstjänster (WMS & WFS) med samma bibliotek som skrivbordsprogrammet för GIS (QGIS). Kartor och utskriftsmallar som skapats i QGIS desktop kan publiceras som webbkartor genom att helt enkelt kopiera QGIS-projektfilen till serverkatalogen. De resulterande webbkartorna ser exakt likadana ut som i skrivbordet. QGIS Server körs vanligtvis som en CGI/FastCGI-modul i Apache Webserver.

Kärnfunktioner¶

  • Webbkarttjänst (WMS) - bilder

  • OGC Web Map Tile Service (WMTS) 1.0.0

  • Web Feature Service (WFS) - vektorer

  • Inbyggd PDF-utmatning

  • Skapa WYSIWYG-kartor med QGIS

  • Avancerad kartografisk symbolisering

  • Stöd för SLD (Styled Layer Descriptor)

  • Extra parametrar som t.ex. FILTER, URVAL, DPI

Implementerade standarder¶

  • OGC Web Map Service (WMS) 1.1.1 och 1.3.0

  • OGC Web Map Tile Service (WMTS) 1.0.0

  • OGC Web Feature Service (WFS) 1.0.0 och Transactional WFS (WFS-T) 1.0.0

  • OGC Webbtäckningstjänst (WCS)

  • OGC:s beskrivare av stiliserade lager (SLD) 1.0
