pycsw¶

Metadatakatalog¶

pycsw gör det möjligt för användare att publicera en katalog med beskrivningar av dataset (metadata) på webben, med hjälp av standardbaserade gränssnitt och format, så att den kan efterfrågas och uppdateras av katalogklienter. pycsw är enkelt att installera i en Apache-webbserver (eller någon WSGI-server) och kan komma åt metadataposter som lagras som XML-filer i en databas.

Metadata är ”data om data” och lagrar sådant som skapandedatum, författare, titel, intresseområde etc. och kodas vanligtvis som XML-filer enligt internationella standarder som ISO 19115, ISO 19139 och FGDC CSDGM.

pycsw är en Python-implementering av Catalogue Service for the Web (CSW). Projektet är certifierat OGC Compliant, och är en OGC Reference Implementation. CSW-standarden definierar gemensamma gränssnitt för att upptäcka, bläddra och fråga metadata om spatiala data, webbtjänster och relaterade resurser.

pycsw är Open Source, släppt under en MIT-licens och körs på alla större plattformar (Windows, Linux, Mac OS X).

Kärnfunktioner¶

  • certifierad OGC-kompatibel och OGC-referensimplementering för OGC CSW 2.0.2 och CSW 3.0.0

  • avverkningsstöd för WMS, WFS, WCS, WPS, WAF, CSW, SOS

  • implementerar INSPIRE Discovery Services 3.0

  • implementerar ISO Metadata Application Profile 1.0.0

  • implementerar FGDC CSDGM Application Profile för CSW 2.0

  • implementerar sökprotokollet Search/Retrieval via URL (SRU)

  • implementerar funktioner för fulltextsökning

  • implementerar OGC OpenSearch Geo/Time-tillägg

  • implementerar Open Archives Initiative Protocol for Metadata Harvesting

  • stödjer metadatamodellerna ISO, Dublin Core, DIF, FGDC, Atom och GM03

  • CGI- eller WSGI-distribution

  • Kompatibel med Python 2 och 3

  • enkel konfiguration

  • transaktionella förmågor (CSW-T)

  • flexibel konfiguration av förrådet

  • GeoNode-anslutning

  • HHypermap-anslutning

  • Anslutning till Open Data Catalog

  • CKAN-anslutning

  • federerad katalog distribuerad sökning

  • xML Schema-validering i realtid

  • utdragbar arkitektur för profilplugin

  • mycket lättviktig (Python + CGI)

Implementerade standarder¶

OGC-standarder:

  • Catalogue Service for the Web (CSW) 2.0.2, 3.0.0

  • Filter Encoding (FE) 1.1.0, 2.0.0

  • OWS Common 1.0.0, 2.0.0

  • Geography Markup Language (GML) 3.1.1

  • Simple Feature Access (SFS) 1.2.1

  • OGC OpenSearch 1.0

Standarder för metadata:

  • Dublin Core 1.1

  • SOAP 1.2

  • ISO 19115:2003

  • ISO 19139:2007

  • ISO 19119:2005

  • NASA DIF 9,7

  • FGDC CSDGM 1998

  • GM03 2.1

  • SRU 1.1

  • OAI-PMH 2.0

  • A9 OpenSearch 1.1
