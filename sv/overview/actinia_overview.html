actinia¶

Geospatial bearbetning med hög prestanda¶

Actinia är ett REST API med öppen källkod för skalbar, distribuerad, högpresterande bearbetning av geografiska data som huvudsakligen använder GRASS GIS för beräkningsuppgifter.

Det tillhandahåller ett REST API för att bearbeta satellitbilder, tidsserier av satellitbilder, godtyckliga rasterdata med geografiska relationer och vektordata.

Kärnfunktioner¶

  • REST API för att bearbeta satellitbilder, tidsserier av satellitbilder, raster- och vektordata.

  • Utforska ”handledningen” <https://actinia-org.github.io/actinia-core/>`_.
