GeoNod¶

System för hantering av geospatialt innehåll¶

GeoNode är ett Content Management System för geospatiala data som möjliggör skapande, delning och kollaborativ användning av geospatiala data. Dataset kan laddas upp i olika format, kartor kan redigeras, stylas och aggregeras via webbläsarbaserade verktyg, kartor och metadata kan publiceras och sökas, och recensioner, betyg och kommentarer kan samlas in från användare.

GeoNode bygger på: GeoServer, GeoExplorer, pycsw, Django och GeoExt.

Kärnfunktioner¶

  • Upptäckt av spatiala data

    • Kraftfull spatial sökmotor

    • Federerade OGC-tjänster

    • Katalog med metadata

  • Importera och hantera geospatiala data

    • Publicera raster-, vektor- och tabelldata

    • Hantera metadata och tillhörande dokument

    • Dela data på ett säkert eller offentligt sätt

    • Versionerad redigerare för geospatiala data

  • Interaktiv kartläggning

    • GeoExplorer GIS-klient

    • Grafisk stilredigerare

    • Skapa interaktiva kartor med flera lager

    • Dela och bädda in kartor i webbsidor

    • Skriv ut kartor som PDF

  • Samarbete

    • Granska, betygsätta och kommentera data

    • Grupper av användare

    • Aktivitetsströmmar

    • Tillkännagivanden och meddelanden

  • … och mycket mer!

Implementerade standarder¶

  • Web Map Service (WMS)

  • Catalogue Service for the Web (CSW)

  • Web Feature Service (WFS)

  • Web Coverage Service (WCS)

  • Filter Encoding (FE)

  • Styled Layer Descriptor (SLD)

  • Geography Markup Language (GML)
