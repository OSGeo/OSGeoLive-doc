SpatiaLite¶

Spatial databas¶

SpatiaLite är en SQLite-databasmotor med spatiala funktioner.

SQLite är ett databashanteringssystem (DBMS) som är enkelt, robust, lättanvänt och mycket lättviktigt. Varje SQLite-databas är helt enkelt en fil. Du kan fritt kopiera den, komprimera den och porta den mellan Windows, Linux, MacOs etc.

Kärnfunktioner¶

SpatiaLite-tillägget gör det möjligt för SQLite att stödja spatiala data som överensstämmer med OGC-specifikationer.

Höjdpunkter:

  • Användarvänligt GUI

  • Standard WKT- och WKB-format

  • SQL spatiala funktioner som AsText(), GeomFromText(), Area(), PointN() …

  • OpenGis spatiala analysfunktioner som tillhandahålls via GEOS, t.ex. Overlaps(), Touches(), Union(), Buffer() …

  • Fullständiga spatiala metadata i enlighet med OpenGis-specifikationerna

  • Många geometriska beteckningar - EWKT, GML, KML och GeoJSON

  • Importera och exportera till shapefiler

  • Återprojicering av koordinater via PROJ.4 och EPSG:s dataset med geodetiska parametrar

  • Lokalt teckensnitt via GNU libiconv

  • Spatialt index baserat på SQLites RTree-tillägg

  • Åtkomst till shapefiler som VIRTUELLA TABELLER, vilket möjliggör standard SQL-frågor på externa shapefiler utan att importera eller konvertera dem

  • Få tillgång till externa CSV/TxtTab-filer eller xls-kalkylblad som VIRTUELLA TABELLER

  • Åtkomst till XML-dokument, lagrade BLOB-komprimerade binära objekt, inklusive syntaktiska ”välformade” och XSF-schema valideringsbegränsade kontroller. Specifikt stöd för ISO-Metadata, SLD/SE-stilar och SVG-grafik. XML-dokument kan efterfrågas med hjälp av standard XPath-syntax.

  • Fråga efter externa WFS-servrar.

  • Analysera externa DXF-filer (alla versioner) och lagra lager och geometrier som hittas.

  • Generera och exportera DXF-filer
