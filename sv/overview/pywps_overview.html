PyWPS¶

Tjänst för webbbearbetning¶

PyWPS ger webbåtkomst till anpassade geospatiala operationer (kallas processer) via standarden Web Processing Service (WPS).

Processerna är skrivna på programmeringsspråket Python och kan integrera verktyg som GRASS GIS, R, GDAL/OGR, proj och andra bibliotek med Python-bindningar. Källdata kan tillhandahållas med WPS-begäranden eller hämtas från data på WPS-servern.

Kärnfunktioner¶

  • liten och lätt

  • enkel och snabb att installera och ställa in

  • integreras sömlöst med Pythons ekosystem av paket:

    • R

    • Shapely
