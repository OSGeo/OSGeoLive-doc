SAGA¶

Desktop GIS¶

SAGA (System for Automated Geoscientific Analyses) är ett geografiskt informationssystem (GIS) med öppen källkod som används för att redigera och analysera spatiala data. Det innehåller ett stort antal moduler för analys av vektor- (punkt, linje och polygon), tabell-, rutnäts- och bilddata. Paketet innehåller bl.a. moduler för geostatistik, bildklassificering, projektioner, simulering av dynamiska processer (hydrologi, landskapsutveckling) och terränganalys. Funktionaliteten kan nås via ett GUI, kommandoraden eller genom att använda C++ API.

SAGA har varit under utveckling sedan 2001, och centrum för SAGA-utvecklingen ligger vid Institute of Geography vid Hamburgs universitet, med bidrag från den växande världsomspännande gemenskapen.

Kärnfunktioner¶

  • Tillgång till ett stort antal vetenskapliga moduler via det grafiska användargränssnittet eller kommandoraden

  • Importera/exportera till olika filformat

  • Reprojicering/omsampling av data

  • Manipulering av vektordata (sammanslagning/korsning/attribut)

  • Manipulering av punktmoln från lidardata

  • Rasterdata: interpolering, kostnadsanalys, …

  • Bildanalys: filter, kantdetektering, klusteranalys, segmentering

  • Digital Terrain Analysis: generera geomorfometriska index, kanalnätverk, profiler, konturlinjer, …

  • Geostatistik: moduler för variogramanpassning och kriging

  • Snabbt användargränssnitt för hantering och visualisering av data

  • 3D visualization

  • Enkelt C++ API för att skapa nya moduler

  • Skriptning via kommandoraden, bindningar för Python

  • SAGA kan nås från det statistiska språket R via RSAGA-modulen
