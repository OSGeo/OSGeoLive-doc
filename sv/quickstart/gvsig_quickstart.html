snabbstart för gvSIG Desktop¶

gvSIG är ett geografiskt informationssystem (GIS), dvs. ett skrivbordsprogram som är utformat för att fånga, lagra, hantera, analysera och distribuera alla typer av geografisk information för att lösa komplexa förvaltnings- och planeringsproblem. gvSIG är känt för att ha ett användarvänligt gränssnitt och kan hantera de vanligaste formaten, både vektor- och rasterformat. Det har ett brett utbud av verktyg för att arbeta med geografisk information (frågeverktyg, layoutskapande, geoprocessing, nätverk etc.), vilket gör gvSIG till det perfekta verktyget för användare som arbetar inom markområdet.

Observera

Om gvSIG:s användargränssnitt inte svarar, prova Alt+f för att öppna filmenyn till exempel. Användargränssnittet bör fungera igen, då. Den här buggen inträffar bara i gvSIG-distributionen för OSGeoLive. Det visas inte i en normal gvSIG-installation.

Observera

Anmärkning för version gvSIG 2.3.1: För att ladda rasterfiler måste användaren ha skrivbehörighet i den mapp där filen finns. Annars kommer programmet att fråga efter en giltig mapp (med skrivbehörighet). Eftersom användaren ”user” inte har skrivbehörighet i mappen ”data”, måste man vid förfrågan välja en mapp där användaren har skrivbehörighet, t.ex. ”/home/user”.

Starta gvSIG och skapa en vy¶

  1. Välj gvSIG från applikationsmenyn, Geospatial ‣ Desktop GIS ‣ gvSIG desktop. Det tar vanligtvis ungefär en minut att starta programmet.

  2. Som standard öppnas gvSIG med en tom projektfil med dokumenttypen ”View” vald. På OSGeoLive har dock gvSIG konfigurerats för att starta med ett standardprojekt. Klicka på knappen New Project eller gå till File ‣ New Project. När du blir tillfrågad kan du välja Discard Changes eftersom vi inte kommer att använda det senaste projektet.

  3. Klicka på knappen New för att skapa en vy. Den kommer att öppnas automatiskt.

Definiera projektionen av din vy¶

  1. Välj View ‣ Properties.

  2. För Current projection, klicka på …-knappen. Detta öppnar dialogrutan Ny CRS.

  3. Använd rullgardinsmenyn för Type och välj EPSG.

  4. Ange EPSG-koden för det koordinatbaserade referenssystemet. I det här exemplet använder vi EPSG=4326, som är WGS84 2D Geographic Projection. Klicka sedan på knappen Search. Då listas det valda CRS i tabellen under inmatningsfältet för sökningen.

  5. Välj OK för att återgå till dialogen View Properties. EPSG-koden är nu 4326 och kartans enheter har ändrats till grader.

  6. Välj OK för att återgå till Project Manager.

    1. Visningsfönstret består av tre zoner.

  • (2) The top-left cell contains a list of vector or raster layers being used in the view (i.e. Table-of-Contents).

  • (3) The bottom-left cell displays the extent of the main view over a selected vector file.

  • (4) The right cell is the main display area where raster and vector data is rendered.

Lägg till lager i vyn¶

  1. Klicka på ikonen Add layer i huvudverktygsfältet för att öppna dialogrutan Add layer.

  2. Välj Add i dialogrutan Add layer. Som standard är den första fliken File så dialogrutan Open file kommer att visas.

  3. Borra ner till katalogen /usr/local/share/data/natural_earth2/.

  4. Välj filerna som visas på bilden.

Observera

Som standard visas alla format, så vi kan välja både vektor- och rasterfiler åt gången genom att hålla ned CTRL-tangenten.

  1. Klicka på knappen OK för att återgå till dialogrutan Lägg till lager.

  2. Klicka på knappen OK i dialogrutan Add layer för att återgå till vyn.

Konfigurera visningsdisplayen¶

  1. Klicka på ikonen Maximera för att få en bättre visning av data.

  2. Storleken på kolumnen Innehållsförteckning kan justeras med musen för att göra det lättare att se långa filnamn.

  3. Om rasterlagret ligger högst upp ska det flyttas manuellt till botten.

Zooma in i en region¶

  1. Som standard är verktyget Zoom in valt i huvudverktygsfältet.

  2. Använd muspekaren för att rita en avgränsningsbox runt sydöstra Australien. Gör detta genom att markera det övre vänstra hörnet av begränsningsrutan, hålla ned vänster musknapp och dra det nedre högra hörnet av begränsningsrutan över det markerade området. Släpp vänster musknapp när du har definierat det ungefärliga området.

Ändra hur lagren ser ut¶

Gör en polygon transparent¶

  1. Välj skiktet ne_10m_admin_1_states_provinces.shp med vänster musknapp och klicka igen med höger musknapp.

  2. Välj undermenyn Properties.

  3. Dialogrutan Layer properties visas.

Ändra symbologin för ett vektorlager¶

  1. Välj fliken Symboler högst upp i dialogrutan Layers egenskaper.

  2. Välj Välj symbol-knappen så att dialogrutan Symbolväljare visas.

Ta bort fyllningssymbolen och ändra kantfärgen¶

  1. Avmarkera rutan Fill color så att polygonerna blir genomskinliga.

  2. Klicka på … i närheten av Border color:.

  3. Välj vit som färg.

  4. Klicka på knappen OK för att återgå till dialogrutan Symbolväljare.

  5. Klicka på knappen OK för att återgå till dialogrutan Layers egenskaper.

Ändra märkningen av ett vektorlager¶

  1. Välj fliken Labelling högst upp i dialogrutan Layers egenskaper.

  2. Ändra textattributen till de som visas på bilden och klicka sedan på knappen Accept.

Se resultatet¶

Observera att detta är en mycket grundläggande vy som visar en punkt, en linje och en polygon vektorfil överlagrad på en rasterfil. Det är lika enkelt att ha ett flygfoto eller en digital terrängmodell som bakgrund till dina vektordata, eller att visa andra vektordata som lagras i olika format.

Lägg till en översikt eller sökfunktion¶

Välj Vy ‣ Locator setup.

  1. Klicka på knappen Add layer….

  2. I fönstret Add layer klickar du på knappen Add.

  3. Välj skiktet HYP_50M_SR_W.tif.

  4. Klicka på knappen Open för att återgå till fönstret Add layer.

  5. Klicka på knappen Ok och stäng fönstret Configure locator.

Nu kan vi enkelt få reda på var utsikten finns i världen.

Välj städer inom en region¶

Välj region¶

  1. Välj lagret ne_10m_admin_1_states_provinces.shp med hjälp av vänster musknapp.

  2. Välj ikonen Select by point i huvudverktygsfältet.

  3. Klicka på den polygon som representerar delstaten New South Wales. Polygonen blir gul eller får någon annan färg beroende på dina användarinställningar.

Öppna verktyget Urval efter lager¶

  1. Välj lagret ne_10m_populated_places.shp med hjälp av vänster musknapp.

  2. Välj Selection ‣ Select by layer för att öppna dialogrutan Selection by Layer

Välj de städer som ingår i en region¶

  1. Ändra det första urvalskriteriet med hjälp av rullgardinsmenyerna på vänster sida i dialogrutan Selection by Layer enligt bilden.

  2. Ändra det andra urvalskriteriet som visas på bilden.

  3. Klicka på knappen New set för att välja städer inom den valda polygonen.

  4. Välj knappen Cancel i dialogrutan Selection by Layer för att återgå till vyn.

Avmarkera polygonen så att du kan se vad som har hänt¶

  1. Välj lagret ne_10m_admin_1_states_provinces.shp med hjälp av vänster musknapp.

  2. Välj ikonen ”Rensa urval” i huvudverktygsfältet.

  3. Du kan nu se att endast de städer som ligger i New South Wales är valda.

Exportera utvalda funktioner till en ny shapefil¶

Välj format för utdatafilen¶

  1. Välj lagret ne_10m_populated_places.shp med hjälp av vänster musknapp.

  2. Välj Layer ‣ Export to… för att starta exporten.

  3. Välj Shape Format.

  4. Klicka på Nästa.

Välj namn på utdatafilen¶

  1. Välj en mapp och skriv in filnamnet.

  2. Klicka på Öppna och sedan på Nästa.

Referenssystem¶

  1. Välj alternativet Original (EPSG:4326) för att behålla referenssystemet för vyn.

  2. Klicka på Nästa.

Alternativ för export¶

  1. Välj alternativet Selected features för att endast exportera städerna i New South Wales.

  2. Klicka på Export.

Se resultatet¶

  1. Det nya lagret har lagts till i vyn.

  2. För att kontrollera att exporten var ok ställde du in originalfilen som osynlig. Endast städerna i New South Wales ska visas.

Skapa en ny karta¶

  1. Välj dokumenttypen Map i Project Manager.

  2. Klicka på knappen New för att skapa en karta.

  3. Ett nytt fönster visas där du kan välja den vy som ska infogas på kartan. Välj den enda vy som du har och tryck på Accept.

  4. Kartan visas med den vy som valts.

Arbeta med kartan¶

  1. Ytterligare element som skala och norrpil kan läggas till på kartan med hjälp av ikonerna i huvudverktygsfältet eller med undermenyerna i menyn Map.

  2. Kartan kan skrivas ut eller exporteras till PDF eller Postscript för att integreras i andra arbeten.

  3. Välj ikonen Stäng fönstret för att återgå till Projektledaren

Spara ditt projekt eller avsluta gvSIG¶

  1. Projekt kan sparas för senare användning genom att använda menyalternativet File ‣ Save as…, eller

  2. Projekt kan avslutas eller stängas genom att använda menyalternativet File ‣ Exit.

Installera ett tillägg¶

Öppna tilläggshanteraren¶

  1. Välj Tools ‣ Addons manager. Efter ett tag kommer fönstret Installera paket att visas.

  2. I fönstret Install package väljer du alternativet Installation från URL.

  3. Välj den URL för repot som visas på bilden.

  4. Klicka på knappen Nästa.

Hitta ett tillägg¶

  1. I tilläggshanteraren hittar vi plugins och symbolbibliotek.

  2. Skriv ”Forestry” i textrutan Fast filter.

  3. Kolla in tillägget som heter Symboler: Forestry (det är ett symbolbibliotek).

  4. Klicka på Nästa.

Slutför installationen¶

  1. Klicka på Starta nedladdning.

  2. Klicka sedan på Nästa.

  3. Klicka slutligen på Finish.

Även om det nya meddelandet anger att en omstart behövs, är det inte nödvändigt för symbolbibliotek. Vi behöver bara starta om när vi installerar plugins.

Lägg till ett OpenStreetMap-lager¶

  1. Nu ska vi lägga till OpenStreetMap-kartografi. I en ny vy, i EPSG 3857, klickar du på Add layer…-knappen.

  2. Välj fliken OSM.

  3. Välj ”Mapnik” bland de tillgängliga lagren.

  4. Klicka på Ok.

Se resultatet¶

Ett nytt lager har lagts till i vyn. Genom att bara zooma över en region kan vi se detaljerade data från Open Street Map.

Lägg till ett WMTS-lager¶

WMTS är en vidareutveckling av WMS OGC-standarden baserad på kakelhantering.

Välj en server¶

  1. I en vy klickar du på knappen Add layer….

  2. Välj fliken WMTS.

  3. Välj den URL som visas på bilden.

  4. Klicka på Connect.

  5. Klicka på Nästa.

Välj ett av de tillgängliga lagren¶

  1. Välj ett av de tillgängliga lagren (t.ex. bluemarble)

  2. Klicka på Add.

  3. Klicka på Nästa.

Välj bildformat och referenssystem¶

  1. Välj image/jpeg som bildformat.

  2. Välj EPSG:4326 (observera att vyn måste ha samma SRS. Om inte, avbryt, ändra och börja om).

  3. Klicka på Accept.

Ett nytt lager har lagts till i vyn.
