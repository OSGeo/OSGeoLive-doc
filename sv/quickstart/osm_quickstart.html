Snabbstart för OpenStreetMap¶

OpenStreetMap är en gratis karta över världen på gatunivå, skapad av en ständigt växande grupp kartritare. Vem som helst kan redigera OpenStreetMap. Webbplatsen Learn OSM <https://learnosm.org>`_ innehåller lättförståeliga steg-för-steg-guider som hjälper dig att komma igång med att bidra till OpenStreetMap, använda OpenStreetMap och använda OpenStreetMap-data. Learn OSM-guiden på länken ovan visar steg för steg hur du navigerar på OpenStreetMap-webbplatsen, visar och skriver ut kartor och registrerar ett användarkonto. När du har ditt eget användarnamn och lösenord kan du lägga till dina första punkter på kartan, lära dig hur du gör gatuundersökningar och arbeta med flygbilder.

I resten av den här guiden presenteras några av de online- och offlineprogram som används för att redigera och rendera OSM-data.

Program som arbetar med OSM-data¶

iD¶

iD är standard online-redigerare för OpenStreetMap (OSM). iD är snabb och enkel att använda och möjliggör kartläggning från olika datakällor, t.ex. markundersökningar, satellit- och flygbilder, GPS-spår eller bilder på gatunivå.

  • Startsida: https://www.openstreetmap.org/edit

  • Dokumentation: https://wiki.openstreetmap.org/wiki/ID

  • Tutorial: https://learnosm.org/en/beginner/id-editor/

JOSM¶

JOSM (Java OpenStreetMap Editor) är en avancerad redigerare för OpenStreetMap (OSM) skriven i Java. Den är kraftfullare och mer komplex än iD, kan användas för massredigering, offline eller på en opålitlig internetanslutning. Den kan utökas med tillägg, anpassade förinställningar och ytterligare tillägg.

  • Hemsida: https://josm.openstreetmap.de

  • Användarhandbok: https://wiki.openstreetmap.org/wiki/JOSM/Guide

  • Tutorial: https://learnosm.org/en/josm/start-josm/

  • JOSM plugin-samling: https://josm.openstreetmap.de/wiki/Plugins

QGIS¶

OpenStreetMap-data kan laddas in i QGIS som vektorlager, som en kärnfunktionalitet eller genom plugins som QuickOSM.

  • QuickOSM: Se avsnittet QuickOSM

  • Onlinehjälp: https://wiki.openstreetmap.org/wiki/QGIS#QGIS_2:_Importing_OSM_vector_layers

  • Tutorial: https://learnosm.org/en/osm-data/osm-in-qgis/

Osmos¶

Osmosis är ett mycket kapabelt verktygsprogram för att utföra många uppgifter på en rå nivå på OSM-data. Detta inkluderar dataimport och export till databaser, sortering, rengöring och skapande av datadumpar.

  • Dokumentation: https://wiki.openstreetmap.org/wiki/Osmosis#Usage

  • Tutorial: https://learnosm.org/en/osm-data/osmosis/

osm2pgsql¶

** osm2pgsql ** är ett verktygsprogram som konverterar OpenStreetMap (.osm) -data till ett format som kan laddas i PostgreSQL (PostGIS). Det används ofta för att göra OSM-data visuellt med Mapnik, eftersom Mapnik kan fråga PostgreSQL för kartdata, men fungerar inte direkt med OSM-filer.

  • Hemsida: https://wiki.openstreetmap.org/wiki/Osm2pgsql

  • Tutorial: https://learnosm.org/en/osm-data/osm2pgsql/

Arbeta med OSM-data i QGIS¶

OSGeoLive levereras med OSM-data för staden där den globala FOSS4G-konferensen äger rum. Du hittar data i filsystemet på /usr/local/share/data/osm/ eller i PostgreSQL-databasen osm_local.

Starta QGIS och ta en titt på data.

  1. Öppna mappen Desktop GIS.

  2. Dubbelklicka på ikonen QGIS.

  3. Gå till webbläsarpanelen till vänster på skärmen.

  4. Navigera till PostGIS och sedan till databasen osm_local. Du hittar data i schemat public.

  5. Dra lagren från Browser-panelen och släpp dem i kartbilden.

Du hittar en introduktion till QGIS i QGIS Quickstart

QuickOSM¶

med ”QuickOSM” kan du arbeta snabbt med OSM-data i QGIS tack vare Overpass API.

  • Hemsida: https://github.com/3liz/QuickOSM

  1. Du måste aktivera pluginet i QGIS. Sök efter QuickOSM från Plugins –> Hantera och installera plugins i QGIS

  2. Öppna QuickOSM från Vector –> QuickOSM –> QuickOSM

  3. Sök efter alla pubar i FOSS4G-staden

  4. Välj nyckel amenity, värde pub, i Buenos Aires

  5. Visa frågan för att se Overpass-frågan

  6. Klicka på Kör fråga. De resulterande data kommer att laddas in i ditt QGIS-projekt som temporala data

  7. Du kan nu spara data via Export -> Save as….. Välj t.ex. Geopackage och spara data som buenos_aires_pubs.

Tips

Du hittar också många OSM-tillägg för QGIS. Sök efter OSM i tilläggsförrådet.

Använda OSM som baskarta i QGIS¶

XYZ Plattor¶

Som standard levereras QGIS med OpenStreetMap baskartlager och det finns i Browser Panel under XYZ Tiles.

QuickMapTjänster¶

plugin QuickMapServices gör att du enkelt kan lägga till olika karttjänster som baskartor med ett klick i QGIS.

  • Hemsida: https://github.com/nextgis/quickmapservices

  1. Du måste aktivera pluginet i QGIS. Sök efter QuickMapServices från Plugins –> Hantera och installera plugins i QGIS

  2. Lägg till en OSM-baskarta från Web –> QuickMapServices –> OSM

Börja arbeta med OSM-data¶

Det finns flera sätt att ladda ner OpenStreetMap-data för din region. Du kan få data i flera olika format.

Du kan använda följande verktyg och tjänster för att extrahera OSM-data för ditt intresseområde.

Geofabrik¶

Geofabrik tillhandahåller dagligen världsomspännande OpenStreetMap-datautdrag per underregion och land. Välj den region eller det land du är intresserad av. Data tillhandahålls i tre format: .osm.pbf, .shp.zip och .osm.bz2. .osm.pbf och .shp.zip kan öppnas direkt i QGIS.

Tips

.osm.bz2 innehåller råa OSM XML-filer .osm, osm2pgsql fungerar bra med dem, ta reda på mer om hur du arbetar med dem här https://wiki.openstreetmap.org/wiki/OSM_XML

Övergång turbo¶

Overpass turbo är ett webbaserat datautvinningsverktyg för OpenStreetMap baserat på Overpass-API.

Ta en titt på webbklienten och ladda ner alla pubar i FOSS4G-staden.

  1. Öppna https://overpass-turbo.eu/

  2. Sök efter alla pubar i FOSS4G-staden

  3. Klicka på knappen Wizard och klistra in frågan i sökkolumnen amenity=pub i ”Buenos Aires”

  4. Klicka sedan på knappen ”Bygg och kör fråga”

  5. Klicka på ”Zooma till data” i kartbilden för att navigera till dina resultat

HOT Exportverktyg¶

Humanitarian OpenStreetMap Team (HOT) tillhandahåller också ett exportverktyg för OpenStreetMap. För att använda det behöver du ett OSM-konto. Du kan ladda ner data i flera format och ha historik för dina nedladdade data och exporter.

Här är en snabbstart: https://export.hotosm.org/en/v3/learn/quick_start
