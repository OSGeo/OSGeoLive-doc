Ändra språk eller tangentbordstyp¶

Byta språk¶

Många vanliga språk är inkluderade på OSGeoLive. Så här byter du språk:

  1. Logga ut

  2. Välj user=”user”, password=”user”

  3. Välj ett annat språk från menyn på den översta panelen.

  4. Tryck på Log in knappen.

  5. Upprepa processen om det behövs eller om du vill byta språk igen.

Systemmenyerna och många av applikationerna kommer nu att visas på det valda språket. Om ett visst program fortsätter att visas på engelska, så har programmet inte en översättning tillgänglig - vänligen överväg volontärarbete för att skriva en översättning. Kontakta programmet av intresse direkt för mer information.

För ytterligare språk se:menuselection:Inställningar –> Språkstöd från huvudmenyn. (Kräver en Internetanslutning).

Stöd för tangentbord¶

Det internationella tangentbords stödet är installerat, men du måste konfigurera systemet för ditt specifika tangentbord. För att ändra inställningar öppna inställningar‣ tangentbord inmatningsmetoder från huvudmenyn och klicka på Input Method-fliken. Klicka sedan på :guilabel:` anpassa aktiva inmatningsmetoder` och sedan på Välj en inmatningsmetod-knappen. Klicka nu på Lägg till knappen till höger för att lägga till en ytterligare layout och sedan Stäng IBus preferenser fönster.

Du kan nu klicka på tangentbordsikonen på den lägre desktopmenyn för att välja den aktiva tangentbordslayouten.
