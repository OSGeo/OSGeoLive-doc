Snabbstart för Open Data Cube¶

Open Data Cube Core tillhandahåller en integrerad miljö för analys av data i rutnät för decennier av analys av färdiga jordobservationssatelliter och relaterade data från flera satelliter och andra förvärvssystem.

Detta avsnitt av Open Data Cube-handledningen

  1. använda Jupyter Notebook för att undersöka data

  2. grundläggande kommandoradsverktyg

Innehåll

  • Analysera en baskarta

  • Saker att prova

  • Hur vill du gå vidare?

Analysera en baskarta¶

Som vår ingångsdataset kommer vi att använda Prizren från esa_worldview_2021 markanvändning / landcover dataset;

Start:

Öppna menyn Geospatial : Geospatiala verktyg : Jupyter Anteckningsbok;

På Jupyters startsida, klicka för att öppna mappen OpenDataCube, klicka för att öppna Notebook, välj ”Run All” från kernel-menyn.

För en snabb titt på en basdatafil kan du öppna en terminal och använda GDAL-kommandot gdalinfo:

#------
$ gdalinfo /usr/local/share/data/odc/esa_10m_2021_prizren.tif

Saker att prova¶

Snabbstart för Datacube kärnkommandon :

$ datacube system check
#--
$ datacube product show
#--
$ datacube dataset search
