Skapa ett OSGeoLive startbart USB-flashminne¶

Denna snabbstart beskriver den föredragna metoden för att skapa ett startbart OSGeoLive USB-flashminne. Uppstart från en flash-enhet är snabbare än från en DVD, och USB-enheten kan konfigureras för att lagra data mellan sessioner (kallas persistens).

Vi har funnit att det finns ett antal besynnerligheter och knep inblandade i framgångsrikt bygga en USB-enhet. Vi har dokumenterat vad vi vet, men det finns sannolikt fler knep och problem. Leta efter senaste tipsen här: https://wiki.osgeo.org/wiki/Live_GIS_Disc_Quick_Start_for_USB

Krav¶

  • Ett USB-minne (8+ Gig rekommenderas).

  • En OSGeoLive ISO-bild (hämtad från: OSGeoLive downloads) eller konverterad från en DVD.

Skapa startbar USB-enhet med dd under Linux¶

ISO-avbildningar för OSGeoLive 10.5 och senare är isohybrid (och för amd64, stöds även för UEFI boot). Under Linux kan dessa avbildningar skrivas direkt till blockenheten för en USB-enhet med kommandoradsverktyget dd:

sudo dd if=osgeolive-10.5-amd64.iso of=/dev/sdX bs=4M; sync

  • /dev/sdX är blockenheten för USB-enheten, där X kan vara b eller c.

  • Använd lsblk för att lista blockenheter och dmesg för att se kärnmeddelanden som identifierar USB-enheten när den är ansluten.

  • USB-enheten ska inte vara monterad.

  • sudo krävs inte om kommandot körs som root.

Varning

Var noga med att välja rätt blockenhet eftersom dd kommer att skriva över den utan tvekan, även om det är din primära systemenhet.

Skapa en startbar USB-enhet med Ubuntu Startup Disk Creator¶

(Detta är den rekommenderade processen för att skapa ett USB-minne. Den är tillämplig på Ubuntu och Ubuntu-varianter som OSGeoLive)

Observera

Den version av Ubuntu som du kör måste vara samma eller nyare än den version av Lubuntu som installeras på din USB. Om du installerar OSGeoLive 8.0 på ett USB-minne måste du köra Ubuntu 14.04 eller nyare.

Ladda ner osgeolive eller osgeolive-mini till din dators hårddisk.

Starta din dator med en ny Ubuntu/Xbuntu-distribution. Du kan göra detta genom att köra från en OSGeoLive DVD, som förklaras i Komma igång med OSGeoLive.

Sätt i USB-minnet i din dator.

Välj System Tools ‣ Startup Disk Creator, eller sök efter Startup Disk Creator från Unity-skrivbordet.

Välj den ISO-bild för osgeolive eller osgeolive-mini som du hämtade tidigare.

Välj ett USB-minne. Om du har extra utrymme kan du justera reglaget för att skapa permanent lagringsutrymme på USB-enheten (rekommenderas). För att det här alternativet ska vara tillgängligt behöver du ett USB-minne på 8+ GB.

Välj Make Startup Disk

Vänta cirka 20 minuter tills USB-minnet har skapats.

Starta från USB-enheten:¶

  1. Börja med att stänga av datorn.

  2. Sätt i USB-enheten i en öppen USB-port.

  3. Slå på maskinen och välj att starta från USB:

    • De flesta datorer är inte inställda på att starta från en USB-enhet som standard.

    • När du startar, leta efter ett meddelande som anger en startmeny (oftast trycker du på F12)

    • Om datorn inte har någon startmeny startar du upp i systemets BIOS. Knappen för att gå in i BIOS varierar beroende på datormärke och modell. Om du inte kan lista ut din egen kan du prova den här webbplatsen som innehåller en lista över många BIOS-genvägsknappar. https://www.mydigitallife.net/comprehensive-list-of-how-key-to-press-to-access-bios-for-various-oem-and-computer-systems/

  4. Välj att starta från USB-enheten.

    • Observera att på vissa datorer måste USB-enheten vara ansluten till datorn för att du ska kunna välja den.

    • Observera också att USB-enheter ofta listas under hårddiskar snarare än under flyttbara enheter.

    • Den här videon <https://www.youtube.com/watch?v=eQBdVO-n6Mg>`_ visar hur man startar från USB.
