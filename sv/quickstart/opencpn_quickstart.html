OpenCPN Snabbstart¶

OpenCPN är en kortfattad kartplotter och navigator (CPN). Som alltid ska du aldrig använda programvara eller elektronisk utrustning som enda navigationshjälpmedel och alltid hålla ett öga ut genom fönstret.

  • Konfigurera din GPS

  • Saker att prova

Konfigurera dina diagram¶

Innan du kör OpenCPN på LiveDVD:n bör du aktivera exempeldata. Detta är ett villkor för att få vidaredistribuera NOAA:s kostnadsfria sjökort. Första gången du öppnar OpenCPN får du möjlighet att göra detta automatiskt.

  1. Välj Geospatial ‣ Navigation and Maps ‣ OpenCPN från Start-menyn.

  2. Klicka på Jag godkänner till NOAA Nautical Chart Data Agreement.

  3. Klicka på OK för att godkänna OpenCPN-licensen.

Du kan ladda ner ytterligare amerikanska sjökort från NOAA på https://www.charts.noaa.gov/

Konfigurera din GPS¶

Om du har en GPS ansluten vill du starta tjänsten gpsd. För att göra det, öppna en terminal och kör:

sudo dpkg-reconfigure gpsd

och

sudo /etc/init.d/gpsd start

BlueTooth GPS-användare bör vara särskilt noga med att läsa på om alternativet utan sond.

Tips

’xgps’ är ett bra program för att kontrollera att Gpsd kan se din GPS ok.

Saker att prova¶

Aktuella väder- och vågprognosdata kan hämtas och draperas över dina diagram med hjälp av GRIB-plugin. I XyGrib-programvaran (finns också på den här skivan) kan du enkelt ladda ner den.

  1. Klicka på ikonen Options i verktygsfältet. Fönstret Options visas.

  2. Välj fliken Plugins.

  3. Välj GRIB och klicka sedan på Enable.

  4. Klicka på OK. En ikon för vindsockan visas nu i verktygsfältet.

Gå igenom XyGrib quickstart och ladda XyGrib-filen över ditt diagram.

Observera

Om du sparade XyGrib-filen i din hemkatalog, navigera till /home och välj din user-katalog som GRIB-datakatalog. Högerklicka sedan på fillistningens bakgrund för att visa dolda filer, inklusive katalogen .xygrib som innehåller GRIB-nedladdningsfilerna.

Det finns inget behov av att komprimera filen. Klicka på ”>” till vänster om filnamnet och välj från listan över tillgängliga prognostider.
