Snabbstart för Zoo Project¶

ZOO-projektet tillhandahåller ett utvecklarvänligt ramverk för Web Processing Service (WPS) för att skapa och kedja Web Processing Services. En WPS ger webbåtkomst till funktioner som kör spatiala algoritmer.

Starta ZOO-projektet¶

Välj Geospatial ‣ Web Services ‣ ZOO Project från Start-menyn.

Firefox kommer att öppna landningssidan för ZOO-projektets demonstration. Härifrån kan du välja en demonapplikation.

Demo av OGR:s spatiala verktyg¶

  1. Klicka på view på OGR:s spatiala verktyg på ZOO-projektets landningssida för demonstration.

  2. Klicka på kartan för att välja en funktion från WFS Server GeoServer, funktionen visas i blått.

  3. Klicka på en enkel geometrisk process som Buffer, Centroid, ConvexHull eller Boundary och resultatet kommer att visas i gult.

  4. Välj nu en annan funktion i närheten av den första du har valt och klicka sedan på en processknapp för flera geometrier som Union, Intersection, Symmetric Difference eller Difference för att få resultatet visat i rött på kartan.

Demo av CGAL-trianguleringar¶

  1. På ZOO-projektets landningssida för demonstration klickar du på view på CGAL-trianguleringar.

  2. Tryck på Voronoi- eller Delaunay-knappen för att visa resultatet av WPS-tjänstens utförande på kartan, enligt bilden nedan.

Demo av OTB-applikationer¶

  1. På landningssidan för ZOO-projektets demonstration, klicka på view på OTB-applikationen.

  2. Fyll i formuläret till vänster för att konfigurera WPS-tjänstens utförande.

  3. När du har valt värden för bearbetningen trycker du på knappen Execute längst ner i det vänstra formuläret.

  4. Om du väljer image/png som utdataformat visas resultatet av din bearbetning på kartan. Nedan kan du se ett exempel på hur KMeansClassification utförs när du väljer utdataformatet image/png.

Hur vill du gå vidare?¶

  • Du kan få tillgång till funktionerna i ZOO WPS-servern:

    http://localhost/cgi-bin/zoo_loader.cgi?Request=GetCapabilities&Service=WPS

  • För ytterligare frågor kan du kontakta ZOO-projektets e-postlista: zoo-discuss@lists.osgeo.org
