Snabbstart för QGIS Server¶

QGIS Server tillhandahåller en webbkarttjänst (WMS) baserad på bibliotek från skrivbordsprogrammet QGIS.

Denna snabbstart beskriver hur du gör:

  • uppdatera ett befintligt QGIS-projekt och rendera det med QGIS Server

  • skapa ett nytt QGIS-projekt och öppna WMS direkt i en webbläsare

Innehåll

  • Starta GeoExt-visningsprogrammet

  • Redigera QGIS-projekt

  • Skapa ett nytt QGIS-projekt

  • Hur vill du gå vidare?

Starta GeoExt-visningsprogrammet¶

OSGeoLive innehåller ett exempel på en webbkartvisare baserad på GeoExt Javascript-ramverket och som använder QGIS Server för att rendera kartan.

  1. Klicka på Geospatial ‣ Desktop GIS ‣ QGIS Desktop. Du bör se ett webbläsarfönster med en webbkartvisare som visar en världskarta.

Alternativt kan du komma åt applikationen med en webbläsare via http://localhost/qgis_server

Redigera QGIS-projekt¶

Vi kommer nu att ändra kartans styling med hjälp av QGIS.

  1. Klicka på Geospatial ‣ Desktop GIS ‣ QGIS Desktop

  2. Välj Project ‣ Open i menyraden.

  3. Välj QGIS-NaturalEarth-Example.qgs och tryck på Open.

    • Du bör se samma världskarta som återges av QGIS. Vi kommer nu att ändra flodstyling och titta på resultatet i webbkartvisaren.

  4. Dubbelklicka på ne_10m_rivers_lake_centerlines i trädet Layers.

  5. Ändra färgen i Outline Options till en annan blå ton.

  6. Tryck på OK. Floder återges nu i en ny färg.

  7. Spara projektet med Project ‣ Save.

  8. Öppna webbläsaren på nytt.

  9. Zooma in. Du kommer nu att se floder återgivna i samma nya färg.

Skapa ett nytt QGIS-projekt¶

  1. Öppna QGIS på nytt.

  2. Välj Projekt ‣ Nytt.

  3. Klicka på Lager ‣ Lägg till lager ‣ Lägg till vektorlager.

  4. Bläddra till datasetet /home/user/data/natural_earth2/ne_10m_admin_0_countries.shp.

  5. Tryck på Open. Du bör se alla världens länder.

  6. Spara projektet med Project ‣ Save.

  7. Use file name /home/user/world.qgz and press Save.

  8. Open the WMS URL. Du bör se alla världens länder, den här gången återgivna av QGIS Server.

Hur vill du gå vidare?¶

Om du vill lära dig mer om QGIS Server är en bra utgångspunkt QGIS Server Guide.

Dokumentation:

  • https://docs.qgis.org/3.10/en/docs/server_manual/

Snabbstart för serverinstallation på Debian/Ubuntu:

  • apt-get install qgis-mapserver libapache2-mod-fcgid

  • -> http://localhost/cgi-bin/qgis_mapserv.fcgi

QGIS webbklient:

  • https://github.com/qgis/qgis-web-client
