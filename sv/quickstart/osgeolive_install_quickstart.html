Installera OSGeoLive till hårddisk¶

OSGeoLive kan installeras på din hårddisk, på samma sätt en Ubuntu-distribution är installerad.

Den här processen kan också köras i en virtuell maskin för att skapa en permanent OSGeoLive virtuell maskin från en DVD-eller ISO-avbildning.

Systemkrav¶

  • 22 GB av extra hårddiskutrymme

Installera OSGeoLive¶

Välj Install RELEASE

Ange password=”user”

Välj språk

Välj tidszon

Välj tangentbord

Ange vilken disk du vill installera på. (Var försiktig, detta kan ta bort en befintlig installerad partition).

Standardinställningar för inställningar är:

  • Namn = ”user”

  • username = ”user”

  • password = ”user”

  • computer name = ”osgeolive”

  • Logga in automatiskt

Välj Install

Gå och ta en fika medan installationen pågår.

Det är det.

Tips

Installationstips Om du installerar med ett annat användarnamn än ”user” måste Tomcat-applikationerna ha behörighet till kommandot service enligt beskrivningen i https://trac.osgeo.org/osgeo/ticket/1153
