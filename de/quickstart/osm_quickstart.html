OpenStreetMap Quickstart¶

OpenStreetMap ist eine freie Karte der Welt, die von einer ständig wachsenden Gemeinschaft von Mappern erstellt wurde. Jeder kann OpenStreetMap bearbeiten. Die Learn OSM website bietet leicht verständliche Schritt-für-Schritt-Anleitungen für Sie, um zu Lernen wie Sie zu OpenStreetMap beitragen zu können, etwas über die Verwendung von OpenStreetMap und OpenStreetMap-Daten zu erfahren. Das Lern-OSM-Handbuch unter dem obigen Link zeigt Schritt für Schritt, wie Sie auf der OpenStreetMap-Webseite navigieren, Karten anzeigen und drucken und sich für ein Benutzerkonto anmelden. Nachdem Sie Ihren eigenen Benutzernamen und Ihr Passwort haben, können Sie Ihre ersten Punkte zur Karte hinzufügen, lernen, wie man Straßenumfragen macht und mit Luftbildern arbeitet.

Im weiteren Verlauf dieses Handbuchs werden einige der Online- und Offline-Software-Projekte vorgestellt, die zum Bearbeiten und Anzeigen von OSM-Daten verwendet werden können.

Programme, die OSM-Daten unterstützen¶

iD¶

iD ist der Online-Standardeditor für OpenStreetMap (OSM). iD ist schnell und einfach zu bedienen und ermöglicht die Zuordnung von verschiedenen Daten Quellen wie Bodenbeschaffenheit, Satelliten- und Luftbildaufnahmen, GPS-Tracks oder Straßen-Level-Daten.

  • Startseite: https://www.openstreetmap.org/edit
  • Dokumentation: https://wiki.openstreetmap.org/wiki/ID
  • Tutorial: https://learnosm.org/en/beginner/id-editor/

JOSM¶

JOSM (Java OpenStreetMap Editor) ist ein funktionsreicher Editor für OpenStreetMap (OSM) und ist in Java geschrieben. Er ist leistungsfähiger und komplexer als iD, einsetzbar für Massenbearbeitung (offline oder mit wackliger Internetverbindung). Er kann durch Plugins, benutzerdefinierte Voreinstellungen und weitere Extensions konfiguriert werden.

  • Homepage: https://josm.openstreetmap.de
  • Bedienungsanleitung: https://wiki.openstreetmap.org/wiki/JOSM/Guide
  • Tutorial: https://learnosm.org/en/josm/start-josm/
  • JOSM-Plugin-Sammlung: https://josm.openstreetmap.de/wiki/Plugins

QGIS¶

OpenStreetMap-Daten können mit QGIS als Vektor-Layer geladen werden. Diese Möglichkeit liegt als Kernfunktionalität vor oder kann durch Plugins wie QuickOSM erfolgen.

  • QuickOSM: Siehe Abschnitt QuickOSM
  • Online-Hilfe: https://wiki.openstreetmap.org/wiki/QGIS#QGIS_2:_Importing_OSM_vector_layers
  • Tutorial: https://learnosm.org/en/osm-data/osm-in-qgis/

Osmosis¶

Osmosis ist ein äußerst fähiges Werkzeug für viele Aufgaben mit OSM-Rohdaten. Dazu gehören Datenimport und Export, Datenbanken, Sortierung, Bereinigung von Daten und Erstellen von Daten-Sicherungen.

  • Dokumentation: https://wiki.openstreetmap.org/wiki/Osmosis#Usage
  • Tutorial: https://learnosm.org/en/osm-data/osmosis/

osm2pgsql¶

osm2pgsql ist ein Hilfsprogramm, das (.osm) OpenStreetMap-Daten in ein Format umwandelt, das in PostgreSQL (PostGIS) geladen werden kann. Es wird oft zum Rendern von OSM-Daten über Mapnik verwendet, da Mapnik PostgreSQL unterstützt, aber OSM-Dateien nicht direkt unterstützt.

  • Webseite: https://wiki.openstreetmap.org/wiki/Osm2pgsql
  • Tutorial: https://learnosm.org/en/osm-data/osm2pgsql/

Arbeiten mit OSM-Daten in QGIS¶

OSGeoLive enthält OSM-Daten für die Stadt, in der die globale FOSS4G- Konferenz stattfindet. Die Daten finden Sie im Dateisystem unter ‚/usr/local/share/data/osm/‘ oder in der PostgreSQL-Datenbank ‚osm_local‘.

Starten Sie QGIS und werfen Sie einen Blick auf die Daten.

  1. Öffnen Sie den Ordner Desktop GIS.
  2. Doppelklicken Sie auf das QGIS-Symbol.
  3. Wechseln Sie in das Browser-Bedienfeld auf der linken Seite des Bildschirms.
  4. Navigieren Sie zu PostGIS und dann zur Datenbank osm_local. Sie finden die Daten im Schema public.
  5. Ziehen Sie die Layer aus dem Browser-Bedienfeld, und legen Sie sie im Kartenbereich ab.

Eine QGIS-Einführung finden Sie im QGIS Quickstart

QuickOSM¶

Mit ‚QuickOSM‘ können Sie dank Overpass API schnell mit OSM-Daten in QGIS arbeiten.

  • Homepage: https://github.com/3liz/QuickOSM
  1. Sie müssen das Plugin in QGIS aktivieren. Suchen Sie nach QuickOSM unter Plugins –> Verwalten und Installieren von Plugins in QGIS
  2. Öffnen Sie QuickOSM über Vector –> QuickOSM –> QuickOSM
  3. Suche nach allen Kneipen in der FOSS4G Stadt
  4. Wählen Sie den Schlüssel amenity Wert pub, in Buenos Aires
  5. Anzeigen der Abfrage zum die Overpass-Abfrage anzuzeigen
  6. Klicken Sie auf Abfrage ausführen. Die resultierenden Daten werden in Ihr QGIS-Projekt als temporäre Daten geladen.
  7. Sie können die Daten jetzt über Export -> Save as…. speichern. Wählen Sie z.B. Geopackage und speichern Sie die Daten als buenos_aires_pubs.

Tipp

Sie finden auch viele OSM Plugins für QGIS. Suchen Sie im Plugin-Repository nach OSM.

Arbeiten mit OSM-Daten als Hintergrundkarte in QGIS¶

XYZ Tiles¶

Standardmäßig kommt QGIS mit einem OpenStreetMap Basemap Layer. Dieses befindet sich im Browser-Panel unter XYZ Tiles.

QuickMapServices¶

Mit dem QuickMapServices-Plugin können Sie ganz einfach verschiedene Karten-Services als Basiskarten mit einem Klick in QGIS hinzufügen.

  • Homepage: https://github.com/nextgis/quickmapservices
  1. Sie müssen das Plugin in QGIS aktivieren. Suchen Sie nach QuickMapServices unter Plugins –> Verwalten und Installieren von Plugins in QGIS
  2. Hinzufügen einer OSM-Grundkarte aus Web –> QuickMapServices –> OSM

Beginnen Sie mit OSM-Daten zu arbeiten¶

Es gibt mehrere Möglichkeiten, OpenStreetMap-Daten für Ihre Region herunterzuladen. Sie können die Daten in verschiedenen Formaten abrufen.

Sie können die folgenden Werkzeuge und Dienste verwenden, um OSM-Daten für Ihren Interessenbereich zu extrahieren.

Geofabrik¶

Die Geofabrik bietet tagesaktuelle weltweite OpenStreetMap-Datenextrakte nach Subregion und Land zum Downlaod. Wählen Sie die Region oder das Land, für das Sie sich interessieren aus. Die Daten werden in drei Formaten bereitgestellt: .osm.pbf, .shp.zip und .osm.bz2., .osm.pbf und .shp.zip und können direkt in QGIS geöffnet werden.

Tipp

.osm.bz2 enthält rohe OSM-XML-Dateien .osm, osm2pgsql funktioniert gut mit bz2-Dateien. Erfahren Sie mehr über die Arbeit mit ihnen unter https://wiki.openstreetmap.org/wiki/OSM_XML

Overpass turbo¶

Overpass turbo ist ein webbasiertes Data Mining Tool für OpenStreetMap basierend auf der Overpass-API.

Werfen Sie einen Blick auf den Web-Client und laden Sie alle Kneipen in der FOSS4G Stadt herunter.

  1. Öffnen Sie https://overpass-turbo.eu/
  2. Suche nach allen Kneipen in der FOSS4G Stadt
  3. Klicken Sie auf den Button Wizard und kopieren Sie die Abfrage in das Suchfeld amenity=pub in „Buenos Aires“
  4. Klicken Sie dann auf die Schaltfläche Abfrage erstellen und ausführen
  5. Klicken Sie im Kartenbereich auf Zoom auf Daten, um zu Ihren Ergebnissen zu navigieren

HOT-Export-Werkzeug¶

Das Humanitarian OpenStreetMap Team (HOT) stellt ebenfalls ein Export-Werkzeug für OpenStreetMap-Daten zur Verfügung. Um es zu verwenden, benötigen Sie ein OSM-Konto. Sie können Daten in verschiedenen Formaten herunterladen und haben eine Historie für Ihre heruntergeladenen Daten und Exporte.

Hier hier finden Sie ein Schnellstart-Dokument: https://export.hotosm.org/en/v3/learn/quick_start
