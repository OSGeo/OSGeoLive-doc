XyGrib Quickstart¶

XyGrib ist ein Programm, um Daten zur Wettervorhersage im standardmäßigen GRIB-Format herunterzuladen und zu visualisieren.

Inhalt

  • Start XyGrib

  • Daten herunterladen

  • Ansichten anpassen

  • Erstellen einer Animation

  • Was kommt als Nächstes?

Start XyGrib¶

  1. Wählen Sie auf dem Desktop Navigation and Maps ‣ XyGrib GRIB file viewer.

  2. Wählen Sie Ihre bevorzugte Sprache und klicken Sie auf OK. Das Programm wird geöffnet und Sie erhalten eine Weltkarte.

  3. Überprüfen Sie die Verbindung zum Datenserver. Klicken Sie auf der Symbolleiste oben rechts auf dem Bildschirm auf das Netzsteckersymbol. Das Fenster Dateiserverstatus wird geöffnet und nach einem Moment wird der Status der Verbindung bestätigt.

Bemerkung

Sie benötigen eine Internetverbindung, um GRIB Wettervorhersagedaten herunterzuladen. Sie können GRIB-Daten aus vielen Quellen beziehen, aber aus Bequemlichkeit unterhalten die XyGrib-Autoren einen Datenserver, der mit XyGrib verwendet werden kann.

Daten herunterladen¶

  1. Klicken und ziehen Sie mit der Maus ein Feld auf der Karte in dem Bereich auf, an dem Sie interessiert sind. Der Golf von Mexiko oder die südliche Spitze Afrikas haben oft dynamische Winde.

  2. Wenn das Feld auf der Karte gezeichnet ist, klicken Sie auf das Zoom-to-Box-Symbol in der Taskleiste (die Lupe mit einem kleinen Feld darunter)

  3. Ziehen Sie erneut einen Bereich mit einem Linksklick-Ziehen auf, das jedoch dieses Mal den größten Teil oder den gesamten vergrößerten Kartenbereich abdeckt.

  4. Wählen Sie File ‣ Download GRIB und wählen Sie die Funktionen aus, die Sie herunterladen möchten. Wählen Sie im Zweifelsfall die Standardwerte.

  5. Wenn Sie fertig sind, klicken Sie auf die Schaltfläche Download und speichern Sie die Datei.

Die Anwendung zeigt eine Wetterkarte mit einem Zeitschieberegler entlang der Unterseite. Dieser ist auf die aktuelle Zeit eingestellt. Sie können das Mausrad verwenden, um die Karte zu vergrößern, wenn Sie möchten.

Ansichten anpassen¶

  1. Bewegen Sie die Maus auf der Karte, sie sehen die genauen Bedingungen im Bereich auf der linken Seite aufgeführt.

  2. Sie können die Uhrzeit ändern, indem Sie den Schieberegler am unteren Rand oder über das Dropdown-Menü oben links im Fenster verschieben.

  3. Wählen Sie im Menü Weather map eine Variable aus, die visualisiert werden soll. Wählen Sie beispielsweise Wetterkarte ‣ Wolkenabdeckung und die Ansicht sollte sich ändern.

Wenn Sie Daten für den Golf von Mexiko heruntergeladen haben, werden Sie daran interessiert sein, die CAPE-Daten anzuzeigen. „CAPE“ steht für Convective Available Potential Energy at ground level. Es ist ein Maß für die atmosphärische Instabilität, und hohe Werte (1000 J/kg) deuten auf die Gefahr von Gewittern und/oder Tornados hin.

Wenn Sie Daten für das südliche Afrika heruntergeladen haben, kann die Cloud-Abdeckung interessanter sein.

Erstellen einer Animation¶

  1. Klicken Sie auf das rechte Symbol auf der Symbolleiste (ein Raketenschiff). Ein neues Fenster wird geöffnet und eine Animation wird generiert.

  2. Warten Sie, während es zum ersten Mal durch die Zeitsequenz läuft. Wenn es fertig ist, wird es zum Anfang der Zeitreihe zurückspulen.

  3. Sie können die Sequenz nach Belieben wiedergeben, anhalten und zurückspulen.

  4. Sie können jeden Ausschnitt auch als Bilddatei als AVI, MPEG oder animiertes GIF mit anderer Software speichern.

  5. Wenn Sie fertig sind, klicken Sie auf den roten Button, zum Schließen.

Was kommt als Nächstes?¶

Die OpenCPN Software (auch auf OSGeoLive) hat ein GRIB-Overlay-Plugin, das Sie mit Ihrer neu heruntergeladenen GRIB-Datei ausprobieren können. Details finden Sie im OpenCPN Quickstart.
