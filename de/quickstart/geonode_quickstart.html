GeoNode Quickstart¶

Open Source Geospatial Content Management System

GeoNode ist eine webbasierte Anwendung und Plattform für die Entwicklung von Geodateninformationssystemen (GIS) und für die Bereitstellung von Geodateninfrastrukturen (SDI).

GeoNode starten¶

Um GeoNode auf OSGeoLive zu starten, müssen Sie Geospatial ‣ Browser Clients ‣ GeoNode ‣ Start GeoNode auswählen und nach wenigen Augenblicken wird die GeoNode-Webseite automatisch bei http://geonode geöffnet. Die Seite sieht wie das Bild unten aus.

Anmelden als registrierter Benutzer¶

Über die oben gezeigte Schnittstelle können vorhandene räumliche Layer und Karten angezeigt und geändert werden sowie Informationen zu anderen GeoNode-Benutzern gefunden werden. Ohne angemeldet zu sein, sind Sie jedoch auf den schreibgeschützten Zugriff auf öffentlicher Ebenen beschränkt. Um eine Karte zu erstellen und Layer hinzuzufügen, müssen Sie sich zuerst anmelden.

Von jeder Seite in der Web-Oberfläche sehen Sie einen Anmelden-Link oben rechts auf der Seite. Klicken Sie auf diesen Link, und geben Sie im angezeigten Dialogfeld die Benutzeranmeldeinformationen ein. Probieren Sie „admin“ und „admin“ für den Benutzernamen und das Kennwort aus.

Hinzufügen einer neuen Ebene¶

Layer sind eine veröffentlichte Ressource, die eine räumliche Raster- oder Vektordatenquelle darstellen. Layern können auch Metadaten, Bewertungen und Kommentaren zugeordnet werden.

  1. Um vorhandene Layer anzuzeigen, wählen Sie in der Navigationsleiste oben auf einer beliebigen Seite das Menü Daten aus und klicken Sie auf den Link Layer.

Das Ergebnis sieht wie folgt aus:

  1. Um Ihrem Konto einen Layer hinzuzufügen, wählen Sie oben rechts auf der Seite Layer die Schaltfläche Upload aus:

Falls Sie Daten von einer anderen Seite hochladen möchten, wählen Sie in der Navigationsleiste das Menü Daten aus und klicken Sie auf den Link Layer hochladen.

Wenn Sie auf einen der oben genannten Upload-Links klicken, gelangen Sie zur Upload-Seite.

  1. Sie haben zwei Möglichkeiten, Ihre Dateien hinzuzufügen. Sie können dies entweder tun, indem Sie drag & drop im Bereich Drop Files Here verwenden, oder Sie können die Schaltfläche Dateien auswählen verwenden, um sie auszuwählen. Beachten Sie, dass Sie einen vollständigen Satz von Dateien hochladen müssen, bestehend aus einem shp-, einem prj-, einem dbf- und einer shx-Datei. Wenn eine dieser Dateien fehlt, wird GeoNode Sie warnen, bevor Sie sie hochladen.

  2. Klicken Sie nun auf Dateien auswählen. Es wird ein Dateiauswahlfenster angezeigt. Navigieren Sie zu /home/user/data/natural_earth2, und wählen Sie alle ne_10m_admin_0_countries Dateien aus.

Sie sollten nun in der Lage sein, alle Dateien anzuzeigen, die Sie hochladen möchten.

Tipp

GeoNode kann einschränken, wer Layer anzeigen, bearbeiten und verwalten kann. Auf der rechten Seite sehen Sie den Abschnitt Berechtigung, in dem Sie den Zugriff auf Ihre Layer einschränken können. Wählen Sie unter Wer diese Daten anzeigen und herunterladen kann, wählen Sie Jeder registrierte Benutzer. Dadurch wird sichergestellt, dass der anonyme Ansichtszugriff deaktiviert ist. Wählen Sie im gleichen Bereich unter Wer diese Daten bearbeiten kann, wählen Sie Ihren Benutzernamen aus. Dadurch wird sichergestellt, dass nur Sie die Daten des Layers bearbeiten können.

  1. Um Daten hochzuladen, klicken Sie unten auf die Schaltfläche Upload.

  2. Nachdem der Layer-Upload abgeschlossen ist, können Sie entweder eine Vorschau des Layers mit der Schaltfläche Layer Info anzeigen oder die Layer-Metadaten mit Metadaten bearbeiten oder den Layer-Stil auf GeoServer mit der Schaltfläche Stile verwalten bearbeiten.

  3. Drücken Sie die Schaltfläche Layer Info, um die Ebenenseite anzuzeigen

Erstellen Sie eine neue Karte und öffnen Sie diese¶

Der nächste Schritt besteht darin, eine Karte zu erstellen und die neu erstellten Layer zu dieser Karte hinzuzufügen.

  1. Klicken Sie auf der Symbolleiste Karten*auf den Link *Karten erkunden. Dadurch wird die Liste der Karten angezeigt.

  2. Momentan gibt es hier keine Karten. Um eine hinzuzufügen, klicken Sie auf die Schaltfläche Neue Karte erstellen.

Eine Kartenkompositionsschnittstelle wird angezeigt.

Wenn eine 404 Popup-Nachricht angezeigt wird, ignorieren Sie sie, dies ist auf ein fehlendes GeoServer-Plugin zurückzuführen, das nicht auf OSGeoLive installiert ist.

In dieser Schnittstelle gibt es eine Symbolleiste, eine Layer-Liste und ein Kartenfenster. Das Kartenfenster enthält standardmäßig den Layer OpenStreetMap.

  1. Klicken Sie auf die Schaltfläche Neue Ebenen und wählen Sie Layer hinzufügen.

Jetzt sollten Sie in der Lage sein, alle verfügbaren Layer zu sehen. Es sollten nun auch die Layer enthalten sein, die Sie zuvor hinzugefügt haben.

  1. Wählen Sie einige Layer aus, indem Sie auf den ersten Eintrag klicken und Strg auf den Rest klicken. Klicken Sie auf Layer hinzufügen, um sie alle zur Karte hinzuzufügen.

Die Layer werden der Karte hinzugefügt.

  1. Um die Karte zu speichern, klicken Sie auf die Schaltfläche Karte in der Symbolleiste, und wählen Sie Karte speichern.

  2. Geben Sie einen Titel und eine Zusammenfassung für die Karte ein.

  3. Klicken Sie auf Speichern. Beachten Sie, dass sich der Link oben rechts auf der Seite geändert hat, um den Namen der Karte widerzuspiegeln.

    Dieser Link enthält einen Permalink zu Ihrer Karte. Wenn Sie diesen Link in einem neuen Fenster öffnen, wird die Karte genau so angezeigt, wie sie gespeichert wurde.

  4. Klicken Sie auf Fertig neben der Schaltfläche Layer hinzufügen, um die Legende Ihrer Karte anzuzeigen.

Teilen Sie Ihre Karte¶

Lassen Sie uns nun unsere Karte fertigstellen.

  1. Rechtsklick auf den Layer USA Population und wählen Sie Zoom to layer extend

  2. Nehmen Sie alle endgültigen Anpassungen an der Kartenzusammensetzung wie gewünscht vor, einschließlich Zoom- und Pan-Einstellungen.

  3. Klicken Sie in der Symbolleiste auf die Schaltfläche Karte, und klicken Sie dann auf Karte veröffentlichen.

  4. Der Titel und die Zusammenfassung, wie sie zuvor erstellt wurden, sollten immer noch vorhanden sein. Nehmen Sie ggf. alle Anpassungen vor, und klicken Sie auf Speichern.

  5. Es wird ein neues Dialogfeld mit Anweisungen zum Einbetten dieser Karte in eine Webseite angezeigt, einschließlich eines Codeausschnitts. Sie können die Parameter bei Bedarf anpassen.

Ihre Karte kann jetzt freigegeben werden.

Was kommt als Nächstes?¶

Jetzt haben Sie einen schnellen Einblick in die Möglichkeiten von GeoNode erhalten. Um mehr über GeoNode und seine Funktionen zu erfahren, besuchen Sie unsere Webseite www.geonode.org.

Um GeoNode auf Ihrem eigenen Server zu installieren, folgen Sie unserem Quick Installation Guide .
