gvSIG Desktop Quickstart¶

gvSIG is a Geographic Information System (GIS), that is, a desktop application designed for capturing, storing, handling, analyzing and deploying any kind of referenced geographic information in order to solve complex management and planning problems. gvSIG is known for having a user-friendly interface, being able to access the most common formats, both vector and raster ones. It features a wide range of tools for working with geographic-like information (query tools, layout creation, geoprocessing, networks, etc.), which turns gvSIG into the ideal tool for users working in the land realm.

Bemerkung

If gvSIG user interface becomes unresponsive, try Alt+f to open the file menu for example. User interface should work again, then. This bug only happens in the gvSIG distro for OSGeoLive. It doesn’t appear in a normal gvSIG installation.

Bemerkung

Note for version gvSIG 2.3.1: To load raster files the user must have write permission in the folder where the file is. Otherwise the application will ask for a valid folder (with write permission). Since the „user“ user doesn’t have write permission in the „data“ folder, it will be necessary to select, when asked, a folder where the user does have write permission, like for example, „/home/user“.

Start gvSIG and create a view¶

  1. Select gvSIG from the application menu, Geospatial ‣ Desktop GIS ‣ gvSIG. The application usually takes about a minute to startup.
  2. By default gvSIG opens with an empty project file with the ‚View‘ document type selected. However, on OSGeoLive, gvSIG has been configured to start with a default project. Click on the New Project button or go to File ‣ New Project. When prompted, you can select Discard Changes as we won’t be using the last project.
  3. Click on the New button to create a view. It will be opened automatically.

Define the projection of your view¶

  1. Select View ‣ Properties.

  2. For the Current projection, click on the … button. This will open the New CRS dialog.

  3. Using the dropdown list for Type, select EPSG.

  4. Enter the EPSG code for the Coordinate Reference System. For this example we are using EPSG=4326, which is the WGS84 2D Geographic Projection. Then click on the Search button. This will list the selected CRS in the table below the search input field.

  5. Select OK to return to the View Properties dialog. The EPSG code is now 4326 and that the map units have changed to degrees.

  6. Select OK to return to the Project Manager.

    1. The view window consists of three zones.
  • (2) The top-left cell contains a list of vector or raster layers being used in the view (i.e. Table-of-Contents).
  • (3) The bottom-left cell displays the extent of the main view over a selected vector file.
  • (4) The right cell is the main display area where raster and vector data is rendered.

Add layers to the view¶

  1. Click on the Add layer icon in the main toolbar to open the Add layer dialog.
  2. Select Add in the Add layer dialog. By default the first tab is File so the Open file dialog will appear.
  3. Drill down to the /usr/local/share/data/natural_earth2/ directory.
  4. Select the files as shown in the picture.

Bemerkung

By default all formats are shown so we can select both vector and raster files at a time by keeping pressed the CTRL key.

  1. Click on the OK button to return to the Add layer dialog.
  2. Click on the OK button on the Add layer dialog to return to the view.

Configure the view display¶

  1. Click on the Maximize icon for a better viewing of the data.
  2. The size of the Table-of-Contents column can be adjusted using the mouse to help see long file names.
  3. In case the raster layer is on the top it should be moved manually to the bottom.

Zoom into a region¶

  1. By default the Zoom in tool is selected on the main toolbar.
  2. Use the mouse pointer to draw a bounding box around South-east Australia. Do this by selecting the top-left corner of the bounding box, holding down the left-mouse-button, and dragging the bottom-right corner of the bounding box over the selected area. Release the left-mouse button once you have the approximate area defined.

Change how the layers look¶

Make a polygon transparent¶

  1. Select the ne_10m_admin_1_states_provinces.shp layer using the left-mouse-button and click again with the right-mouse-button.
  2. Select the Properties submenu.
  3. The Layer properties dialog box will appear.

Change the symbology of a vector layer¶

  1. Select the Symbols tab at the top of the Layers properties dialog.
  2. Select Choose symbol button so the Symbol selector dialog appears.

Remove the fill symbol and changing the border color¶

  1. Uncheck the Fill color box so the polygons are transparent.
  2. Click on the … close to Border color:.
  3. Choose white as color.
  4. Click on the OK button to return to the Symbol selector dialog.
  5. Click on the OK button to return to the Layers properties dialog.

Change the labelling of a vector layer¶

  1. Select the Labelling tab at the top of the Layers properties dialog.
  2. Change the text attributes to those shown on the image, then click on the Accept button.

View the result¶

Note that this is a very basic view showing a point, a line and a polygon vector file superimposed over a raster file. It is just as easy to have an aerial photograph or Digital Terrain Model as a backdrop to your vector data, or to show other vector data stored in different formats.

Add an overview or locator¶

Select View ‣ Locator setup.

  1. Click on the Add layer… button.
  2. Within the Add layer window, click on the Add button.
  3. Select the HYP_50M_SR_W.tif layer.
  4. Click on the Open button to return to the Add layer window.
  5. Click on the Ok button and close the Configure locator window.

Now we can easily know where the view is located in the world.

Select towns within a region¶

Select the region¶

  1. Select the ne_10m_admin_1_states_provinces.shp layer using the left-mouse-button.
  2. Select the Select by point icon in the main toolbar.
  3. Click the polygon representing the State of New South Wales. The polygon will go yellow or some other colour depending on your user preferences.

Open the Selection by layer tool¶

  1. Select the ne_10m_populated_places.shp layer using the left-mouse-button.
  2. Select Selection ‣ Select by layer to open the Selection by Layer dialog

Select the towns contained within a region¶

  1. Change the first selection criteria using the dropdown boxes on the left-hand side of the Selection by Layer dialog as shown in the picture.
  2. Change the second selection criteria as shown in the picture.
  3. Click the New set button to select towns within the selected polygon.
  4. Select the Cancel button in the Selection by Layer dialog to return to the view.

Deselect the polygon so you can see what has happened¶

  1. Select the ne_10m_admin_1_states_provinces.shp layer using the left-mouse-button.
  2. Select the ‚Clear selection‘ icon in the main toolbar.
  3. You can now see that only those towns within New South Wales are selected.

Export selected features to a new shapefile¶

Select the output file format¶

  1. Select the ne_10m_populated_places.shp layer using the left-mouse-button.
  2. Select Layer ‣ Export to… to start the export.
  3. Select Shape Format.
  4. Click on Next.

Select the output file name¶

  1. Choose a folder and type the file name.
  2. Click on Open and then on Next.

Reference system¶

  1. Choose the option Original (EPSG:4326) in order to keep the reference system of the view.
  2. Click on Next.

Export options¶

  1. Choose the option Selected features in order to export only the towns of New South Wales.
  2. Click on Export.

View the result¶

  1. The new layer has been added to the view.
  2. In order to check that the export was ok set the original file as invisible. Only the towns of New South Wales should be shown.

Create a new map¶

  1. Select the Map document type in the Project Manager.
  2. Click on the New button to create a map.
  3. A new window is shown, where you can select the view to insert on the Map. Select the only view that you have and press Accept.
  4. The map will appear with the view selected.

Work with the map¶

  1. Additional elements like a scale and north arrow can be added to the map using the icons in the main toolbar or with the submenus in the Map menu.
  2. The map can be printed or exported to PDF or Postscript for incorporation into other works.
  3. Select the Close window icon to return to the Project manager

Save your project or exiting gvSIG¶

  1. Projects can be saved for use later by using the File ‣ Save as… menu option, or
  2. Projects can be exited or closed by using the File ‣ Exit menu option.

Install an add-on¶

Open the add-ons manager¶

  1. Select Tools ‣ Addons manager. After a while, the Install package window will appear.
  2. Within the Install package window, select the option Installation from URL.
  3. Choose the repo URL shown in the picture.
  4. Click on the Next button.

Find an add-on¶

  1. At the add-ons manager we can find plugins and symbol libraries.
  2. Type ‚Forestry‘ in the Fast filter text box.
  3. Check the add-on called Symbols: Forestry (it’s a symbol library).
  4. Click on Next.

Finish the installation¶

  1. Click on Start downloading.
  2. Then click on Next.
  3. Finally click on Finish.

Although the new message indicates that a restart is needed, for symbol libraries it’s not necessary. We only have to restart when we install plugins.

Add an OpenStreetMap layer¶

  1. Now we are going to add OpenStreetMap cartography. Within a new view, in EPSG 3857, click on the Add layer… button.
  2. Choose the OSM tab.
  3. Select ‚Mapnik‘ from the available layers.
  4. Click on Ok.

View the result¶

A new layer has been added to the view. Just doing zoom over a region we could see the detailed Open Street Map data.

Add a WMTS layer¶

WMTS is an evolution of WMS OGC standard based on tile management.

Select a server¶

  1. Within a view, click on the Add layer… button.
  2. Choose the WMTS tab.
  3. Choose the URL shown in the picture.
  4. Click on Connect.
  5. Click on Next.

Select one of the available layers¶

  1. Select one of the available layers (e.g. bluemarble)
  2. Click on Add.
  3. Click on Next.

Select image format and reference system¶

  1. Select image/jpeg as image format.
  2. Select EPSG:4326 (notice that the view must have the same SRS. If not then cancel, change it and start again).
  3. Click on Accept.

A new layer has been added to the view.
