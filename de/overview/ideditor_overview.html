iD Editor¶

Browserbasierter Editor für OpenStreetMap¶

Der iD Editor ist ein browserbasierter OpenStreetMap Editor. iD ist schnell und leicht zu handhaben und erlaubt das Erfassen auf Basis von unterschiedlichen Datenquellen z.B. Satelliten und Luftbildern, GPS, Field Papers oder Mapillary.

Mit dem iD Editor können sehr gut kleine und leichte Änderungen durchgeführt werden, die nicht auf die erweiterten Möglichkeiten des komplexeren JOSM Editors angewiesen sind.

Kernfunktionen¶

  • OpenStreetMap Bearbeitung

  • Schnell

  • Intuitiv

  • Browserbasiert

Unterstützte Standards¶

  • Web Map Tile Service (für Basemaps)

Details¶

Webseite: https://wiki.openstreetmap.org/wiki/ID

Lizenz: ISC

Software Version: 2.21.1

Unterstützte Platformen: Alle Plattformen, die einen Browser unterstützen: Windows, GNU/Linux, Mac, Android, iOS

Support: https://help.openstreetmap.org/
