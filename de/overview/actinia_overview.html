actinia¶

Hochperformante Geoprozessierung¶

Actinia ist eine Open-Source-REST-API für die skalierbare, verteilte, leistungsstarke Verarbeitung von Geodaten, die vornehmlich GRASS GIS für die Berechnungen verwendet.

Es bietet eine REST-API zur Verarbeitung von Satellitenbildern, Zeitreihen von Satellitenbildern, beliebigen Rasterdaten mit geografischen Beziehungen und Vektordaten.

Kernfunktionen¶

  • REST API zur Verarbeitung von Satellitenbildern, Zeitreihen von Satellitenbildern, Raster- und Vektordaten.
