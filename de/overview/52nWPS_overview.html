52°North WPS¶

Über¶

Der 52°North Web Processing Service (WPS) ermöglicht den webbasierten Zugang auf raumbezogene Verarbeitungsalgorithmen, die verfügbar sind in Sextante, ArcGIS Server, R, GRASS 7 oder auch eigene definierte Algorithmen. Dabei können die Algorithmen einfach sein wie eine Bestimmung der unterschiedlichen Einflüsse der verschiedenen Jahreszeiten bis hin zu komplizierten globalen Klimaänderungsmodellen.

Kerneigenschaften¶

GIS Kernfunktionen:

  • SimpleBufferAlgorithm

  • DouglasPeuckerAlgorithm

  • IntersectionAlgorithm

  • AddRasterValuesAlgorithm

  • CoordinateTransformAlgorithm

Backend Algorithmen stehen zur Verfügung in:

  • Sextante

  • ArcGIS Server

  • R

  • GRASS 7

Unterstützte Ein- und Ausgabeformate:

  • Vektordaten: GML2, GML3, SHP-Dateien, KML, WKT (Well-known Text), DGN, JSON

  • Rasterdaten: Geotiff, AsciiGrid, NetCDF, JPEG, PNG, HDF-EOS, ERDAS HFA

  • Ausgaben können in GeoServer/MapServer abgelegt werden

Browserbasierter Client

Implementierte Standards¶

  • OGC GML2 SimpleFeatures

  • OGC GML3 SimpleFeatures
