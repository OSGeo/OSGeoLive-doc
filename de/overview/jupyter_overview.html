Jupyter Notebook¶

Mischen von Rich Media in der Dokumentation¶

Das Jupyter Notebook ist eine Webanwendung, mit der Sie Dokumente erstellen und freigeben können, die Live-Code, Gleichungen, Visualisierungen und erklärenden Text enthalten.

Es vereint zwei Komponenten:

  • Eine Webanwendung, die browserbasierte, interaktive Erstellung von Nootebook-Dokumenten bereitstellt.
  • Nootebook-Dokumente, in denen eine Darstellung aller in der Webanwendung sichtbaren Inhalte gespeichert ist, einschließlich Ein- und Ausgänge von Berechnungen, erläuterndem Text, Mathematik, Code, Bildern, Videos, Grafiken, Karten und anderen Rich Media-Darstellungen von Objekten.

Notebooks können für andere auf GitHub, Dropbox und dem Jupyter Notebook Viewer freigegeben werden.

Zu den Verwendungsmöglichkeiten gehören: Datenbereinigung und -transformation, numerische Simulation, statistische Modellierung, maschinelles Lernen und vieles mehr.

Kernfunktionen¶

Create Jupyter Notebooks which can contain:

  • Marked up Text (mit Überschriften, Formatvorlagen, Absätzen usw.)
  • Formeln, Mathematik, Plots, Karten, …
  • Imported external libraries to add functionality
  • Code from multiple programming languages, including Python, R, Julia, Bash and many more.

Jupyter Notebooks on OSGeoLive can use geospatial functions from:

  • Numpy, SciPy Matplotlib, IRIS, Cartopy, gdal, Geopandas, pyshp, Fiona, netCDF4, PostgreSQL, psycopg2, R, rpy2, und weitere.
