PyWPS¶

Web Processing Service¶

PyWPS stellt Webzugriff für anpassbare räumliche Operationen (Prozesse) über den Web Processing Service (WPS) Standard bereit.

Die Prozesse werden in der Programmiersprache Python geschrieben und können Programme wie GRASS GIS, R, GDAL/OGR, proj und andere Bibliotheken mit Python-Bindings integrieren. Daten können über WPS-Anfragen bereitgestellt werden.

Kernfunktionen¶

  • Klein und leichtgewichtig

  • Leicht und schnell zu installieren und aufzusetzen

  • Reine Pythonimplementierung

  • Nahtlosen Integration in das Python-Paket-Ökosystem:

    • GRASS 7

    • R

    • GDAL/OGR

    • Proj

    • Numpy

    • Shapely

Implementierte Standards¶

OGC Standards:

  • WPS 1.0.0
