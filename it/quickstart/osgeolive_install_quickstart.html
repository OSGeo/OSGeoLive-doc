Installa OSGeoLive nel disco rigido¶

OSGeoLive può essere installato sul disco rigido, nello stesso modo in cui si installa una distribuzione Ubuntu.

Questo processo può anche essere eseguito all’interno di una macchina virtuale per creare una macchina virtuale OSGeoLive permanente da un DVD o un’immagine ISO.

Requisiti di Sistema¶

  • 20 GB Spazio Hard Disk di riserva

Installa OSGeoLive¶

Selezionare Install RELEASE

Inserire password=»user»

Selezionare il linguaggio

Selezionare il fuso orario

Selezionare la tastiera

Specifica su quale disco si vuole installare. (Attenzione, questo potrebbe cancellare una partizione già installata).

Le impostazioni di base sono:

  • Nome = «user»

  • nome utente = «user»

  • password = «user»

  • nome del computer = «osgeolive»

  • Accesso automatico

Selezionare Install

Avviare e andare a prendere un tazza di caffè mentre l’installazione è in avanzamento.

Questo è tutto

Suggerimento

Suggerimenti per l’installazione: Nel caso in cui si stia installando usando un nome utente diverso da «utente», le applicazioni tomcat richiedono di avere il permesso di eseguire il comando service come descritto in https://trac.osgeo.org/osgeo/ticket/1153
