Guida rapida OpenCPN¶

OpenCPN è un sintetico Chart Plotter and Navigator (CPN). Come sempre, non usate mai un software o un’attrezzatura elettronica come unico mezzo di navigazione e tenete un occhio fuori dalla finestra.

Impostazione dei grafici¶

Prima di eseguire OpenCPN sul LiveDVD dovresti attivare i dati di esempio. Questa è una condizione per ridistribuire le carte nautiche gratuite della NOAA. La prima volta che apri OpenCPN ti viene presentata l’opzione di farlo automaticamente.

  1. Dal menu Start, seleziona Geospatial ‣ Navigation and Maps ‣ OpenCPN.

  2. Clicare I agree per accettare il contratto NOAA Nautical Chart Data.

  3. Clicca OK per accettare la licenza OpenCPN.

Potete scaricare altre carte nautiche americane dal NOAA all’indirizzo https://www.charts.noaa.gov/

Impostazione del GPS¶

Se avete un GPS collegato, vorrete avviare il servizio gpsd. Per farlo, aprite un terminale ed eseguite:

sudo dpkg-reconfigure gpsd

e

sudo /etc/init.d/gpsd start

Gli utenti di BlueTooth GPS dovrebbero prestare particolare attenzione all’opzione no-probe.

Suggerimento

“xgps” è un buon programma per controllare che Gpsd possa vedere il tuo GPS.

Cose da provare¶

I dati meteorologici attuali e le previsioni delle onde possono essere scaricati e drappeggiati sui tuoi grafici usando il plugin GRIB. Il XyGrib software (anche su questo disco) è un modo semplice per scaricarlo.

  1. Fai clic sull’icona Options sulla barra degli strumenti. Viene visualizzata la finestra Opzioni.

  2. Selezionare il pannello Plugins

  3. Scegliere GRIB e poi cliccare Enable.

  4. Fare clic su OK. Un’icona a forma di manica a vento viene ora visualizzata sulla barra degli strumenti.

Esegui il XyGrib quickstart e carica il file XyGrib sul tuo grafico.

Nota

Se hai salvato il file XyGrib nella tua home directory, vai a /home e seleziona la tua directory user come directory dei dati GRIB. Poi clicca con il tasto destro sullo sfondo dell’elenco dei file per mostrare i file nascosti, inclusa la directory .xygrib che contiene i file GRIB da scaricare.

Non è necessario decomprimere il file. Clicca sul «>» a sinistra del nome del file e seleziona dalla lista dei tempi di previsione disponibili.
