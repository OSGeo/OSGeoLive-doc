Guida rapida deegree¶

deegree è un software open source per infrastrutture di dati spaziali e il web geospaziale. deegree è un’applicazione Java per servire mappe (e dati vettoriali) da mostrare ad altri client.

I componenti disponibili sono:

  • Web-Services: WFS, WMS, WMTS, CSW, WPS

deegree 3 è la nuova generazione del framework deegree Java per applicazioni GIS e implementazioni di servizi OGC. Può essere usato per costruire infrastrutture di dati geospaziali basate sul web (GDI).

Avvia i webservice di deegree 3¶

Scegli “Start deegree” dal menu Geospatial ‣ Web Services ‣ deegree. L’applicazione impiegherà qualche istante per avviarsi.

Firefox apre la console dei servizi deegree 3. (Puoi anche aprirla manualmente puntando il tuo browser su http://localhost:8033). Questa è una semplice interfaccia grafica basata sul web per modificare i file di configurazione dello spazio di lavoro deegree.

Carica un workspace¶

deegree 3 usa configurazioni conformi a tutti i webservice basati su deegree 3. Queste configurazioni sono memorizzate nello spazio di lavoro.

  1. Per gestire gli spazi di lavoro inclusi e pronti all’uso, clicca su “spazi di lavoro” sul lato sinistro. Ora sei nella gestione degli spazi di lavoro dell’istanza di deegree in esecuzione. Puoi vedere quale spazio di lavoro è attualmente attivo (solo uno può essere attivo alla volta), gli spazi di lavoro inattivi disponibili localmente e gli spazi di lavoro ufficiali che possono essere importati se hai una connessione internet.

  2. Premi start su deegree-workspace-utah-light.

L’area di lavoro demo contiene una configurazione di web mapping basata sui dati dello stato dello Utah. Contiene una configurazione WMS (1.3.0 e 1.1.1) composta da layer vettoriali e una varietà di stili di rendering. Inoltre, è configurato un WFS (2.0.0, 1.1.0 e 1.0.0) che consente di accedere ai dati vettoriali grezzi, supportati da shapefile, in formato GML.

Suggerimento

Per ulteriori esempi di configurazione, compresi i dati raster, è possibile importare lo spazio di lavoro deegree-workspace-utah attraverso la gestione dello spazio di lavoro (è necessaria una connessione a Internet).

Vedi una mappa¶

  1. Clicca su “see layers” per aprire un semplice map-client. Il servizio è configurato per visualizzare i dati e una mappa di base (resa da OpenStreetMap).

  2. Fare clic su + sul lato destro per aprire un albero dei layer che mostra tutti i layer disponibili. Qui è possibile attivare una serie di layer serviti da deegree WMS.

Suggerimento

Il client della mappa è basato su OpenLayers. Puoi fare una panoramica della mappa tenendo premuto il tasto sinistro del mouse e trascinando la mappa. Lo zoom è possibile tramite la rotella del mouse o gli elementi di navigazione sul lato sinistro.
