Guida rapida QGIS Server¶

QGIS Server fornisce un servizio di mappe web (WMS) basato sulle librerie dell’applicazione desktop QGIS.

Questa Guida Rapida descrive come:

  • aggiorna un progetto QGIS esistente e visualizzalo con QGIS Server

  • crea un nuovo progetto QGIS e apri il WMS direttamente in un browser

Contenuti

  • Start il visualizzatore GeoExtGeo

  • Modifica del progetto QGIS

  • Crea un nuovo progetto QGIS

  • E ora?

Start il visualizzatore GeoExtGeo¶

OSGeoLive contiene un esempio di visualizzatore di mappe web basato sul framework Javascript GeoExt e che usa QGIS Server per il rendering della mappa.

  1. Clicca su Geospatial ‣ Web Services ‣ QGIS Server. Dovresti vedere una finestra del browser con un visualizzatore di mappe web che mostra una mappa del mondo.

In alternativa, è possibile accedere all’applicazione usando un browser attraverso http://localhost/qgis_server

Modifica del progetto QGIS¶

Ora cambierai lo stile della mappa usando QGIS.

  1. Clicca su Geospatial ‣ Desktop GIS ‣ QGIS.

  2. Scegli Progetto ‣ Apri dalla barra dei menu.

  3. Seleziona QGIS-NaturalEarth-Example.qgs e premi Open.

    • Dovresti vedere la stessa mappa del mondo resa da QGIS. Ora cambierai lo stile del fiume e guarderai il risultato nel visualizzatore di mappe web.

  4. Doppio clic su ne_10m_rivers_lake_centerlines nell’albero dei layer.

  5. Cambia il colore in Outline Options con una tonalità di blu diversa.

  6. Premi OK. I fiumi sono ora resi con un nuovo colore.

  7. Salva il progetto con Progetto ‣ Salva.

  8. Riapri il browser web.

  9. Ingrandisci. Ora vedrai i fiumi resi nello stesso nuovo colore.

Crea un nuovo progetto QGIS¶

  1. Riapri QGIS.

  2. Scegli Progetto ‣ Nuovo.

  3. Clicca su Layer ‣ Aggiungi Layer ‣ Aggiungi Layer Vettore….

  4. Naviga al set di dati /home/user/data/natural_earth2/ne_10m_admin_0_countries.shp.

  5. Premi Apri. Dovresti vedere tutti i paesi del mondo.

  6. Salva il progetto con Progetto ‣ Salva.

  7. Usa il nome del file /home/user/world.qgs e premi Salva.

  8. Open the WMS URL. Dovresti vedere tutti i paesi del mondo, questa volta visualizzati da QGIS Server.

E ora?¶

Per saperne di più su QGIS Server, un buon punto di partenza è la QGIS Server Guide.

  • -> http://localhost/cgi-bin/qgis_mapserv.fcgi
