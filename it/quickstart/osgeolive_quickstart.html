Cominciare con OSGeoLive¶

Questa guida rapida descrive come avviare OSGeoLive da un DVD o USB.

Requisiti di Sistema¶

Il vostro computer dovrebbe avere almeno:

  • RAM: minimo 2 GB (raccomandati 4 GB).

  • CPU: 1 GHz 64 bit.

  • Disco rigido:

  • Non è richiesto se eseguito direttamente dal DVD

  • 20 GB se vuoi installare il sistema operativo.

  • Mouse: un mouse con 3 pulsanti è utile, specialmente per gli utenti Mac.

Eseguire:¶

Procurati un DVD OSGeoLive, ISO disponibile da: [Download] Metti il DVD nell’unità CD del tuo computer, poi riavvia il computer.

Se il vostro computer è impostato per avviarsi dal CD (la maggior parte dei casi) dovreste avere il boot in un sistema Lubuntu, con le applicazione geospaziali installate. Altrimenti dovrete aggiustare le impostazioni di avvio all’interno del BIOS premendo il pulsante del menu di boot appena acceso il computer.

Aspettate mentre il sistema si avvia.

Provare le molte applicazioni installate dal menu Geospatial.
