Guida rapida Zoo Project¶

ZOO-Project fornisce un ambiente di sviluppo amichevole per Web Processing Service (WPS) per creare e concatenare Web Processing Services. Un WPS fornisce accesso web a funzioni che eseguono algoritmi spaziali

Avvia ZOO-Project¶

Dal menu di avvio, seleziona Geospatial ‣ Web Services ‣ ZOO Project.

Firefox sarà aperto con la pagina principale di dimostrazione di ZOO Project. Da qui, si può selezionare un’applicazione demo

Demo strumento spaziale OGR¶

  1. Sulla pagina principale delle demo del ZOO Project, cliccare visualizza su strumenti spaziali OGR.

  2. Clicca sulla mappa per selezionare un elemento dal Server WFS di GeoServer, l’elemento sarà visualizzato in blu.

  3. Cliccare su un processo con geometry semplice come Buffer, Centroid, ConvexHull o Boundary e il risultato sarà visualizzato in giallo.

  4. Ora selezionare un altro elemento vicino il primo già selezionato, quindi cliccare su un processo a geometrie multiple come Union, Intersection, Symmetric Differences o Differences per ottenere il risultato visualizzato in rosso nella mappa

Demo triangolazioni GCAL¶

  1. Sulla pagina principale delle demo del ZOO Project, cliccare visualizza su strumenti spaziali GCAL.

  2. Premere il bottone Voronoy o Delaunay per visualizzare sulla mappa il risultato del servizio WPS in esecuzione, come mostrato sotto.

Demo applicazioni OTB¶

  1. Sulla pagina principale delle demo del ZOO Project, cliccare visualizza su strumenti spaziali OTB.

  2. Compila il modulo sulla sinistra per configurate l’esecuzione del servizio WPS.

  3. Quando sono stati selezionati i valori per il processamento, premere il pulsante Esecuzione in basso nel modulo di sinistra.

  4. Se si seleziona image/png come formato di uscita, si vedranno il risultato del processo visualizzato sulla mappa. Si può vedere sotto un esempio dell’esecuzione di KMeansClassification quando si seleziona il formato di uscita image/pnf

E ora?¶

  • Si può accedere la capabilities del Server ZOO WPS:

    http://localhost/cgi-bin/zoo_loader.cgi?Request=GetCapabilities&Service=WPS

  • Per maggiori informazioni puoi contattare la mailing list di ZOO Project: zoo-discuss@lists.osgeo.org
