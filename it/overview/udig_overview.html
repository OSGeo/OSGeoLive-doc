uDig¶

Desktop GIS¶

User-friendly Desktop Internet GIS (uDig) è un visualizzatore/editore di dati spaziali open source, con particolare enfasi sugli standard OpenGIS per Internet GIS, gli standard Web Map Server (WMS) e Web Feature Server (WFS).

uDig è:

  • User-friendly, fornendo un ambiente grafico familiare per gli utenti GIS;

  • Desktop located, funzionante nativamente su Windows, Mac OS/X e Linux;

  • Internet oriented, consumando servizi web geospaziali standard e de facto; e,

  • GIS ready, fornendo la struttura su cui possono essere costruite capacità analitiche complesse, e gradualmente sussumendo queste capacità nell’applicazione principale.

Per gli sviluppatori uDig fornisce una piattaforma Java comune per costruire applicazioni spaziali con componenti open source. Il sito web fornisce una serie di chiari tutorial che coprono la realizzazione di un semplice strumento fino al rilascio della propria applicazione personalizzata.

Caratteristiche principali¶

  • Client desktop con integrazione drag and drop per File explorer e browser web

  • Integrazione con la vostra infrastruttura esistente: ArcSDE, Oracle, DB2 e altro

  • Lavora con i file locali: Shapefile, jpeg, png, tiff e altro.

  • Lavora con formati raster spaziali avanzati: ECW, MrSID, JPEG 2000

  • Supporto for Web Map Servers conformi (GeoServer, MapServer testato)

  • Esperienza web integrata con un browser incorporato che riconosce i servizi web OGC e aggiunge facilmente i link alla mappa sullo schermo

  • Controllo del descrittore di stile del rendering che ti permette di pubblicare la tua mappa usando le stesse impostazioni in uDig come con i popolari Web Map Servers

  • Integrazione profonda degli standard che permette all’applicazione di passare al servizio appropriato come richiesto per la visualizzazione, l’interazione e l’editing

  • Stampa e generazione di PDF

  • Per Sviluppatori

    • Applicazione Java costruita usando i GeoTools, JTS Topology Suite (JTS)

    • Sistema industriale di plug-in fornito da Eclipse Rich Client Platform

    • Uso di widget nativi

Standard implementati¶

  • OGC Web Map Server (WMS)

  • OGC Web Feature Server (WFS)

  • OGC Web Feature Server Transactional (WFS-T)

  • OGC Simple Features for SQL (SFQL)

  • Web Map Server Cache (WMS-C)

  • OGC Style Layer Descriptor (SLD)
