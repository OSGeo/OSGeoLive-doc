PyWPS¶

Web Processing Service¶

PyWPS fornisce accesso web per operazioni spaziali personalizzabili (denominati con Processi) attraverso lo standard Web Processing Service (WPS)

I processi sono scritti nel linguaggio di programmazione Python e possono integrare strumenti come GRASS GIS, R, GDAL/OGR, proj e altre librerie con binding Python. I dati di origine possono essere forniti con richieste WPS o accessibili dai dati sul server WPS.

Caratteristiche principali¶

  • piccolo e leggero

  • facile e veloce da installare e configurare

  • implementato interamente in Python

  • si integra perfettamente con l’ecosistema dei pacchetti di Python:

    • GRASS 7

    • R

    • GDAL/OGR

    • Proj

    • Numpy

    • Shapely
