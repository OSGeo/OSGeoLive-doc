iD editor¶

Editor per OpenStreetMap basato su browser¶

L’editor iD editor è l’editor su browser di riferimento per OpenStreetMap . iD è veloce e semplice da usare, e consente di mappare a partire da varie fonti di dati, quali immagini aeree, GPS, mappe cartacee o Mapillary.

L’editor ID è un sistema interessante per eseguire modifiche semplici e di piccola entità che non richiedono le caratteristiche avanzate del più avanzato editor JOSM.

Caratteristiche principali¶

  • OpenStreetMap Editing
  • Veloce
  • Intuitivo
  • Basato sul browser

Standards implementati¶

  • Web Map Tile Service (per le mappe di base)
