XyGrib¶

Previsioni del tempo¶

XyGrib è un lettore di file Grib e visualizza i dati meteorologici fornendo una capacità off-line per analizzare le previsioni del tempo o gli hindcast. XyGrib è la continuazione del pacchetto software zyGrib con un nuovo team di volontari. È inteso per essere usato come una stazione di lavoro meteorologica capace per chiunque abbia un serio interesse nell’esame del tempo. Questo include i membri della comunità velica, gli aviatori privati e sportivi, gli agricoltori, gli appassionati di meteo e molti altri.

Caratteristiche principali¶

  • Visualizzazione di dati meteorologici da file in formato GRIB

  • Scaricamento automatico di previsioni meteomarine

  • Scaricamento automatico di dati IAC (fleet code)

  • Visualizza animazioni di previsioni su otto giorni

  • Create le vostre mappe meteo regionali (per tutto il mondo), o visualizzate una previsione quantitativa per un punto preciso.

  • Traccia grafici a partire da dati di vento, pressione, temperatura, umidità, precipitazione, neve, copertura nuvolosa, punto di rugiada, altezza delle onde e dati di alta quota
