QGIS Server¶

Servizio Web¶

QGIS Server fornisce mappe Web e servizi di funzionalità (WMS e WFS) utilizzando le stesse librerie dell’applicazione desktop GIS (QGIS). Le mappe e i modelli di stampa creati nel desktop QGIS possono essere pubblicati come mappe web semplicemente copiando il file di progetto QGIS nella directory del server. Le mappe Web risultanti hanno esattamente lo stesso aspetto del desktop. QGIS Server viene solitamente eseguito come modulo CGI/FastCGI all’interno del Server Web Apache.

Caratteristiche principali¶

  • Web Map Service (WMS) - immagini

  • OGC Web Map Tile Service (WMTS) 1.0.0

  • Web Feature Service (WFS) - vettori

  • Output PDF integrato

  • Creazione di mappe WYSIWYG con QGIS

  • Simbolizzazione cartografica avanzata

  • Supporta lo Styled Layer Descriptor (SLD)

  • Parametri extra come ad es. FILTER, SELECTION, DPI

Standard implementati¶

  • OGC Web Map Service (WMS) 1.1.1 e 1.3.0

  • OGC Web Map Tile Service (WMTS) 1.0.0

  • OGC Web Feature Service (WFS) 1.0.0 and Transactional WFS (WFS-T) 1.0.0

  • OGC Web Coverage Service (WCS)

  • OGC Styled Layer Descriptor (SLD) 1.0
