
|osgeo-live-version| 의 이용을 환영합니다 .
================================================================================

`OSGeo-Live <http://live.osgeo.org>`_ 는  `Lubuntu <http://www.xubuntu.org/>`_ 에 기반을 두어,  다양한 오픈 소스 GIS 프로그램들을 DVD나 USB에 담아서 설치 없이 바로 실행해 볼 수 있도록 하는
 프로젝트로, 완전히 무료 소프트웨어로 구성되어 있기 때문에 분포, 복사 등이 자유롭습니다. 

 
.. image:: ../images/screenshots/800x600/osgeolive_menu.png
  :scale: 70 %
  :alt: boot select
  :align: right

데이터의 저장, 출판, 검색, 분석 등을 포함한 다양한 GIS 분석의 환경설정을 사전에 가능하게 하고,
견본의 데이터와 문서 등을 포함하고 있습니다.

다음을 따라 이용해보세요 :

#. 컴퓨터나 버츄얼 머신에 DVD 혹은 USB 드라이브를 삽입합니다.
#. 컴퓨터를 재부팅합니다.
#. 시작과 로그인을 위해서 "Enter" 키를 누릅니다.
#. "Geospatial" 메뉴를 이용합니다.

`Apple OSX <../MacInstallers/>`_ 와  `Microsoft Windows <../WindowsInstallers/>`_ 의 기반에서도 많은 기능이 제공됩니다.
