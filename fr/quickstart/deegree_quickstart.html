Guide de démarrage avec deegree¶

deegree est un logiciel open source pour les infrastructures de données spatiales et le web géospatial. deegree est une application Java pour servir des cartes (et des données vectorielles) que d’autres clients peuvent afficher.

Les composants disponibles sont :

  • Services Web : WFS, WMS, WMTS, CSW, WPS

deegree 3 est la nouvelle génération du cadre de développement Java deegree pour les applications SIG et les implémentations de services OGC. Il peut être utilisé pour construire des infrastructures géospatiales-données (IDG) basées sur le Web.

Démarrer les services Web deegree 3¶

Choisissez “Start deegree” à partir du menu Geospatial ‣ Web Services ‣ deegree. Le démarrage de l’application prendra quelques instants.

Firefox ouvre la console deegree 3 services. (Vous pouvez également l’ouvrir manuellement en pointant votre navigateur vers http://localhost:8033). Il s’agit d’une interface graphique web simple pour l’édition des fichiers de configuration de l’espace de travail deegree.

Charger un espace de travail¶

deegree 3 utilise des configurations conformes à tous les services Web basés deegree 3. Ces configurations sont stockées dans l’espace de travail.

  1. Pour gérer les espaces de travail inclus et prêts à l’emploi, cliquez sur “espaces de travail” sur le côté gauche. Vous êtes maintenant dans la gestion de l’espace de travail de l’instance en cours d’exécution deegree. Vous pouvez voir quel espace de travail est actuellement actif (un seul peut être actif à la fois), espaces de travail inactifs disponibles localement et espaces de travail officiels qui peuvent être importés si vous avez une connexion Internet.
  2. Appuyez sur démarrer sur l’espace de travail deegree-inspire-workspace.

Cet espace de travail comprend une configuration d’exemple pour les Services INSPIRE View et Download. Il se compose d’un WFS transactionnel (2.0.0 et 1.1.0) pour tous les thèmes de données de l’annexe I et d’un WMS (1.3.0 et 1.1.1) qui dessert trois couches de ces thèmes de données annexe I. On y compte quelques unités administratives néerlandaises harmonisées, des parcelles cadastrales et des adresses. Le WFS dans cet exemple agit comme un service INSPIRE Download (accès direct) et des sorties valides, harmonisées INSPIRE GML. En outre, il prend en charge plusieurs types de demandes et requêtes.

Astuce

Cet espace de travail charge des fichiers INSPIRE GML harmonisés à partir de fichiers locaux. Cela peut facilement changer pour des gestionnaires de bases de données PostGIS, Oracle Spatial ou Microsoft SQL Server comme backend.

Voir une carte¶

  1. Cliquez sur “see layers” pour ouvrir une simple carte-client. Le service est configuré pour visualiser les données et un fond de carte (rendu par OpenStreetMap).
  2. Cliquez sur le + sur le côté droit pour ouvrir un arbre de couche montrant toutes les couches disponibles. Vous pouvez activer la couche INSPIRE desservie par le WMS deegree ici.

Astuce

Le client de la carte est basé sur OpenLayers. Vous pouvez parcourir la carte en tenant le bouton de la souris gauche et faire glisser la carte. Le zoom est possible via la roue de la souris ou via les éléments de navigation sur le côté gauche.

Plus d’informations et des tutoriels étape par étape sur l’espace de travail INSPIRE peuvent être trouvés dans la documentation deegree.

Ensuite ?¶

Ce n’était qu’une brève introduction aux caractéristiques de deegree 3. Il y a plus d’informations à propos des installations de démonstration sur le site de deegree.
