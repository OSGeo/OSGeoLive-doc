Guide de démarrage avec QGIS Server¶

QGIS Server fournit un service de carte Web (WMS) basé sur les bibliothèques de l’application de bureau QGIS.

Ce guide décrit comment :

  • Mettre à jour un projet QGIS existant et l’afficher avec QGIS Server

  • Créez un nouveau projet QGIS et ouvrir le flux WMS directement dans un navigateur

Contenu

  • Démarrez la visionneuse GeoExt

  • Modifier le projet QGIS

  • Créez un nouveau projet QGIS

  • Ensuite ?

Démarrez la visionneuse GeoExt¶

OSGeoLive contient un exemple de visionneuse de cartes Web basée sur le framework Javascript GeoExt et de QGIS Server pour l’affichage de la carte.

  1. Cliquez sur Geospatial ‣ Web Services ‣ QGIS Server. Vous devriez voir une fenêtre de navigateur avec une visionneuse de carte Web montrant une carte du monde.

Vous pouvez également accéder à l’application à l’aide d’un navigateur via http://localhost/qgis_server

Modifier le projet QGIS¶

Nous allons maintenant changer le style de la carte à l’aide de QGIS.

  1. Cliquez sur Geospatial ‣ Desktop GIS ‣ QGIS.

  2. Choisissez Projet ‣ Ouvrir depuis la barre de menu.

  3. Sélectionnez QGIS-NaturalEarth-Example.qgs et appuyez sur Ouvrir.

    • Vous devriez voir la même carte du monde affichée par QGIS. Nous allons maintenant changer le style des rivières et regarder le résultat dans la visionneuse de cartes web.

  4. Double cliquez sur ne_10m_rivers_lake_centerlines dans l’arborescence Couches.

  5. Modifiez la couleur dans les options de contour pour une tonalité de bleu différente.

  6. Appuyez sur OK. Les rivières sont maintenant affichées dans une nouvelle couleur.

  7. Sauvegarder le projet avec Project ‣ Sauvegarder.

  8. Réouvrez le navigateur Web.

  9. Zoomez vers l’avant. Vous verrez maintenant des rivières affichées dans la nouvelle couleur.

Créez un nouveau projet QGIS¶

  1. Rouvrir QGIS.

  2. Choisissez Projet ‣ Nouveau.

  3. Cliquez Couche ‣ Ajouter une couche ‣ Ajouter une couche vectorielle.

  4. Accédez au dataset :file:”/home/user/data/natural_earth2/ne_10m_admin_0_countries.shp”.

  5. Cliquez sur Open. Vous devriez voir tous les pays du monde.

  6. Sauvegarder le projet avec Project ‣ Sauvegarder.

  7. Utilisez le nom du fichier /home/user/world.qgs et appuyez sur Sauvegarder.

  8. Ouvrez l”URL WMS. Vous devriez voir les pays mondiaux, cette fois affichés par QGIS Server.

Ensuite ?¶

Pour en savoir plus sur QGIS Server, un bon point de départ est le QGIS Server Guide.

Documentation :

  • https://docs.qgis.org/3.10/en/docs/server_manual/

Guide de démarrage rapide pour l’installation sur serveur Debian/Ubuntu :

  • apt-get install qgis-mapserver libapache2-mod-fcgid

  • -> http://localhost/cgi-bin/qgis_mapserv.fcgi
