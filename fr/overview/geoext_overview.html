GeoExt¶

Une boîte à outils JavaScript pour les applications riches de cartographie Web¶

GeoExt est Open Source et permet de créer des applications SIG bureautique sur le Web. Il s’agit d’un cadre JavaScript qui combine les fonctionnalités SIG d”OpenLayers avec l’interface utilisateur de la bibliothèque ExtJS fournie par Sencha.

Les composants de l’interface utilisateur comprennent un panneau Carte, un panneau de Calques sous forme d’arbre, une grille de fonctionnalités et une carte d’aperçu. Les composants GeoExt peuvent être utilisés comme n’importe quel autre composant ExtJS et peuvent être facilement configurés et étendus. Des classes de niveau inférieur sont également disponibles permettant aux fonctionnalités et aux couches OpenLayers d’être utilisées comme modèles de données ExtJS, et pour que divers types de couches de carte soient sérialisés.

GeoExt3 est double licence. La licence que vous choisissez dépend de la licence du cadre ExtJS que vous utilisez avec elle. Si vous utilisez la version GPL v3 du framework ExtJS, vous l’utiliserez sous GPLv3. Si vous possédez une licence commerciale Sencha ExtJS, vous avez le droit d’utiliser GeoExt3 sous la licence BSD.

Fonctionnalités principales¶

  • Composants d’interface

    • Panneau Carte

    • Carte d’aperçu

    • Pop-ups

    • Moteur de rendu des entités

    • Arbre de couche, légendes, et ordre des couches par drag-and-drop

  • Modèles et magasins de données

    • Intégration du modèle de données OpenLayers avec le modèle de données ExtJS permettant le tri et le filtrage des entités à l’aide des composants ExtJS

    • Magasins et modèles ExtJS pour les entités et couches OpenLayers

  • Divers

    • Intégration avec MapFish Print et GeoServer Printing Module

    • Serialisuers de couche (WMS, WMTS, XYZ)
