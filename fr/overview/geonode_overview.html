GeoNode¶

Système de Gestion de Contenus Géospatial¶

GeoNode est un système de gestion de contenu pour les données géospatiales qui fournit la création, le partage et l’utilisation collaborative des données géospatiales. Les jeux de données peuvent être téléchargés dans différents formats, les cartes peuvent être modifiées, stylisées et agrégées grâce à des outils basés sur le navigateur, cartes et métadonnées peuvent être publiées et cherchées, et les révisions, évaluations et commentaires peuvent être renseignés par les utilisateurs.

GeoNode est construit sur : GeoServer, GeoExplorer, pycsw, Django et GeoExt.

Caractéristiques principales¶

  • Découverte de données spatiales

    • Moteur de recherche spatial puissant

    • Services fédérés OGC

    • Catalogue de métadonnées

  • Importer et gérer les données géospatiales

    • Publier les données raster, vecteur et tabulaires

    • Gérer les méta-données et documents associés

    • Partager les données en sécurité et en public

    • Éditeur de données géospatiales versionnées

  • Cartographie interactive

    • Client GeoExplorer SIG

    • Éditeur de style graphique

    • Créer des cartes interactives multi-couches

    • Partager et incorporer des cartes dans des pages web

    • Imprimer des cartes en tant que PDF

  • Collaboration

    • Revue, notation et commentaire sur les données

    • Groupes d’utilisateurs

    • Flux d’activité

    • Annonces et notifications

  • … et plus!

Standards implémentés¶

  • Web Map Service (WMS)

  • Catalogue Service for the Web (CSW)

  • Web Feature Service (WFS)

  • Web Coverage Service (WCS)

  • Filter Encoding (FE)

  • Styled Layer Descriptor (SLD)

  • Geography Markup Language (GML)
