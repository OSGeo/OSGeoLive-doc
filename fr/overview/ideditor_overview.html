Editeur iD¶

Éditeur pour OpenStreetMap basé sur des technologies web¶

L’éditeur iD est l’éditeur de fait, dans le navigateur d”OpenStreetMap . iD est rapide et facile à utiliser, et il permet d’utiliser différentes sources de données comme de l’imagerie satellite, des coordonnées GPS et les Field Papers et Mapillary.

L’éditeur iD est un bon moyen pour éditer des changements simples et faciles qui ne nécessitent pas les fonctionnalités avancées d’un éditeur plus avancé comme JOSM.

Caractéristiques principales¶

  • Édition d’OpenStreetMap

  • Rapide

  • Intuitif

  • Dans le navigateur

Standards implémentés¶

  • Web Map Tile Service (pour les fonds de plan)

Détails¶

Site Internet: https://wiki.openstreetmap.org/wiki/ID

Licence: ISC

Version du logiciel: 2.2.2

Plates-formes supportées: Tout ce qui supporte un navigateur web: Windows, GNU/Linux, Mac, Android, iOS

Support: https://help.openstreetmap.org/
