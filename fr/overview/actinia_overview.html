Actinia¶

Traitement géospatial haute performance¶

Actinia est une API REST open source pour le traitement évolutif, distribué et à haute performance des données géographiques qui utilise GRASS GIS pour les tâches de calcul.

Il fournit une API REST pour traiter les images satellitaires, les séries chronologiques d’images satellitaires, les données arbitraires de raster avec des relations géographiques et les données vectorielles.

Fonctionnalités principales¶

  • L’API REST pour traiter les images satellitaires, les séries temporelles d’images satellitaires, les données raster et vectorielles.
