SAGA¶

SIG bureautique¶

SAGA (System for Automated Geoscientific Analyses) est un système d’information géographique (SIG) open source utilisé pour l’édition et l’analyse de données spatiales. Il comprend un grand nombre de modules pour l’analyse de données vecteur (point, ligne et polygone), de tables, de grilles et d’images. Entre autres, le package comprend des modules pour la géostatistique, la classification des images, les projections, la simulation de processus dynamiques (hydrologie, aménagement paysager) et l’analyse de terrain. La fonctionnalité est accessible via une interface graphique, la ligne de commande ou en utilisant l’API C++.

SAGA est en développement depuis 2001, et le centre de développement de SAGA est situé dans l’Institut de Géographie de l’Université de Hambourg, avec des contributions de la communauté mondiale croissante.

Fonctionnalités principales¶

  • Accès à un grand nombre de modules scientifiques via l’interface utilisateur graphique ou la ligne de commande
  • Import/Export vers différents formats de fichier
  • Reprojection/échantillonnage de données
  • Manipulation des données vecteur (fusion/intersection/attributs)
  • Manipulation de nuages de points à partir de données lidar
  • Données raster: interpolation, analyse des coûts,….
  • Analyse d’images: filtres, détection de bords, analyse de cluster, segmentation
  • Analyse Numérique de Terrain: génére des index géomorphométriques, réseaux de canaux, des profils, des courbes de niveau,…
  • Géostatistique: modules pour l’adaptation de variogrammes et le kriging
  • Interface utilisateur rapide pour gérer et et visualiser des données
  • Visualisation 3D
  • API C+++ simple pour la création de nouveaux modules
  • Scripting par ligne de commande, bindings pour Python
  • SAGA est accessible à partir du langage statistique R via le module RSAGA
