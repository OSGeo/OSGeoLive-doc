ncWMS¶

Web Map Service¶

ncWMS publishes multi-dimensional data (such as environmental and weather data) as an OGC compliant Web Map Service (WMS). ncWMS quickly and efficiently extracts the map images from the very large data files typically associated with environmental data.

Caractéristiques principales¶

  • Extraction de données rapide
  • Génération de séries temporelles et diagrammes de profils verticaux
  • Capacité à dessiner de la donnée, le long d’un transect
  • Reconnaissance automatique de composants vecteur et génération de flèches vecteur
  • Prise en charge d’un large éventail de projections (pour les données sources et les images cibles)
  • Support pour la création d’animations
  • Client web inclus
  • Support pour l’export vers le format Google Earth KML
  • Formats d’entrée: NetCDF, GRIB, OPeNDAP
  • Formats de sortie: PNG, JPEG, GIF animé, KML

Standards implémentés¶

  • Web Map Service (WMS) (v1.1.1, v1.3.0)
  • KML
