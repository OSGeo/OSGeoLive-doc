Guía de Inicio Rápido de OpenStreetMap¶

OpenStreetMap es un mapa libre de nivel calle del mundo, creado por una comunidad permanentemente creciente de mappers. Cualquiera puede editar OpenStreetMap. El Sitio Web Aprenda OSM proporciona guías fáciles de entender, paso a paso, para que usted pueda comenzar a contribuir a OpenStreetMap y usando los datos OpenStreetMap . La guía Aprenda OSM en el enlace de arriba paso a paso muestra cómo navegar por el Sitio Web de OpenStreetMap, vistas y mapas de impresión y regístrese para una cuenta de usuario. Después de tener tu propio nombre de usuario y contraseña, podrás añadir tus primeros puntos en el mapa, aprende a hacer encuestas de calle y trabajo con imágenes aéreas.

El resto de esta guía presenta parte del software en línea y fuera de línea utilizada para editar y representar datos OSM.

Programas que funcionan con datos OSM¶

iD¶

** iD ** es el editor en línea de predeterminado para OpenStreetMap (OSM). iD es rápido y fácil de usar y permite la asignación de datos de fuentes como la encuesta de terreno, vía satélite e imaginarios aéreas, GPS rastros o imágenes a escala vial.

  • Página de Inicio: https://www.openstreetmap.org/edit
  • Documentación: https://wiki.openstreetmap.org/wiki/ID
  • Tutorial: https://learnosm.org/en/beginner/id-editor/

JOSM¶

JOSM (Java OpenStreetMap Editor) es un avanzado editor para OpenStreetMap (OSM), escrito en Java. Es más potente y complejo que iD, puede utilizarse para la edición masiva, fuera de línea o en una conexión a Internet no fiable. Se puede extender con plugins, preconfiguraciones personalizadas, y más extensiones.

  • Página Web: https://josm.openstreetmap.de
  • Guía de usuario: https://wiki.openstreetmap.org/wiki/JOSM/Guide
  • Tutorial: https://learnosm.org/en/josm/start-josm/
  • Colección de conectores de JOSM: https://josm.openstreetmap.de/wiki/Plugins

QGIS¶

Los datos de OpenStreetMap se pueden cargar en ** QGIS ** como capa vectorial, como una funcionalidad central o a través de plugins como QuickOSM.

  • QuickOSM: Consulte la sección QuickOSM
  • Ayuda en línea: https://wiki.openstreetmap.org/wiki/QGIS#QGIS_2:_Importing_OSM_vector_layers
  • Tutorial: https://learnosm.org/en/osm-data/osm-in-qgis/

Osmosis¶

Osmosis es un programa muy capaz para realizar muchas tareas a un nivel bruto en datos OSM. Esto incluye datos de importación y exportación a bases de datos, clasificación, limpieza y creación de volcados de datos.

  • Documentación: https://wiki.openstreetmap.org/wiki/Osmosis#Usage
  • Tutorial: https://learnosm.org/en/osm-data/osmosis/

osm2pgsql¶

osm2pgsql es un programa que convierte datos de OpenStreetMap (.osm) en un formato que puede cargarse en PostgreSQL (PostGIS). A menudo se utiliza para renderizar visualmente los datos de OSM utilizando Mapnik, ya que Mapnik puede consultar PostgreSQL para datos del mapa, pero no trabaja directamente con archivos OSM.

  • Página Web: https://wiki.openstreetmap.org/wiki/Osm2pgsql
  • Tutorial: https://learnosm.org/en/osm-data/osm2pgsql/

Trabajando con datos OSM en QGIS¶

OSGeoLive viene con datos OSM para la ciudad donde se lleva a cabo la conferencia global FOSS4G. Los datos se encuentran en el sistema de archivos /usr/local/share/data/osm/ o en la base de datos osm_local de PostgreSQL.

Iniciar QGIS y echar un vistazo a los datos.

  1. Abrir la carpeta Desktop GIS.
  2. Doble clic en el icono de QGIS.
  3. Vaya al panel del navegador a la izquierda de la pantalla.
  4. Navegar hacia PostGIS y luego a la base de datos osm_local. Los datos se encuentran en el esquema público.
  5. Arrastre las capas desde el panel Navegador y suéltelos en el lienzo del mapa.

En el QGIS Quickstart se encuentra una introducción a QGIS

QuickOSM¶

QuickOSM le permite trabajar rápidamente con datos OSM en QGIS gracias a Overpass API.

  • Página de inicio: https://github.com/3liz/QuickOSM
  1. Es necesario habilitar el Plugin en QGIS. Busque QuickOSM en Plugins –> Administrar e instalar plugins en QGIS
  2. Abrir QuickOSM desde Vector –> QuickOSM –> QuickOSM
  3. Buscar todos los pubs de la ciudad del FOSS4G
  4. Elegir clave amenity, valor pub, en Buenos Aires
  5. Mostrar la consulta para ver la consulta Overpass
  6. Clic en Ejecutar consulta. Los datos resultantes se cargarán en su proyecto QGIS como datos temporales
  7. Ahora puede guardar los datos a través de Exportar -> Guardar como…. Elija Geopackage y guarde los datos como buenos_aires_pubs.

Truco

También encontrará muchos plugins de OSM para QGIS. Busque OSM en el repositorio del plugin.

Uso de OSM como mapa base en QGIS¶

XYZ Tiles¶

De forma predeterminada, QGIS viene con la capa de mapa base OpenStreetMap y se encuentra en el Panel del navegador debajo de XYZ Tiles.

QuickMapServices¶

El plugin QuickMapServices permite agregar fácilmente varios servicios de mapas como mapas base con un solo clic en QGIS.

  • Página de inicio: https://github.com/nextgis/quickmapservices
  1. Usted tiene que habilitar el Plugin en QGIS. Busque QuickMapServices en Plugins –> Administrar e instalar plugins en QGIS
  2. Agregue un mapa base OSM desde Web –> QuickMapServices –> OSM

Empezar a trabajar con datos OSM¶

Hay varias maneras de descargar datos de OpenStreetMap para su región. Puede obtener los datos en varios formatos.

Puede utilizar las siguientes herramientas y servicios para extraer datos de OSM para su área de interés.

Geofabrik¶

Geofabrik proporciona diariamente extractos de datos penstreetmap en todo el mundo por subregión y país. Elija la región o país que le interesa. Los datos se proporcionan en tres formatos: .osm.pbf, .shp.zip y .osm.bz2. .osm.pbf y`.shp.zip` pueden abrirse directamente en QGIS.

Truco

.osm.bz2 contiene archivos XML de OSM sin procesar .osm, osm2pgsql funciona bien con ellos, obtenga más información sobre cómo trabajar con ellos aquí https://wiki.openstreetmap.org/wiki/OSM_XML

Overpass turbo¶

Overpass turbo es una herramienta de minería de datos basada en web para OpenStreetMap basada en la Overpass-API.

Echa un vistazo al cliente web y descarga todos los pubs de la ciudad del FOSS4G.

  1. Abierto https://overpass-turbo.eu/
  2. Buscar todos los pubs de la ciudad del FOSS4G
  3. Haga clic en el botón Asistente y pegue la consulta a la columna de búsqueda amenity=pub in «Buenos Aires»
  4. A continuación, haga clic en el botón compilar y ejecutar consulta
  5. Clic en zoom a datos en el lienzo del mapa para navegar a sus resultados

Herramienta de Exportación HOT¶

El Equipo Humanitario de OpenStreetMap (HOT) también proporciona una herramienta de exportación para OpenStreetMap. Para usarlo necesita una cuenta de OSM. Puede descargar datos en varios formatos y tener historial para sus datos descargados y exportaciones.

Aquí hay una guía de inicio rápido: https://export.hotosm.org/en/v3/learn/quick_start
