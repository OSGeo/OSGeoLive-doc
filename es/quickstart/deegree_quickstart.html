Inicio Rápido de deegree¶

deegree es un software de código abierto para infraestructuras de datos espaciales y la web geoespacial. deegree es una aplicación Java para servir mapas (y datos vectoriales) para que otros clientes los muestren.

Componentes disponibles son:

  • Servicios web: WFS, WMS, WMTS, CSW, WPS

deegree 3 es la nueva generación del framework Java deegree para aplicaciones SIG e implementaciones de servicios OGC. Se puede utilizar para construir infraestructuras de datos geoespaciales (GDI) basadas en la web.

Iniciar deegree 3 webservices¶

Elija “Iniciar deegree” en el menú Geospatial ‣ Web Services ‣ deegree . La aplicación tardará unos minutos en iniciarse.

Firefox abre la consola de servicios deegree 3. (También puede abrir esto manualmente apuntando su navegador a http://localhost:8033). Esta es una sencilla GUI basada en web para editar los archivos de configuración del espacio de trabajo deegree.

Cargar un espacio de trabajo¶

deegree 3 utiliza configuraciones conformes a todos los servicios web basados en deegree 3. Esas configuraciones se almacenan en el espacio de trabajo.

  1. Para administrar los espacios de trabajo incluidos y listos para usar, haga clic en “workspaces” en el lado izquierdo. Ahora se encuentra en la administración del espacio de trabajo de la instancia deegree en ejecución. Puede ver qué espacio de trabajo está activo actualmente (solo uno puede estar activo a la vez), espacios de trabajo disponibles localmente inactivos y espacios de trabajo oficiales que se pueden importar si tiene una conexión a Internet.
  2. Pulse iniciar en deegree-inspire-workspace.

Este espacio de trabajo incluye una configuración de ejemplo para los Servicios de visualización y descarga de INSPIRE. Consiste en un WFS transaccional (2.0.0 y 1.1.0) para todos los Temas de datos del Anexo I y un WMS (1.3.0 y 1.1.1) que atiende a tres capas de esos Temas de datos del Anexo I. Se incluyen algunas unidades administrativas holandesas armonizadas, parcelas catastrales y direcciones. El WFS en este ejemplo actúa como un servicio de descarga de INSPIRE (acceso directo) y genera un GML de INSPIRE armonizado válido. Además, admite múltiples tipos de solicitudes y consultas.

Truco

Este espacio de trabajo carga INSPIRE GML armonizado de archivos locales. Esto se puede cambiar fácilmente a PostGIS, Oracle Spatial o Microsoft SQL Server como back-end.

Ver un mapa¶

  1. Click en “ver capas” para abrir un mapa-cliente simple. El servicio está configurado para visualizar los datos y un mapa base (prestado por OpenStreetMap).
  2. Click en + en el lado derecho para abrir un árbol de capas que muestre todas las capas disponibles. Puede activar la capa INSPIRE servida aquí por el deegree WMS.

Truco

El mapa cliente se basa en “OpenLayers<https://openlayers.org/>`_. Se puede desplazar por el mapa manteniendo pulsado el botón izquierdo del ratón y arrastrando el mapa. El zoom es posible a través de la rueda del ratón o los elementos de navegación en el lado izquierdo.

Puede encontrar más información y tutoriales paso a paso sobre el espacio de trabajo de INSPIRE en la documentación de deegree.
