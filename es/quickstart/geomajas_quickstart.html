Inicio Rápido de Geomajas¶

Geomajas es en sí mismo es un marco de aplicación que le permite construir soluciones GIS web muy diversas y avanzadas.

En este Live DVD de OSGeo hemos incluido una aplicación de Geomajas lista para usarse. Esta aplicación presenta un mapa, controladores de navegación, varias capas de datos, una barra de herramientas de cinta con herramientas SIG y el Árbol de Capas.

Iniciar Geomajas¶

En el escritorio, seleccione Browser Clients ‣ Start Geomajas. Espere a que Geomajas se abra en una ventana del navegador.

Como alternativa, puede acceder a la aplicación utilizando la siguiente dirección URL: http://localhost:8080/geomajas/.

Truco

Cuando haya terminado de usar la aplicación, seleccione Browser Clients ‣ Stop Geomajas.

Navegando por el mapa¶

Puede arrastrar el mapa con el ratón para que el área deseada se centre en su pantalla.

Para obtener un vistazo más de cerca, puede utilizar la rueda del ratón para acercar o utilizar los botones de zoom proporcionados en la parte superior izquierda de la aplicación.

También puede seleccionar un área para mostrar haciendo click en el icono de la lupa y seleccionando el área deseada (mediante la selección rectangular).

Entender la leyenda¶

En la parte superior derecha, se proporciona un cuadro de leyenda conmutable. Esta leyenda le ofrece una visión general de todas las capas cargadas actualmente en la aplicación, con la opción de activarlas o desactivarlas. Esto también muestra información más detallada sobre cada capa con respecto a los iconos y el color.

Recopilación de información¶

Las aplicaciones de Geomajas tienen la capacidad de mostrar una variedad de información dentro de las capas. Al hacer click en el mapa se mostrará la información relacionada si está disponible. Haga click en el icono de información en la parte superior izquierda para alternar entre información básica o más detallada.

Cosas para probar¶

Si desea utilizar esta aplicación de muestra para crear y personalizar su propia solución de SIG web, visite http://www.geomajas.org/client-gwt. La página de descarga siempre cuenta con las últimas y mejores aplicaciones de muestra, documentación, consejos y trucos para que usted lo haga.

¿Qué sigue?¶

Para continuar utilizando la demostración web, visite http://www.geomajas.org/demo.
Para continuar usando la aplicación independiente, visita http://files.geomajas.org/downloads/2.3.0/geomajas-standalone-gwt2.zip.
