Inicio Rápido de EOxServer¶

EOxServer es un servidor para presentar archivos de datos y metadatos de Observación de la Tierra (EO) a través de Estándares Abiertos. Se basa en una pila de software de código abierto que incluye Python, MapServer, Django/GeoDjango, GDAL, PROJ and a SpatiaLite o base de datos de PostGIS

Esta Guía de Inicio Rápido describe cómo:

  • Utilice el Cliente Web para ver, filtrar, crear subconjuntos y descargar datos de EO
  • Utilice el Cliente de Administración para inspeccionar los archivos de datos de EO

Iniciar EoxServer¶

Elija Geospatial ‣ Web Services ‣ EOxServer. Esto lanza un explorador que muestra la instancia de demostración disponible en http://localhost/eoxserver/

Trabajar con el Cliente Web¶

Click en el vínculo Cliente Web para abrir el cliente EOxServer integrado.

Dentro de este cliente puede explorar el contenido de la instancia de EOxServer. La instancia de demostración está llena de escenas de ENVISAT MERIS.

El cliente consta de la vista de mapa y widgets que contienen las interacciones principales. El área principal es el widget de mapa del cliente con un conjunto de conjuntos de datos ya mostrados. En la parte inferior, encontrará el widget de timeslider, que le permite explorar el contenido en la dimensión temporal.

El widget Selección de Capas se puede ampliar haciendo clic en el ícono de engranaje en la parte superior izquierda, que le permite activar o deshabilitar la visibilidad y configurar la representación de las capas. Además, puede mostrar u ocultar la superposición de calles y seleccionar la capa de fondo. La pestaña Filtros de este widget permite consultar diferentes valores espaciales y temporales.

De manera similar a un mapa, puede hacer zoom o mover el actual tiempo de interés visible y seleccionar un tiempo de interés dibujando en el área que contiene los puntos rojos. Tambén puede mover el cursor sobre un punto para ver el conjunto de datos ID. Haciendo click en el punto, el mapa se acerca automáticamente a la extensión del conjunto de datos.

Cuando un filtro espacial está habilitado, puede dibujar un cuadro delimitador, polígono o punto directamente en el mapa. La función dibujada se utiliza para consultar dentro de la Herramienta de Descarga.

Puede obtener más información de los servicios y metadatos disponibles en el producto seleccionado pasando el cursor sobre el producto en el widget de resultados de búsqueda y haciendo clic en el icono en la parte superior derecha.

Cuando la Herramienta de Descarga está habilitada, se envía una consulta al servidor. Esta consulta incluye la hora de interés y el cuadro delimitador seleccionados. Los resultados se muestran en el widget Descargar, donde puede seleccionar el formato, la proyección y los conjuntos de datos que desea descargar.

This was a quick introduction to the EOxServer Web Client. Please refer to the online documentation for more information on this topic.

Trabajar con el Cliente de Administración¶

Desde la página principal de EOxServer, haga click en el enlace Cliente Administrador e inicie sesión con el usuario admin y la contraseña admin.

The Admin Client is Djangos standard admin and allows you to configure the available data. Please feel free to explore the client. More information can be found in the operations” guide.

Por ejemplo, puede inspeccionar la colección que se carga de forma predeterminada haciendo clic en el vínculo «Colecciones» y, a continuación, en el vínculo «MER_FRS_1P_RGB_reduced».

Puede obtener una vista previa de las coberturas volviendo y haciendo clic en el enlace «Coberturas».
