deegree¶

Acerca de¶

deegree proporciona servicios web geoespacial basado en especificaciones de Open Geospatial Consortium (OGC) y INSPIRE Network Services . deegree es compatible con:

  • Servicios de Características Web (WFS): servidor de datos vectoriales geospaciales
  • Servicio de Cartografía Web (WMS): Servidor de renderización de mapas geoespaciales
  • Servicios Web Mapas con Mozaico (WMTS): serivdor de mapas con mozaicos pre-renderizados
  • Catalogue Services for the Web (CSW): Realiza búsquedas de conjuntos de datos geoespaciales y servicios
  • Servicios de Procesamiento WEB (WPS): Ejecuta procesos geoespaciales

Características Principales¶

WFS

  • Implementa estádares WFS 1.0.0, 1.1.0 y 2.0.0
  • Completamente transaccional (incluso para los modelos de datos enriquecidos)
  • Soporta peticiones KVP, XML y SOAP
  • GML 2/3.0/3.1/3.2 entrada y salida
  • Soporte para peticiones de GetGmlObject y XLinks
  • Excelente escalabilidad y alto rendimiento
  • Transformación de coordenadas sobre la marcha
  • Diseñado para ricos modelos de datosde abajo a arriba
  • Los backends soportan mapeo flexible de esquemas de aplicación GML a modelos relacionales
  • ISO 19107- obediente modelo de geometría: Geometrías complejas (por ejemplo curvas no-lineales)
  • Soporte avanzado de expresión de filtro basado en XPath 1.0
  • Soporta varios backends, como PostGIS, Oracle Spatial, MS SQL Server, Shapefiles o documentos de instancia GML.

WMS

  • Implementa los estándarres WMS 1.1.1 y 1.3.0
  • Amplia compatibilidad con estilos de lenguajes SLD/SE versiones 1.0.0 y 1.1.0
  • Excelente escalabilidad y alto rendimiento
  • Representación de alta calidad
  • Estilo dependiente de la escala
  • El soporte para SE elimina la necesidad de una gran cantidad de extensiones de propietario.
  • Configuración sencilla de HTML y otros formatos de salida para respuestas GetFeatureInfo
  • Utiliza el acceso a datos basado en secuencias con una huella de memoria mínima
  • Soporte casi completo para la simulación de ráster según se define en SE (con algunas extensiones)
  • Soporte completo para TIEMPO/ELEVACION y otras dimensiones para datos de raster y características
  • Admite numerosos backends, como PostGIS, Oracle Spatial, Shapefiles o documentos de instancia GML
  • Puede graficar directamente modelos de datos

WMTS

  • Implementa el estándar básico WMTS 1.0.0 (KVP)
  • Excelente escalabilidad y alto rendimiento
  • Soporta diferentes backends, como GeoTIFF, WMS remoto o jerarquías de imágenes de mosaico del sistema de archivos
  • Admite caché sobre la marcha (usando EHCache)
  • Soporta GetFeatureInfo para remoto backends WMS

CSW

  • Implementa el estándar CSW 2.0.2
  • Completamente transaccional
  • Soporta peticiones KVP, XML y SOAP
  • Excelente escalabilidad y alto rendimiento
  • Perfil de Aplicación de Metadatos ISO 1.0.0
  • La capa enchufable y modular de acceso a datos permite agregar soporte para nuevos APs y backends
  • Arquitectura de inspector modular permite para validar registros a insertar varios criterios
  • Inspectores estándar: validez del esquema, integridad del identificador, requerimientos de INSPIRE
  • Maneja todas las propiedades consultables definidas (para Dublin Core y para el perfil ISO)
  • Expresiones complejas de filtro

WPS

  • Implementa el estándar WPS 1.0.0
  • Soporta peticiones KVP, XML y SOAP
  • Capa de proveedor de procesos conectable
  • API fácil de usar para implementar procesos Java
  • Soporta todas las variantes de los parámetros de entrada/salida: literal, bbox, complejo (binario y xml)
  • Acceso de transmisión por complejos parámetros de entrada/salida
  • Procesamiento de grandes cantidades de datos con una huella de memoria mínima
  • Admite almacenamiento de respuesta documentos/parámetros de salida
  • Soporta parámetros de entrada dados en línea y por referencia
  • Compatible con respuestas RawDataOutput/ResponseDocument
  • Soporta ejecución asincrónica (con sondeo del estado del proceso)

Estándares implementados¶

  • OGC Servicio de Funciones Web (WFS) 1.1.0, 1.0.0 y 2.0.0
  • OGC Servicio de Mapas Web (WMS) 1.1.1 y1.3.0
  • OGC Servicio Web de Mosaico de Mpas (WMTS) 1.1.0
  • OGC Catalogue Service-Web (CSW) 2.0.2
  • Servicio de Procesamiento Web OGC (WPS) 1.0.0
  • OGC GML 2.12, 3.0.1, 3.1.1 y 3.2.1
