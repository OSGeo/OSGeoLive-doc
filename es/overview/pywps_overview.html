PyWPS¶

Servicio Web de Procesamiento¶

PyWPS proporciona acceso web a operaciones geoespaciales personalizadas (denominadas Procesos) a través de la Web Processing Service (WPS) estándar.

Los procesos están escritos en el lenguaje de programación Python y pueden integrar herramientas como GRASS GIS, R, GDAL/OGR, proj y otras librerías con enlaces de Python. Los datos de origen se pueden proporcionar con solicitudes WPS o se puede acceder a ellos desde los datos del servidor WPS.

Características Principales¶

  • pequeño y ligero

  • fácil y rápido de instalar y configurar

  • implementado en Python puro

  • se integra perfectamente con el ecosistema de paquetes de Python:

    • GRASS 7

    • R

    • GDAL/OGR

    • Proj

    • Numpy

    • Shapely
