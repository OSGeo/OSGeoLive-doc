OGC¶

El Consorcio Geoespacial Abierto (OGC) es un consorcio internacional de más de 500 empresas, agencias gubernamentales, organizaciones de investigación y universidades impulsadas para hacer que la información y los servicios geoespaciales (de ubicación) sean FAIR: localizables, accesibles, interoperables y reutilizables. El proceso de consenso impulsado por los miembros de OGC crea estándares geoespaciales abiertos, libres de regalías y disponibles al público. Al existir en la vanguardia, OGC analiza y anticipa activamente las tendencias tecnológicas emergentes, y ejecuta un laboratorio de Investigación y Desarrollo (I + D) colaborativo y ágil, el Programa de Innovación de OGC, que crea y prueba soluciones de prototipos innovadores para los casos de uso de los miembros. Los miembros de OGC juntos forman un foro global de expertos y comunidades que utilizan la ubicación para conectar a las personas con la tecnología y mejorar la toma de decisiones en todos los niveles. OGC se compromete a crear un futuro sostenible para nosotros, nuestros hijos y las generaciones futuras. Una mirada a los OGC Domain Working Groups muestra el amplio alcance de la actividad actual en el OGC.

La Línea Base de Estándares OGC consiste en los estándares Open Geospatial Consortium (OGC) para interfaces, codificaciones, perfiles, esquemas de aplicación y documentos de mejores prácticas. El OGC Reference Model (ORM) describe estos estándares y las relaciones entre ellos y los estándares ISO relacionados. El ORM proporciona una visión general de los estándares OGC y sirve como un recurso útil para definir arquitecturas para aplicaciones específicas.

En 2020, la familia de estándares OGC API comenzó a desarrollarse para facilitar que cualquiera proporcione datos geoespaciales a la web. Estos estándares se basan en el legado de los estándares de servicios web de OGC (WMS, WFS, WCS, WPS, etc.), pero definen API centradas en recursos que aprovechan las prácticas modernas de desarrollo web. Esta página web proporciona información sobre estos estándares en una ubicación consolidada. Estos estándares se están construyendo como «bloques de construcción» que se pueden utilizar para ensamblar API novedosas para el acceso web a contenido geoespacial. Los bloques de construcción se definen no solo por los requisitos de los estándares específicos, sino también a través de la creación de prototipos y pruebas de interoperabilidad en el Programa de Innovación de OGC.

Para obtener más detalles, consulte la lista completa de estándares OGC.
