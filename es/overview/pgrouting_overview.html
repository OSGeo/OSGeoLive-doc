pgRouting¶

Herramientas GIS¶

pgRouting extends the PostGIS/PostgreSQL geospatial database to provide geospatial routing and other network analysis functionality.

Ventajas del enfoque de enrutamiento de la base de datos son:

  • Data and attributes can be modified by many clients, like QGIS and uDig through JDBC, ODBC, or directly using Pl/pgSQL. The clients can either be PCs or mobile devices.
  • Cambios en los datos pueden ser reflejados instantáneamente a través del motor de enrutamiento. No hay necesidad de precálculo.
  • El parámetro de «cost» puede ser calculado dinámicamente a través de SQL y su valor puede venir de varios campos o tablas.

pgRouting is available under the GPLv2+ license and is supported by a growing community of individuals, businesses and organizations.

Características Principales¶

pgRouting proporciona funciones para:

  • All Pairs Shortest Path * Johnson’s Algorithm * Floyd-Warshall Algorithm
  • Shortest Path * A* algorithm * Dijkstra algorithm
  • Bi-directional Shortest Path * A* algorithm * Dijkstra algorithm
  • Distancia de conducción
  • K-Shortest Path * Yen’s algorithm
  • Vendedor viajante
  • And many more

Learn more about pgRouting in the documentation.
