actinia¶

Procesamiento geoespacial de alto rendimiento¶

Actinia es una API REST de código abierto para el procesamiento escalable, distribuido y de alto rendimiento de datos geográficos que utiliza principalmente GRASS GIS para tareas computacionales.

Proporciona una API REST para procesar imágenes satelitales, series temporales de imágenes satelitales, datos ráster arbitrarios con relaciones geográficas y datos vectoriales.

Características Principales¶

  • REST API para procesar imágenes satelitales, series temporales de imágenes satelitales, datos ráster y vectoriales.

  • Explora el tutorial.
