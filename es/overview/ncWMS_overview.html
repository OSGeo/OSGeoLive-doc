ncWMS¶

Servicio Cartográfico Web¶

ncWMS publica datos multidimensionales (como datos de clima y ambientales) como un OGC compatible con Servicio Cartográfico Web (WMS). ncWMS rápida y eficientemente extrae las imágenes del mapa de los archivos de datos muy grandes asociados típicamente con datos ambientales.

Características Principales¶

  • Extracción rápida de datos

  • Generación de series de tiempo y gráficos de perfil vertical

  • Capacidad de trazar datos a lo largo de un transecto

  • Reconocimiento automático de los componentes del vector y la generación de flechas vector

  • Soporte para una amplia gama de proyecciones (para ambas datos fuente e iágenes destino)

  • Apoyo para la creación de animaciones

  • Navegador cliente incluido

  • Apoyo para exportar a formato KML de Google Earth

  • Formatos de entrada: OPeNDAP NetCDF, GRIB,

  • Formatos de salida: PNG, JPEG, GIF, KML animados

Estándares implementados¶

  • Web Map Service (WMS) (v1.1.1, v1.3.0)

  • KML
