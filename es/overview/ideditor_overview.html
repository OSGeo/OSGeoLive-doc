Editor iD¶

Editor Navegador para OpenStreetMap¶

El editor iD es de facto, el editor basado en navegador OpenStreetMap iD es rápido y fácil de usar y permite mapeo desde varias fuentes de datos, tales como satélite e imágenes aéreas, GPS, trabajos de campo o Mapillary.

El editor iD es una gran manera de editar cambios pequeños y fáciles que no requieren las funciones de los editores más avanzados JOSM

Características Principales¶

  • Edición de OpenStreetMap
  • Rápido
  • Intuitivo
  • Basado en navegador

Normas implementadas¶

  • Web Map Tile Service (para mapas base)

Detalles¶

Sitio web: https://wiki.openstreetmap.org/wiki/ID

Licencia: ISC

Versión de software: 2.2.2

Plataformas compatibles: Cualquier sistema que soporta un navegador: Windows, GNU/Linux, Mac, Android, iOS

Soporte: https://help.openstreetmap.org/
