uDig¶

SIG de escritorio¶

User-friendly Desktop Internet GIS (uDig) es un visor/editor de datos espaciales de código abierto, con especial énfasis en los estándares OpenGIS para GIS de Internet, el servidor de mapas web (WMS) y los estándares del servidor de funciones web (WFS).

uDig es:

  • Amigable a Usuario, proporciona un entorno gráfico familiar para los usuarios de GIS;
  • Situado en Escritorio, ejecuta nativamente en Windows, Mac OS/X y Linux;
  • Orientado a Internet, consumiendo los servicios estándar web y de facto geoespaciales; y,
  • SIG listo, proporcionando el marco en el cual se pueden construir capacidades analíticas complejas y gradualmente poner esas capacidades en la aplicación principal.

Para los desarrolladores, uDig proporciona una plataforma Java común para construir aplicaciones espaciales con componentes de Código Abierto. El sitio web ofrece una serie de tutoriales claros que cubren la creación de una herramienta simple para lanzar su propia aplicación personalizada.

Características Principales¶

  • Cliente de escritorio con integración Arrastrar y Soltar para el explorador de archivos y navegador Web
  • Integración con su infraestructura existente: ArcSDE, Oracle, DB2 y más
  • Trabajar con archivos locales: Shapefile, jpeg, png, tiff y más.
  • Trabajar con formatos raster espacial avanzada: ECW, MrSID y JPEG 2000
  • Soporte para Servidores de Cartografía Web compatibles ( GeoServer, MapServer probado)
  • Experiencia web integrada con un navegador incrustado que reconoce los servicios web de OGC y suavemente agrega enlaces al mapa en pantalla
  • Control de Descriptor de Capa de Estilo de Rendering que le permite publicar su Mapa usando la misma configuración en uDig que con servidores de mapas Web populares
  • Integración profunda de estándares que permite a la aplicación intercambiar entre el servicio apropiado según sea necesario para mostrar, interactuar y editar
  • Impresión y generación de PDF
  • Para Desarrolladores
    • Aplicación de Java usando GeoTools, JTS Suite de Topología (JTS)
    • Systema de plug-in de fuerza industrial proporcionada por la plataforma cliente de Eclipse
    • Uso de widgets nativos

Estándares implementados¶

  • OGC Web Map Server (WMS)
  • Servidor Web de Características (WFS) de OGC
  • Servidor Web de Características transaccional (WFS-T) de OGC
  • Funciones simples para SQL (SFSQL) de OGC
  • Cache de Servidor de Cartografía Web (WMS-C)
  • Descriptor de Estilos de Capa (SLD) de OGC
