Jupyter Notebook¶

Mezcla de medios ricos en documentación¶

Jupyter Notebook es una aplicación web que permite crear y compartir documentos que contienen código en vivo, ecuaciones, visualizaciones y texto explicativo.

Combina dos componentes:

  • Una aplicación web que proporciona la creación interactiva basada en navegador de documentos de bloc de notas.

  • Documentos de cuadernos que almacenan una representación de todo el contenido visible en la aplicación web, incluidas entradas y salidas de cálculos, texto explicativo, matemáticas, código, imágenes, vídeos, gráficos, mapas y otras representaciones multimedia enriquecidas de objetos.

Los blocs de notas se pueden compartir con otros en GitHub, Dropbox y en el Visor de notas de Jupyter.

Los usos incluyen: limpieza y transformación de datos, simulación numérica, modelado estadístico, aprendizaje automático y mucho más.

Características Principales¶

Crear cuadernos jupyter que puedan contener:

  • Texto marcado (con encabezados, estilos, párrafos, etc.)

  • Fórmulas, matemáticas, tramas, mapas, …

  • Bibliotecas externas importadas para agregar funcionalidad

  • Código de varios lenguajes de programación, incluyendo python, R, Julia, Bash y muchos más.

Los cuadernos Jupyter de OSGeo-Live pueden utilizar funciones geoespaciales de:

  • Numpy, SciPy Matplotlib, IRIS, Cartopy, gdal, Geopandas, pyshp, Fiona, netCDF4, PostgreSQL, psycopg2, R, rpy2, y más.
