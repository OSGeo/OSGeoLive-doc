Mapbender¶

Marco Geoportal¶

Mapbender es un proyecto graduado de OSGeo. Es un marco de geoportal basado en web para publicar, registrar, ver, navegar, monitorear y otorgar acceso seguro a servicios de infraestructura de datos espaciales.

Mapbender es la punta del iceberg, una meta capa de software que proporciona acceso a los componentes de SDI que se adhieren a los estándares internacionales. Mapbender permite a los usuarios crear fácilmente clientes de navegador personalizados a partir de una amplia gama de widgets (sin necesidad de / con mínima) programación requerida. Los widgets se integran con la funcionalidad del servidor para proporcionar funciones avanzadas como proxying de seguridad, digitalización, ajuste automático y más.

Las aplicaciones de gestión de Mapbender son sencillas y fáciles de usar, proporcionando una funcionalidad altamente enfocada para el funcionamiento de componentes SDI distribuidos a gran escala. Es la herramienta preferida por los administradores de SDI que necesitan mantener, categorizar y monitorear los servicios de mapas y funciones. Mapbender otorga acceso a individuos, grupos y gestiona el acceso de máquina a máquina. La adhesión a servicios estandarizados, como OGC WMS, permite aprovechar los servicios interoperables de una multitud de plataformas de servidor.

Mapbender es basado en OpenLayers, Symfony y jQuery.

Características Principales¶

  • Crear fácilmente interfaces de mapas utilizando aplicaciones de plantilla web
    • aplicaciones para añadir/quitar/personalizar
  • Crear aplicaciones con funcionalidad de digitalización
  • Crear aplicaciones móviles
  • Imprimir mapas de alta calidad usando sus propias plantillas PDF.
  • Digitalización mediante SQL
  • Subir y compartir remotos OGC WMS (capacidades de almacenamiento en caché)
    • organizar WMS subido para superposición
    • edición de visibilidad de capa
    • orden
    • formato
  • Crear usuarios y grupos, compartir y permitir el acceso a los servicios y módulos
  • Restringir el acceso de los usuarios con el módulo de proxy OWS de seguridad vía servidores de confianza
  • Protocolos cifrados monitoreo y servicio de notificación y actualización automática de estado remoto servicios OGC
  • … y más!

Estándares implementados¶

  • Web Map Service (WMS) 1.1.1, 1.3.0 de OGC
