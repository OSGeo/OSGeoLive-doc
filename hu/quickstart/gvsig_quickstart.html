gvSIG Desktop gyorstalpaló¶

A gvSIG egy térinformatikai rendszer (GIS), egy asztali alkalmazás, melyet bármilyen térinformatikai adatok rögzítésére, tárolására, kezelésére, elemzésére és szétosztására terveztek, abból a célból, hogy komplex vezetési és tervezési problémákat oldjanak meg. A gvSIG a felhasználóbarát kezelő felületéről ismert, képes megnyitni a legelterjedtebb vektor és raszter formátumokat. Számos eszközt tartalmaz a térinformatikai adatokkal történő munkához (lekérdező eszközök, nyomtatái kép, geoprocessing, hálózatok, stb.), melyek a téradatokkal dolgozó felhasználók számára ideális eszközzé teszik.

Megjegyzés

Ha a gvSIG interfész nem válaszol, próbálja meg az Alt+f billentyűt a fájl menü megnyitásához. Ezután a felhasználói felület működni fog. Ez a hiba csak az OSGeoLive kiadásban fordul elő. A normál gvSIG telepítésnél nem jelenik meg.

Megjegyzés

Megjegzés a gvSIG 2.3.1 verzióhoz: Raszter fájlok betöltéséhez a felhasználónak írási joggal kell rendelkeznie ahhoz a mappához, melyben a fájl van. Különben az alkalmazás egy megfelelő mappát (írási joggal) fog kérni. Mivel a „user” felhasználó nem rendelkezik írási joggal a „data” mappában, szükséges egy egy olyan mappa kiválasztása ahová a felhasználó rendelkezik írási joggal, például „/home/user”.

gvSIG indítása és egy nézet létrehozása¶

  1. Válassza a gvSIG-et az alkalmazás menüből, Geospatial ‣ Desktop GIS ‣ gvSIG. Általában az alkalmazás elindulása egy percbe telik.
  2. Alapértelmezés szerint a gvSIG egy üres projektet fájlt nyit meg és a «View» dokumentum típust választja. Habár az OSGeoLive-on a gvSIG-et úgy konfiguráltuk, hogy egy alapértelmezett projekttel induljon. Kattintson a New Project gombra vagy menjen a menüben a File ‣ New Project-re. Válassza a Discard Changes, mivel nem akarjuk az legutóbbi projektet használni.
  3. Kattintson a New gombra egy nézet létrehozásához. Automatikusan megnyílik.

A nézet vetületének beállítása¶

  1. Válassza a View ‣ Properties.

  2. Az aktuális vetülethez, kattintson a guilabel:… gombra. Ez a New CRS  párbeszédablakot nyitja meg.

  3. A legördülő lista használata Type, EPSG kiválasztásra.

  4. Írja be a a koordináta referencia rendszer EPSG kódját. Ehhez a példához EPSG=4326 értéket használunk, mely a WGS84 2D földrajzi koordináták. Ezután kattintson a guilabel:Search gombra. Ezt kilistázza a kiválasztott CRS-t a keresés mező alatti táblában.

  5. Válassza az guilabel:OK gombot, hogy visszatérjen View Properties párbeszédablakhoz. Az EPSG kód most 4326 és a térképi egységek fokokra változott.

  6. Válassza a OK gombot a Project Manager-hez visszatéréshez.

    1. A nézet ablak három részből áll
  • (2) The top-left cell contains a list of vector or raster layers being used in the view (i.e. Table-of-Contents).
  • (3) The bottom-left cell displays the extent of the main view over a selected vector file.
  • (4) The right cell is the main display area where raster and vector data is rendered.

Rétegek hozzáadása a nézethez¶

  1. Kattintson az Add layer ikonra fő eszköztárban az Add layer  párbeszédablak megnyitásához.
  2. Válassza az guilabel:Add gombot a párbeszédablakban. Alapértelmezésben az első a guilabel:File fül, így az Open* fájl párbeszédablak jelenik meg.
  3. Ásson le a /usr/local/share/data/natural_earth2/` könyvtárig.
  4. Válasszon fájlokat, ahogy a képen látszik.

Megjegyzés

Alapértelmezésben az összes formátum megjelenik, így vektor és raszter fájlokat kiválaszthat a CTRL billentyű nyomvatartásával.

  1. Kattintson az OK gombra az Add layer párbeszédablakhoz visszatéréshez.
  2. Kattintson az OK gombra az Add layer párbeszédablakban a nézethez visszatéréshez.

A nézet konfigurálása¶

  1. Kattintson a Maximize ikonra az adatok jobb megjelenítéséhez.
  2. A tartalomjegyzék oszlop módosítható az egér használatával a hosszú fájl nevek megnézéséhez.
  3. Abban az esetben, ha egy raszter réteg a felül van, akkor kézzel alulra kell mozgatni.

Nagyítás egy régióra¶

  1. Alapértelmezés szerint a Zoom in eszköz a kiválasztott a fő eszköztárban.
  2. Használja az egeret dél-kelet Ausztrália köré egy befoglaló téglalap rajzolásához. Úgy csinálja ezt, hogy a téglalap bal-felső sarkát kijelöli az egérrel, majd az egérgombot nyomva tartva húzza a kiválasztott terület felett a téglalap jobb-alsó sarkáig. Engedje el a bal egérgombot amikor a megfelelő területet megadta.

Réteg megjelenítés módosítása¶

Felület átlátszóvá tétele¶

  1. Válassza ki a ne_10m_admin_1_states_provinces.shp réteget a bal egérgombbal és kattintson ismét a jobb egérgombbal.
  2. Válassza a Properties almenüt.
  3. A Layer properties párbeszédablak jelenik meg.

Vektor rétek jelrendszerének módosítása¶

  1. Válassza a guilabel:Symbols fület a Layers properties párbeszédablak tetején.
  2. Válassza a guilabel:Choose symbol gombot, hogy a Symbol selector  párbeszédablak megjelenjen.

Szüntesse meg a szimbólum kitöltést és módosítsa a körvonal színét¶

  1. Kapcsolja ki a Fill color négyzetet, hogy a területek átlátszóak legyenek.
  2. Kattintson a …-ra a Border color mellett.
  3. Válassza a fehéret mint szín.
  4. Kattintson a OK gombra, hogy a Symbol selector párbeszédablakhoz visszatérjen.
  5. Kattintson a OK gombra, hogy a Layers properties párbeszédablakhoz visszatérjen.

Vektor réteg címkézésének módosítása¶

  1. Válassza a Labelling fület a Layers properties párbeszédablak tetején.
  2. Módosítsa a szöveg attribútumokat képen láthatókra, majd kattintson a Accept gombra.

Eredmény megtekintése¶

Vegye észre, hogy ez egy nagyon egyszerű nézet, amely egy pontot, egy vonalat és egy sokszögvektort tartalmazó fájlt jelenít meg egy raszter felett. Ez ugyanolyan egyszerű, ahogy egy légi fénykép vagy digitális terep modell háttér előtt a különböző formátumokban tárolt vektor adatok megjelenítése.

Áttekintő vagy lokátor hozzáadása¶

Válassza a View ‣ Locator setup-ot.

  1. Kattintson a Add layer… gombra.
  2. A réteg hozzáadás ablakban kattintson az Add gombot.
  3. Válassza a HYP_50M_SR_W.tif réteget.
  4. Kattintson az Open gombra a réteg hozzáadása ablakhoz visszatéréshez.
  5. Kattintson az Ok gombra és zárja le a lokátor konfiguráció ablakot.

Városok kiválasztása egy régióban¶

Válassza ki a régiót¶

  1. Válassza ki az ne_10m_admin_1_states_provinces.shp réteget a bal egérgomb használatával.
  2. Válassza a Select by point ikont a fő eszközsorban.
  3. Kattintson az Új Dél Wales államot reprezentáló területre. A terület sárgára vagy más színűre változik a felhasználói beállításoknak megfelelően.

Nyissa meg a réteggel szelektálás eszközt¶

  1. Válassza az ne_10m_populated_places.shp réteget a bal egérgombbal.
  2. Válassza a Selection ‣ Select by layer-t a Selection by Layer párbeszédablak megnyitásához

Válassza ki a régióba eső városokat¶

  1. Módosítsa az első kiválasztási feltételt a Selection by Layer párbeszédablak bal oldali legördülő listájában, ahogy a képen látja.
  2. Módosítsa a második szelekció feltételt, ahogy a képen látszik.
  3. Kattintson a New set gombra a kiválasztott területbe eső városok kiválasztásához.
  4. Válassza a Cancel gombot a Selection by Layer párbeszédablakban a térkép nézethez visszatéréshez.

Szüntesse meg a terület kiválasztását, hogy lássa mi történt¶

  1. Válassza ki az ne_10m_admin_1_states_provinces.shp réteget a bal egérgomb használatával.
  2. Válassza a «Clear selection» ikont a fő eszközsorban.
  3. Most láthatja, hogy csak Új dél Wales-en belüli városokat választottuk ki.

Kiválasztott elemek exportálása egy új shape fájlba¶

Válassza ki az output fájl formátumot¶

  1. Válassza az ne_10m_populated_places.shp réteget a bal egérgombbal.
  2. Válassza a Layer ‣ Export to… menüpontot az export indításához.
  3. Válassza a Shape Format-ot.
  4. Kattintson a Next-re.

Válassza ki az output fájl nevét¶

  1. Válasszon egy mappát és írja be a fájl nevét.
  2. Kattintson az Open-re majd a Next-re.

Referencia rendszer¶

  1. Válassza az Original (EPSG:4326) beállítást a nézet referencia rendszerének megőrzéséhez.
  2. Kattintson a Next-re.

Export beállítások¶

  1. Válassza a Selected features, hogy csak az Új dél Wales-be eső városokat exportálja.
  2. Kattintson az Export-ra.

Eredmény megtekintése¶

  1. Az új réteget hozzáadtuk a nézethez.
  2. Annak érdekében, hogy ellenőrizze, hogy az export rendben volt, állítsa át az eredeti fájlt láthatatlanná. Csak Új-Dél-Wales városait kell látnia.

Új térkép létrehozása¶

  1. Válassza a Map dokumentum típust a Project Manager-ben.
  2. Kattintson a New gombra egy térkép létrehozásához.
  3. Egy új ablak jelenik meg, ahol kiválaszthatja a térképre beszúrni kívánt nézetet. Válassza ki az egyetlen nézetet, amellyel rendelkezik, és nyomja le az „Accept* gombot.
  4. A térkép a kijelölt nézettel jelenik meg.

Munka a térképekkel¶

  1. A fő eszköztár ikonjaival vagy a Map menü almenüivel további elemeket adhat hozzá a térképhez, például egy léptéket és egy északi nyilat.
  2. A térképet kinyomtathatja vagy exportálhatja PDF-be vagy Postscriptbe, hogy más munkába beillessze.
  3. Válassza a Close window ikont a Project manager-hez visszatéréshez

Projektjének mentése és kilépés a gvSIG-ből¶

  1. A későbbi használathoz a projektek elmenthetők a File ‣ Save as… menüponttal vagy
  2. A projektekből kiléphet vagy lezárhatja a File ‣ Exit menüponttal.

Egy bővítmény telepítése¶

Nyissa meg a bővítmény kezelőt¶

  1. Válassza a Tools ‣ Addons manager-t. Kis idő múlva az Install package ablak jelenik meg.
  2. Az Install package ablakon belül válassza a Installation from URL lehetőséget.
  3. Válassza a képen látható repo URL-t.
  4. Kattintson a Next gombra.

Bővítmény keresés¶

  1. A bővítmény kezelőben modulokat és szimbólum könyvtárakat találunk.
  2. Gépelje be a Fast filter szövegmezőbe «Forestry».
  3. Kattintson a Symbols: Forestry bővítményre (ez egy szimbólum könyvtár)
  4. Kattintson a Next-re.

Telepítés befejezése¶

  1. Kattintson a Start downloading-ra.
  2. Majd kattintson a Next-re.
  3. Végül kattintson a Finish-re.

Bár az üzenet azt jelzi, hogy újraindításra van szükség, a szimbólum könyvtárakhoz ez nem szükséges. Csak modul telepítésekor kell újraindítani.

OpenStreetMap réteg hozzáadása¶

  1. Most hozzáadjuk az OpenStreetMap térképet. Egy új nézetben, EPSG 3857, kattintson az Add layer… gombra.
  2. Válassza az OSM fület.
  3. Válassza a «Mapnik»-et az elérhető rétegek közül.
  4. Kattintson az Ok-ra.

Eredmény megtekintése¶

Egy új réteget adtunk a nézethez. Egy régióra ránagyítva láthatjuk a részletes OpenStreetMap adatokat.

WMTS réteg hozzáadás¶

A WMTS az OGC WMS szabvány egy továbbfejlesztése, mely csempék kezelésén alapul.

Válasszon egy szervert¶

  1. Egy nézeten belül kattintson a Add layer… gombra.
  2. Válassza a WMTS fület.
  3. Válassza a képen látható URL-t.
  4. Kattintson a Connect-re.
  5. Kattintson a Next-re.

Válasszon egyet az elérhető rétegekből¶

  1. Válasszon egyet a rendelkezésre álló rétegek közül (pl. bluemarble)
  2. Kattintson az Add-ra.
  3. Kattintson a Next-re.

Válasszon képformátumot és referencia rendszert¶

  1. Válassza az image/jpeg-et a képformátumnak.
  2. Válassza az EPSG:4326-ot (vegye figyelembe, hogy a nézethez ugyanez az SRS kell, ha ez nem igaz lépjen vissza, módosítsa és kezdje elölről).
  3. Kattintson az Accept-re.

Egy új réteget adott a nézethez.
