QGIS Server gyorstalpaló¶

A QGIS Server egy internetes térkép szolgáltatást (WMS) biztosít a QGIS asztali változat könyvtáraira alapozva.

Ez a gyorstalpaló leírja:

  • egy létező QGIS projekt aktualizálását és QGIS Serverrel megjelenítését
  • egy új QGIS projekt létrehozását és a WMS közvetlen megnyitását a böngészőből

Indítsa el a GeoExt nézegetőt¶

Az OSGeoLive tartalmaz egy webes térkép megjelenítő példát, mely a GeoExt Javascript keretrendszeren alapul és a QGIS Servert használja a a térkép megjelenítésre.

  1. Click Geospatial ‣ Web Services ‣ QGIS Server. You should see a browser window with a web map viewer showing a world map.

Alternatív megoldásként a böngészőből is elérheti az alkalmazást a http://localhost/qgis_server címen.

QGIS projekt szerkesztése¶

Most megváltoztatjuk a térkép stílusát a QGIS használatával.

  1. Kattintson ide Geospatial ‣ Desktop GIS ‣ QGIS.
  2. Choose Project ‣ Open from the menu bar.
  3. Válassza a QGIS-NaturalEarth-Example.qgs-t nyomja meg az Open gombot.
    • Ugyanazt a világtérképet kell látnia a QGIS-ban. Megváltoztatjuk a folyók stílusát és megnézzük az eredményt a webes térkép megjelenítőben.
  4. A réteg fában kattintson duplán a ne_10m_rivers_lake_centerlines elemre.
  5. Módosítsa a színt a körvonal beállításokban egy másik kék tónusra.
  6. Press OK. Rivers are now rendered in a new color.
  7. Mentse el a projektet Project ‣ Save.
  8. Reopen the web browser.
  9. Zoom in. You will now see rivers rendered in the same new color.

Új QGIS projekt létrehozása¶

  1. Reopen QGIS.
  2. Válassza a Project ‣ New-t a menüből.
  3. Click Layer ‣ Add Layer ‣ Add Vector Layer.
  4. Tallózzon az adatokhoz /home/user/data/natural_earth2/ne_10m_admin_0_countries.shp.
  5. Press Open. You should see all world countries.
  6. Mentse el a projektet Project ‣ Save.
  7. Használja a /home/user/world.qgs fájlnevet és nyomja meg a Save gombot.
  8. Open the WMS URL. You should see all world countries, this time rendered by QGIS Server.

Mi a következő?¶

További QGIS Server tanuláshoz a QGIS User Guide egy jó kiindulási pont.

Dokumentáció:
  • https://docs.qgis.org/3.4/en/docs/user_manual/working_with_ogc/server/index.html

Gyorstalpaló a szerver telepítéshez Debian/Ubuntu rendszereken:

  • apt-get install qgis-mapserver libapache2-mod-fcgid
  • -> http://localhost/cgi-bin/qgis_mapserv.fcgi
QGIS Web kliens:
  • https://github.com/qgis/qgis-web-client
