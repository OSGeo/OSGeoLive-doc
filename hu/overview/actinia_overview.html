actinia¶

Nagy teljesítményű térinformatikai feldolgozás¶

Az Actinia egy nyílt forráskódú REST API térinformatikai adatok skálázható, megosztott, nagy teljesítményű feldolgozásához, mely GRASS GIS-t használ a számítási feladatokhoz.

Egy REST API-t biztosít a műholdképek, műholdkép idősorok, térbeli pozícióval bíró tetszőleges raszter adatok és vektor adatok feldolgozására.

Alapfunkciók¶

  • REST API to process satellite images, time series of satellite images, raster and vector data.

  • Fedezze fel az oktatóanyagot
