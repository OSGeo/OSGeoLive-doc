ncWMS¶

Web térkép szolgáltatás¶

Az ncWMS többdimenziós adatokat publikál (mint például környezeti és időjárás adatok) egy OGC szabványnak megfelelő web térkép szolgáltatásként (WMS). Az ncWMS gyorsan és hatékonyan állítja elő a képeket a nagyon nagy adatfájlokból, melyek tipikusan a környezeti adatokhoz kapcsolódnak.

Alapfunkciók¶

  • Adatok gyors előhozása
  • Idősorok és függőleges metszetek generálása
  • Adatok ábrázolása egy keresztmetszet mentés
  • Vektor komponensek automatikus felismerése és vektor nyilak generálása
  • A vetületek széles tartományának támogatása (a forrás adatok és az előállított képek esetén is)
  • Támogatás animációk létrehozásához
  • Böngésző klienst tartalmaz
  • Exportálás Google Earth KML formátumba
  • Input formátumok: NetCDF, GRIB, OPeNDAP
  • Output formátumok: PNG, JPEG, animált GIF, KML

Implementált szabványok¶

  • Web Map Service (WMS) (v1.1.1, v1.3.0)
  • KML
