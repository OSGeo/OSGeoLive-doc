52 North WPS¶

Névjegy¶

Az 52°North Web Processing Service (WPS) webes hozzáférést tesz lehetővé a Sextante, ArcGIS szerver, R, GRASS 7 által biztosított és egyedileg kidolgozott feldolgozási algoritmusokhoz. Az algoritmus olyan egyszerű is lehet mint az influenzás esetek különbsége két különböző időszakban vagy olyan komplikált mint egy globális klímaváltozási modell.

Alapfunkciók¶

GIS alapfüggvények:

  • SimpleBufferAlgorithm
  • DouglasPeuckerAlgoritmus
  • IntersectionAlgorithm
  • AddRasterValuesAlgorithm
  • CoordinateTransformAlgorithm

Háttér algoritmusok elérhetők:

  • Sextante
  • ArcGIS Server
  • R
  • GRASS 7

Támogatott input és output formátumok:

  • Vektor adatok: GML2, GML3, SHP fájlok, KML, WKT (Well-known Text), DGN, JSON
  • Raszter adatok: Geotiff, AsciiGrid, NetCDF, JPEG, PNG, HDF-EOS, ERDAS HFA
  • Az eredmények GeoServerben vagy MapServerben tárolhatók
