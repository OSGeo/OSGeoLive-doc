pycsw¶

Metaadat katalógus¶

A pycsw lehetővé teszi a felhasználók számára, hogy közzétégyék az adatleírások (metaadatok) katalógusát szabványon alapuló interfészek és formátumok használatával, így a katalógushoz kapcsolódó kliensek lekérdezhetik és frissíthetik azokat. A pycsw-t egyszerűen beállíthatjuk egy Apache webszerveren (vagy bármilyen WSGI szerveren), és hozzáférhetünk az adatbázisban XML fájként tárolt metaadat-rekordokhoz.

A metaadat «adatok az adatokról», olyan adatokat tartalmaz mint a létrehozás dátuma, szerző, cím, érdekeltségi kör, stb. és általában XML fájlokban kódolják, a nemzetközi szabványokat követve, mint ISO 19115, ISO 19139 és FGDC CSDGM.

A pycsw az Catalogue Service Python implementációja . A projekt OGC Compliant tanúsítvánnyal rendelkezik, és egy OGC Reference Implementation. A CSW-szabvány közös interfészeket határoz meg a téradatok, a webes szolgáltatások és a kapcsolódó erőforrások metaadatainak felfedezésére, böngészésére és lekérdezésére.

A Pywcs nyílt forráskódú, MIT licenc alatt adják ki és fut az összes nagyobb platformon (Windows, Linux, Mac OS X).

Alapfunkciók¶

  • hitelesített OG megfelelőség és OGC referencia implementáció az OGC CSW 2.0.2 ls CSW 3.0.0 szolgáltatásokhoz

  • támogatás WMS, WFS, WCS, WPS, WAF,CSW, SOS szolgáltatásokhoz

  • INSPIRE Discovery Services 3.0-t implementálja

  • Az ISO Metadata Application Profile 1.0.0-t implementálja

  • A FGDC CSDGM Application Profile for CSW 2.0-t implementálja

  • Implementálja a Search/Retrieval via URL (SRU) protokollt

  • Implementálja a Full Text Search képességeket

  • Implementálja az OGC OpenSearch Geo/Time bővítményeket

  • implementálja az Open Archives Initiative Protocol for Metadata Harvesting-et

  • támogatja az ISO, Dublin Core, DIF, FGDC, Atom és GM03 metaadat modelleket

  • CGI vagy WSGI telepítés

  • Python 2 és 3 kompatibilis

  • egyszerű konfigurálás

  • tranzakciós képességek (CSW-T)

  • rugalmas tárház konfiguráció

  • GeoNode kapcsolat

  • HHypermap kapcsolat

  • Open Data Catalog kapcsolat

  • CKAN kapcsolat

  • elosztott katalógus keresés

  • valós idejű XML séma validáció

  • bővíthető modul architektúra

  • nagyon könnyűsúlyú (Python + CGI)

Implementált szabványok¶

OGC szabványok:

  • Catalogue Service 2.0.2, 3.0.0

  • Filter Encoding (FE) 1.1.0, 2.0.0

  • OWS Common 1.0.0, 2.0.0

  • Geography Markup Language (GML) 3.1.1

  • Simple Feature Access (SFS) 1.2.1

  • OGC OpenSearch 1.0

Metaadat szabványok:

  • Dublin Core 1.1

  • SOAP 1.2

  • ISO 19115:2003

  • ISO 19139:2007

  • ISO 19119:2005

  • NASA DIF 9.7

  • FGDC CSDGM 1998

  • GM03 2.1

  • SRU 1.1

  • OAI-PMH 2.0

  • A9 OpenSearch 1.1
