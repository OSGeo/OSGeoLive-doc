MapCache¶

Web szolgáltatás¶

A MapCache egy szerver, mely csempe gyorsítótárazást valósít meg a WMS rétegek hozzáférésének gyorsítására. Az elsődleges cél, hogy gyors és egyszerűen telepíthető legyen, miközben elvárható alapvető funkciókat (és többet!) biztosítson egy csempe gyorsítótárazás megoldásából.

A MapCache a MapServer család része, de beállítható, hogy bármely WMS-t használó térinformatikai szerverhez gyorstárazott adatokat szolgáltasson. MapCache mint Apache modul futtatható vagy bármely web szerveren, amely támogatja a CGI/FastCGI-t, mint az IIS, Nginx, és Jetty.

Alapfunkciók¶

  • Gyorstárazott csempék tárolását teszi lehetővé
    • háttértárolón
    • felhőben REST használatával (S3, Azure, Google)
    • adatbázisban többek között SQLite, Berkeley DB, Riak
    • Memcache-ben
  • Tartalmaz egy feltöltő alkalmazást
  • Dimenziók alapján történő gyorsítótárazás (például idő)
  • Lehetővé teszi Mapbox Vector (MVT) és UTFGrid csempék gyorsítótárazását
  • Testreszabható rácsok
  • Lehetővé teszi a csempék „összegyűjtését” - megengedi a különböző adatállományok csempéinek összevonását egyetlen csempét küldi az ügyfélnek
  • Tartalmaz egy feltöltő alkalmazást, mellyel a parancssorból előre feltöltheti a gyorstárat
  • Lehetővé teszi a kérések proxyn keresztüli visszaküldését a kiszolgálóhoz
  • Egy beépített demó intefész

Implementált szabványok¶

  • Számos Open Geospatial Consortium (OGC) szabvány támogatása
    • Web Map Service (WMS)
    • OpenGIS Web Map Tile ServiceW (WMTS)
  • További szabványok:
    • TMS
    • Google térkép XYZ
    • KML SuperOverlays
