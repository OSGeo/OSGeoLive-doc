uDig¶

Asztali GIS¶

Felhasználóbarát asztali internet GIS (uDig) egy nyílt forráskódú téradat megjelenítő/szerkesztő, különös tekintettel az OpenGIS internetes GIS szabványaira Internet, a Web Map Server (WMS) és a Web Feature Server (WFS) szabványok .

A uDig:

  • Felhasználóbarát, ismerős grafikus felhasználói környezetet biztosít;
  • Asztali gépekre natív módon fut Windows, Max OS/X és Linux rendszereken;
  • Internet orientált, szabványos és de facto térinformatikai webszolgáltatásokból táplálkozik; és,
  • GIS kész, amely biztosítja a keretrendszert , melyre alapozva komplex analitikai képességek építhetők. és fokozatosan bekerülnek ezek a képességek a fő alkalmazásba.

A fejlesztőknek a uDig egy közös Java platformot biztosít térinformatikai alkalmazások készítéséhez, nyílt forráskódú összetevőkkel. A honlap világos oktatóanyagok sorozatát kínálja, melyek lefedik az egyszerű eszköz készítésétől a saját egyedi alkalmazás kiadásáig terjed skálát.

Alapfunkciók¶

  • Asztali kliens Fogd és vidd integrációval a fájl kezelőhöz és Web böngészőhöz
  • Integrálható a meglévő infrastruktúrába: ArcSDE, Oracle, DB2, és továbbiak
  • Helyi fájlokkal dolgozik: shape fájl, jpeg, png, tiff és így tovább.
  • Együtt dolgozik a modern raszter formátumokkal: ECW, MrSID, JPEG 2000
  • Támogatás megfelelő web térkép kiszolgálókhoz (GeoServer, MapServer testelt)
  • Integrált Web tapasztalat egy beágyazott böngészőben, amely felismeri az OGC webes szolgáltatásokat és egyszerűen hozzáadja a linkeket a képernyőn megjelenő térképhez
  • A rajzolás Style Layer Description-nal történő vezérlése lehetővé teszi, hogy a térképeit ugyanazokkal a beállításokkal tegye közzé a web térkép szervereken mint a uDig-ben
  • Szabványok alapos integrációja, amely lehetővé teszi az alkalmazásoknak, hogy szükség szerint váltsanak a megfelelő szolgáltatások között a megjelenítéshez, interakcióhoz és szerkesztéshez
  • Nyomtatás és PDF generálás
  • Fejlesztőknek
    • Java alkalmazás, mely a GeoTools, JTS Topology Suite (JTS) <https://sourceforge.net/projects/jts-topo-suite/>`_ felhasználásával készül
    • Az Eclipse Rich Client Platform által biztosított Ipari erősségű modul rendszer
    • Natív vezérlők használata

Implementált szabványok¶

  • OGC Web Map Server (WMS)
  • OGC Web Feature Server (WFS)
  • OGC Web Feature Server Transactional (WFS-T)
  • OGC Simple Features for SQL (SFQL)
  • Web Map Server Cache (WMS-C)
  • OGC Style Layer Descriptor (SLD)
