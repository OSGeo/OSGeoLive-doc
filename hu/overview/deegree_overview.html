deegree¶

Névjegy¶

degree térinformatikai web szolgáltatásokat biztosít az Open Geospatial Consortium (OGC) és az INSPIRE Network Services specifikációra alapozva. A degree támogatja a következőket:

  • Web Feature Service (WFS): téradatokból vektor adat szolgáltatás
  • Web Map Service (WMS): téradatokból kirajzolt térkép szolgáltatása
  • OpenGIS Web Map Tile ServiceW (WMTS): előre kirajzolt térképcsempék szolgáltatása
  • Catalogue Service: Téradatok és szolgáltatások keresése
  • Web Processing Service (WPS): térinformatikai feldolgozás végrehajtása

Alapfunkciók¶

WFS

  • WFS 1.0.0, 1.1.0 és 2.0.0szabványok implementálása
  • Teljes tranzakció kezelés (még gazdag adatmodellekre is)
  • KVP, XML és SOAP kérések támogatása
  • GML 2/3.0/3.1/3.2 output/input
  • GetGmlObject kérések és XLinks támogatás
  • Nagy teljesítmény és tökéletes skálázhatóság
  • Vetületi transzformáció röptében
  • Alulról felfelé tervezték a gazdag adatmodellekhez
  • A háttérprogramok támogatják a GML alkalmazási sémák rugalmas illesztését a relációs modellekhez
  • ISO 19107-nek megfelelő geometria modell: Komplex geometriák (pl. görbék)
  • XPath 1.0-ra alapozott haladó szűrő kifejezések
  • Számos háttér rendszert támogat, mint például PostGIS, Oracle Spatial, MSSQL Server, Shape fájlok vagy GML dokumentumok

WMS

  • WMS 1.1.1 és 1.3.0 szabványok implementálása
  • Az SLD/SE 1.0.0 és 1.1.0 verziójú stílusnyelvek széleskörű támogatása
  • Nagy teljesítmény és tökéletes skálázhatóság
  • Kiváló minőségű kirajzolás
  • Méretarány függő stílusok
  • Az SE támogatás szükségtelenné tesz egy csomó kereskedelmi bőviményt
  • Könnyen konfigurálható HTML és más output formátumok a GetFeatureInfo válaszokhoz
  • Stream alapú adatelérést használ, minimális memóriahasználattal
  • Szinte teljes mértékben támogatják az SE-ben definiált raszteres jelrendszert (néhány bővítménnyel)
  • Idő/magasság és más dimenziók , teljes támogatása a vektoros és a raszteres adatokra
  • Számos háttérprogramot támogat, mint például PostGIS, Oracle Spatial, SHP-fájlok vagy GML dokumentumok
  • Gazdag adatmodelleket közvetlenül meg tud jeleníteni

WMTS

  • Az alap WMTS szabvány 1.0.0 (KVP) implementálása
  • Nagy teljesítmény és tökéletes skálázhatóság
  • Különböző háttérprogramok támogatása, mint például GeoTIFF, távoli WMS vagy a hierarchikus csempe képek a fájlrendszerben
  • Támogatja a röptében gyorsító tárazást (EHCache használatával)
  • Támogatja a GetFeatureInfo kérést a távoli WMS-ektől

CSW

  • CSW 2.0.2 szabvány implementálása
  • Teljesen tranzakciós
  • KVP, XML és SOAP kérések támogatása
  • Nagy teljesítmény és tökéletes skálázhatóság
  • ISO Metadata Application Profile 1.0.0
  • A hozzáadható és moduláris adathozzáférés réteg lehetővé teszi, hogy új API-khoz és háttérprogramokhoz támogatást adjunk
  • A moduláris ellenőr architektúra lehetővé teszi, hogy a beillesztendő rekordokat ellenőrizzük különböző kritériumok alapján
  • Standard ellenőrzők: séma validálás, azonosító integritás, INSPIRE követelmények
  • Kezeli az összes definiált lekérdezhető tulajdonságot (Dublin Core, valamint ISO profil)
  • Komplex szűrő kifejezések

WPS

  • WPS 1.0.0 szabvány implementálása
  • KVP, XML és SOAP kérések támogatása
  • Beilleszthető folyamat szolgáltató réteg
  • Könnyen használható API a Java folyamatok implementálásához
  • Támogatja a bemeneti/kimeneti paraméterek összes változatát : szöveges, bbox, összetett (bináris és xml)
  • Adatfolyam-hozzáférés komplex bemeneti/kimeneti paraméterekhez
  • A hatalmas mennyiségű adat feldolgozása minimális memória használattal
  • Támogatja a válasz dokumentumok/kimeneti paraméterek tárolását
  • Támogatja az inline és hivatkozással megadott bemeneti paramétereket
  • Támogatja a RawDataOutput/ResponseDocument válaszokat
  • Támogatja az aszinkron végrehajtást (a folyamat állapot folyamatos lekérdezésével)

Implementált szabványok¶

  • Web Feature Service (WFS) 1.0.0, 1.1.0 és 2.0.0
  • Web Map Service (WMS) 1.1.1 és 1.3.0
  • OpenGIS Web Map Tile ServiceW (WMTS) 1.1.0
  • OGC Catalogue Service-Web (CSW) 2.0.2
  • Web Processing Service (WPS) 1.0.0
  • Geography Markup Language (GML) 2.12, 3.0.1, 3.1.1 és 3.2.1
