t-rex¶

Vektor csempe csolgáltatás¶

A t-rex saját adatokból történő MVT csempék publikálására specializált vektor csempe szolgáltatás

Alapfunkciók¶

  • PostGIS adatbázisok és GDAL vektor formátumok támogatása

  • Rétegek automatikus detektálása az adatforrásban

  • Beépített megjelenítő az adatok megtekintésére és vizsgálatára

  • Csempe generáló parancs egyszerű párhuzamosítással

  • Automatikus vetületi átszámítás

  • Egyéni csempe rács támogatása

Részletek¶

Honlap: https://t-rex.tileserver.ch/

API interfészek: http, Mapbox Vector Tiles
