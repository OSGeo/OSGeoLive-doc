iD editor¶

Böngésző alapú OpenStreetMap szerkesztő¶

Az iD szerkesztő a de facto, böngésző alapú OpenStreetMap szerkesztő. Az iD gyors és könnyen használható és lehetővé teszi a térképezést különböző adat forrásokból mint például űrfelvételek, légifényképek, GPS, papír térképek vagy Mapillary.

Az iD szerkesztő egy nagyszerű lehetőség a kis és egyszerű módosítások elvégzéséhez, melyek nem igénylik a JOSM szerkesztő gazdagabb funkcionalitását.

Alapfunkciók¶

  • OpenStreetMap szerkesztés

  • Gyors

  • Intuitív

  • Böngésző alapú

Implementált szabványok¶

  • Web Map Tile Service (alatérképekhez)
