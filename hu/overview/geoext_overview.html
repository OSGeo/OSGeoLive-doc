GeoExt¶

Egy JavaScript eszköztár gazdag web térkép alkalmazásokhoz¶

A GeoExt nyílt forráskódú és asztali alkalmazáshoz hasonló, a weben elérhető GIS készítését teszi lehetővé. Ez egy JavaScript keretrendszer, mely kombinálja az OpenLayers GIS funkcionalitását az ExtJS könyvtár felhasználói felületével, melyet a Sencha szolgáltat.

Felhasználói felület komponensei tartalmazzák a térkép panelt, a rétegfa panelt, az adattábla rácsot és az áttekintő térképet. A GeoExt komponensek éppen ugyanúgy használhatóak mint bármelyik másik ExtJS komponens és könnyen konfigurálhatók és kibővíthetők. Vannak alacsony szintű osztályok, melyek lehetővé teszik OpenLayers térképi elemek és rétegek ExtJS adatmodellként használatát és különféle térképi rétegtípusok szerializálását.

Alapfunkciók¶

  • UI komponensek

    • Térkép panel

    • Áttekintő térkép

    • Felbukkanó ablakok

    • Térképi elem megjelenítő

    • Rétegfa, jelmagyarázat és rétegsorrend módosítás húzással

  • Adatmodellek és adatárak

    • OpenLayers data modell integráció az ExtJS adatmodellel, ExtJS komponensek felhasználásával a térképi elemek rendezése és szűrése

    • ExtJS tárolás és modellezés OpenLayers térképi elemekhez és rétegekhez

  • Egyebek

    • Integration with

      • MapFish Print

      • and GeoServer Printing Module

    • Réteg szerializálók (WMS, WMTS, XYZ)
