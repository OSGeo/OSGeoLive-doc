Jupyter jegyzettömb¶

Multimédiás tartalom hozzáadása a dokumentációhoz¶

A Jupyter jegyzettömb egy web alkalmazás, mely lehetővé teszi önnek élő kódot, egyenleteket, vizualizációt és magyarázó szöveget tartalmazó dokumentumok létrehozását és megosztását.

Két összetevőt egyesít:

  • Egy web alkalmazás, mely egy böngésző alapú , interaktív jegyzettömb dokumentáció készítését biztosítja.

  • Notebook dokumentumok, amelyek a webes alkalmazásban látható összes tartalom ábrázolását tárolják, beleértve a számítások bemeneteit és kimeneteit, magyarázó szöveget, matematikát, kódot, képeket, videókat, grafikonokat, térképeket és más multimédiás objektumokat.

A jegyzettömbök megoszthatók másokkal a GitHun-on, Dropbox-on és a Jupyter Notebook Viewer-en keresztül.

A felhasználás lehet: adat tisztítás és transzformálás, numerikus szimuláció, statisztikai modellezés, gépi tanulás és még továbbiak.

Alapfunkciók¶

Create Jupyter Notebooks which can contain:

  • Jelölt szöveg (címsorokkal, stílusokkal, bekezdésekkel stb.)

  • Képletek, matematika, grafikonok, térképek, …

  • Imported external libraries to add functionality

  • Code from multiple programming languages, including Python, R, Julia, Bash and many more.

Jupyter Notebooks on OSGeoLive can use geospatial functions from:

  • Numpy, SciPy Matplotlib, IRIS, Cartopy, gdal, Geopandas, pyshp, Fiona, netCDF4, PostgreSQL, psycopg2, R, rpy2, és még továbbiak.
