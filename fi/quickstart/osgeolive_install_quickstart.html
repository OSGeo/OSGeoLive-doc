Asenna OSGeoLive kovalevyllesi¶

OSGeoLive voidaan asentaa kovalevyllesi samalla tavalla kuin Ubuntu jakelu on asennettu.

Tämä prosessi vaidaan suorittaa Virtuaali koneessa jotta luodaan pysyvä OSGeoLive Virtuaali kone DVDltä tai ISO Imagesta.

Järjestelmävaatimukset¶

  • 20 GB vapaata kovalevytilaa

Asenna OSGeoLive¶

Valitse Install RELEASE

Anna salasana=”user”

Valitse kieli

Valitse aikavyöhyke

Valitse näppäimistö

Määritä mille levylle haluat asennuksen. (Ole varovainen, tämä voi tuhota olemassa olevan partition).

Oletusarvot asennukselle ovat:

  • Nimi = ”user”

  • käyttäjänimi = ”user”

  • salasana = ”user”

  • tietokoneen nimi = ”osgeolive”

  • Sisäänkirjoitus automaattisesti

Valitse Install

Mene kahville sillä välin kun asennusta suoritetaan.

Siinä kaikki.

Vihje

Asennus vinkki siinä tapauksessa kun käytät eri käyttäjänimeä kuin ”user”, tomcat sovellutukset tarvitsevat luvan palvelukomentoon kuten on kuvattu ohjeessa https://trac.osgeo.org/osgeo/ticket/1153
